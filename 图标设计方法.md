# SVG 图标设计方法

## 设计原则

统一图标风格，保持一致性：
- 统一尺寸
- 统一颜色
- 保留原始路径内容

## 参考样式

```svg
<svg xmlns="http://www.w3.org/2000/svg" width="128" height="128">
<path fill="#0F3B5A" d="..."/>
</svg>
```

**关键属性：**
- `width="128" height="128"` — 统一尺寸
- `fill="#0F3B5A"` — 统一颜色（深蓝色）

## 修改步骤

### 1. 尺寸统一

将 `width` 和 `height` 从原始值（如 200）改为 128：

```xml
<!-- 修改前 -->
<svg width="200" height="200">

<!-- 修改后 -->
<svg width="128" height="128">
```

### 2. 保留 viewBox

如果原始 SVG 有 `viewBox`，保留它以维持比例：

```xml
<svg width="128" height="128" viewBox="0 0 1024 1024">
```

### 3. 统一颜色

给每个 `<path>` 添加 `fill="#0F3B5A"`：

```xml
<!-- 修改前 -->
<path d="M746.642286 87.478857..."/>

<!-- 修改后 -->
<path fill="#0F3B5A" d="M746.642286 87.478857..."/>
```

### 4. 清理冗余属性

移除非标准属性（如 `t`, `class`, `p-id`, `version`）：

```xml
<!-- 修改前 -->
<svg t="1784171270552" class="icon" viewBox="0 0 1024 1024" version="1.1" xmlns="http://www.w3.org/2000/svg" p-id="5310" width="200" height="200">

<!-- 修改后 -->
<svg xmlns="http://www.w3.org/2000/svg" width="128" height="128" viewBox="0 0 1024 1024">
```

## 完整示例

### 修改前

```xml
<svg t="1784174957947" class="icon" viewBox="0 0 1024 1024" version="1.1" xmlns="http://www.w3.org/2000/svg" p-id="6080" width="200" height="200">
<path d="M805.306182 273.984646H319.367758..."></path>
<path d="M207.225535 199.226182H655.786667..."></path>
</svg>
```

### 修改后

```xml
<svg xmlns="http://www.w3.org/2000/svg" width="128" height="128" viewBox="0 0 1024 1024">
<path fill="#0F3B5A" d="M805.306182 273.984646H319.367758..."></path>
<path fill="#0F3B5A" d="M207.225535 199.226182H655.786667..."></path>
</svg>
```

## 注意事项

- 不修改路径数据（`d` 属性）
- 不修改 `viewBox`（如有）
- SVG 标准属性 `xmlns` 必须保留
- 颜色值使用大写十六进制 `#0F3B5A`
