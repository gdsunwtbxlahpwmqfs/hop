# Hop 用户手册

Qi Hop 用户手册包含了您使用 Qi Hop 开发和部署数据解决方案所需的全部信息。

我们列出了一些对您可能有用的入门参考。请在左侧查看完整的话题列表。如果您是开发者，想要扩展 Hop、构建新功能或修复发现的 bug，请查阅[开发者文档](http://hop.apache.org/dev-manual/latest/)

> **💡 提示:** Hop 将文档视为软件的一部分，同样可能存在 bug 或功能缺失。如果您发现任何不正确的信息或缺少关键文档，请[在 GitHub 上创建 issue](https://github.com/apache/hop/issues)。
