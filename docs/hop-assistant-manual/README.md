# hfxt data process 插件完整手册（hop-assistant-manual）

> 本手册系统整理了 hfxt data process 全部插件相关的介绍性文档与说明材料，涵盖插件体系架构、开发指南，以及转换插件（Transform）、动作插件（Action）、数据库插件（Database）、元数据类型（Metadata）、虚拟文件系统插件（VFS）、日志插件与密码编码插件等全部类别。原文档为英文，本手册已将其全面翻译并规范归并整理为中文，每个组件/模块独立成篇，按类别存放于不同子文件夹下。

### 第三部分　转换插件（[03-转换插件](03-转换插件)）

转换（Transform）是 HOP 的基石，用于处理和转换数据。Hop 提供超过 200 个转换插件，按功能分为以下类别：

| 子文件夹 | 说明 |
|----------|------|
| [输入类](03-转换插件/输入类) | 从各种数据源读取数据进入管道流 |
| [输出类](03-转换插件/输出类) | 将管道数据写入各种目标 |
| [数据库操作类](03-转换插件/数据库操作类) | 数据库增删改查与 SQL 执行 |
| [批量加载类](03-转换插件/批量加载类) | 各数据库的批量加载器 |
| [流程控制类](03-转换插件/流程控制类) | 过滤、分支、阻塞、延迟等 |
| [查找与连接类](03-转换插件/查找与连接类) | 流查找、合并连接、笛卡尔积 |
| [统计与分组类](03-转换插件/统计与分组类) | 分组聚合、去重、分析查询 |
| [字符串与文本处理类](03-转换插件/字符串与文本处理类) | 字符串操作、正则、NLP |
| [计算与字段操作类](03-转换插件/计算与字段操作类) | 计算器、常量、序列、值选择 |
| [脚本与编程类](03-转换插件/脚本与编程类) | Java/JS/Groovy 脚本、规则引擎 |
| [映射与子管道类](03-转换插件/映射与子管道类) | 映射、管道/工作流执行器 |
| [文件与编码操作类](03-转换插件/文件与编码操作类) | 文件存在性检查、编码转换、压缩 |
| [Beam大数据类](03-转换插件/Beam大数据类) | 基于 Apache Beam 引擎的转换 |
| [云服务类](03-转换插件/云服务类) | AWS、Azure、Google Cloud、Salesforce |
| [消息队列类](03-转换插件/消息队列类) | Kafka 消费与生产 |
| [网络与服务类](03-转换插件/网络与服务类) | HTTP、REST、Web 服务、邮件 |
| [安全与加密类](03-转换插件/安全与加密类) | PGP 加解密 |
| [Neo4j图数据库类](03-转换插件/Neo4j图数据库类) | Neo4j 图数据读写 |
| [日志与监控类](03-转换插件/日志与监控类) | 日志写入、数据探针 |
| [数据验证与质量类](03-转换插件/数据验证与质量类) | 数据验证、模式映射 |
| [Avro数据格式类](03-转换插件/Avro数据格式类) | Avro 编解码与文件读写 |
| [AI与语言模型类](03-转换插件/AI与语言模型类) | LLM 对话、事实核查等 AI 能力接入 |

| [网络与通信类](04-动作插件/网络与通信类) | HTTP、FTP、SFTP、Ping、Telnet、邮件 |
| [消息与监控类](04-动作插件/消息与监控类) | SNMP、Nagios、日志写入 |

| [XML与验证类](04-动作插件/XML与验证类) | XSD/DTD 验证、XSL 转换、XML 格式检查 |
