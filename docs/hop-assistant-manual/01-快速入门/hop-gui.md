# Hop GUI

Qi Hop GUI 是你的本地开发环境，用于构建、运行、预览和调试 Workflow 和 Pipeline。

## 启动 Qi Hop GUI

正如我们在 [Hop 工具](getting-started/hop-tools.md)中看到的，启动 Hop GUI 非常简单：

====
Windows::
--

hop-gui.bat
--

Linux, macOS::
--

./hop-gui.sh
--
====
