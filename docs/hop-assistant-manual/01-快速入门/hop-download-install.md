# 下载和安装 Hop

Hop 的设计尽可能简单易用。

获取和启动 Hop 再简单不过了：

## 下载

从[下载页面](http://hop.apache.org/download/)下载 Hop。

## 前提条件

Hop 是自包含的，你只需要一个 Java 运行时。
Hop 官方 Java 版本为 21。请查看 [Java 文档](https://java.com/en/download/help/download_options) 以下载并安装适用于你操作系统的 Java 运行时。

## 安装

将 Hop 解压到你选择的文件夹。
在该文件夹中，你可以通过相应的脚本运行各种 Hop [工具](hop-tools/index.md)。
