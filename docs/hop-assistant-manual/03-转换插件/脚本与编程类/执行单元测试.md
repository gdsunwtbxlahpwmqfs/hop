# 执行单元测试（Execute Unit Tests）

## 功能概述


执行单元测试转换用于获取并执行当前项目的可用单元测试（unit tests）。

## 主要选项

| 选项 | 说明 |
|------|------|
| 测试类型（Test type） | 选择要执行的单元测试类型 |
| 输出字段（Output fields） | 测试执行结果输出的字段，包括测试名称、状态等信息 |

## 注意事项

- 仅支持 Qi 数据治理平台 Engine 引擎，不支持 Spark、Flink、Dataflow 引擎。
- 适用于在管道中集成单元测试执行，验证数据处理逻辑的正确性。
