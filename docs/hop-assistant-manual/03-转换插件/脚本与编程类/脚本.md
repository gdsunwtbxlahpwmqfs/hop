# 脚本（Script）

## 功能概述


脚本转换允许您使用 [JSR-223](https://en.wikipedia.org/wiki/Scripting_for_the_Java_Platform) 标准支持的任何语言编写代码。
默认情况下，Hop 项目附带以下语言支持：
- [ECMAScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)（通过 [GraalVM JavaScript](https://www.graalvm.org/javascript/) 实现的 JavaScript，JSR 223）
- [Groovy](https://groovy-lang.org/)
- [Python](https://www.python.org/)（[Jython](https://www.jython.org/)）
在 Java 17 及更高版本中，JVM 不再包含 ECMAScript 引擎；Hop 内置了 GraalVM JavaScript，使得脚本转换无需额外配置即可使用。

## 主要选项

| 选项 | 说明 |
|------|------|
| 脚本语言（Script language） | 选择脚本语言，如 JavaScript、Groovy、Python 等 |
| 脚本（Script） | 编写的脚本代码 |
| 字段（Fields） | 脚本中使用的输入和输出字段定义 |

## 注意事项

- 在 Java 17 及以上版本中，Hop 内置 GraalVM JavaScript 引擎，无需额外配置。
- 支持 Qi 数据治理平台 Engine 引擎；Spark、Flink、Dataflow 引擎支持情况视具体实现而定。
