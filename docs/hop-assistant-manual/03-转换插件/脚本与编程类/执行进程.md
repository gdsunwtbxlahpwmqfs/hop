# 执行进程（Execute a process）

## 功能概述


执行进程转换用于在运行管道的主机上执行 shell 脚本。
此转换类似于工作流中的 Shell 动作（action），但可以在管道中对每一行数据执行。

## 主要选项

| 选项 | 说明 |
|------|------|
| 进程命令（Process command） | 要执行的 shell 命令或脚本 |
| 输出字段名（Result fieldname） | 存放进程输出结果（stdout）的字段名称 |
| 错误字段名（Error fieldname） | 存放进程错误输出（stderr）的字段名称 |
| 退出码字段名（Exit value fieldname） | 存放进程退出码的字段名称 |

## 注意事项

- 此转换在运行管道的主机上执行命令，需注意系统安全与权限。
- 支持 Qi 数据治理平台 Engine 引擎；Spark、Flink、Dataflow 引擎支持情况视具体实现而定。
