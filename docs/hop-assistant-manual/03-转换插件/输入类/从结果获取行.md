# 从结果获取行

从结果获取行（Get Rows from Result）转换返回先前由工作流中另一个管道生成的行。这些行通过"复制行到结果（Copy rows to result）"转换或管道执行器（Pipeline Executor）转换传递到此转换。

你无需在"从结果获取行"转换中定义任何字段即可检索所有数据行字段。

你可以在转换中输入期望从工作流中先前管道获取的字段元数据。

## 注意事项

- 该转换仅支持 Hop Engine，不支持 Spark、Flink、Dataflow。
- 从结果获取行主要是出于历史原因保留，用于通过工作流构建循环。请查阅最佳实践文档，了解 Hop 中更好的循环实现方式。
