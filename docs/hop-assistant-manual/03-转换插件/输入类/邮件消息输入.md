# 邮件消息输入

邮件消息输入（Email Messages Input）转换允许你使用 POP3、IMAP 或 MBOX 标准协议从邮件服务器检索消息及其附件。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 协议 | 使用的邮件协议（POP3、IMAP 或 MBOX） |
| 服务器 | 邮件服务器地址 |
| 端口 | 邮件服务器端口 |
| 用户名/密码 | 邮箱账户凭据 |
| 使用 SSL | 是否使用 SSL 加密连接 |
| 文件夹 | 要读取的邮箱文件夹（如 INBOX） |

## 注意事项

- 支持 POP3、IMAP 和 MBOX 三种协议。
- 可同时检索邮件正文和附件信息。
- 注意邮件服务器的连接限制和认证配置。
