# 从结果获取文件

从结果获取文件（Files from Result）转换读取通过其他转换中"将文件名加入结果（add filename to result）"选项添加的文件名及附加元数据。

每当文件在管道或工作流中被处理、使用或创建时，如果设置了"将文件名加入结果"选项（例如在文本文件输出转换中），则文件详情、工作流动作、转换等信息都会被捕获并添加到内部结果集中。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |

## 注意事项

- 该转换仅支持 Hop Engine，不支持 Spark、Flink、Dataflow。
- 它读取的文件信息来自先前转换或管道中通过"将文件名加入结果"选项收集的结果集。
