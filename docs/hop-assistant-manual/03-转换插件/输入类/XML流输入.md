# XML 流输入（StAX）

XML 流输入（StAX）转换使用 XML 流式 API（StAX）解析器从 XML 文件中读取数据。

## 用途

该转换非常适合快速处理大型和复杂的数据结构。

与使用内存处理且可能需要清除文件部分内容的"从 XML 获取数据（Get Data from XML）"转换不同，XML 流输入（StAX）转换将处理逻辑移入管道中。

该转换本身提供原始 XML 数据流以及额外的处理信息。

## 注意事项

- 当其他转换存在限制或需要在以下情况解析 XML 时，推荐使用此流式转换：
  - XML 文件过大，无法全部装入内存；
  - 需要在管道中进一步处理解析逻辑；
  - 需要处理复杂且深层的 XML 数据结构。
- StAX 流式解析相比基于 DOM 的内存解析更节省内存，适合处理大文件。
