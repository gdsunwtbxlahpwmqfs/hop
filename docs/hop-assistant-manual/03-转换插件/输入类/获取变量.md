# 获取变量

获取变量（Get Variables）转换允许你获取变量的值并从中创建字段。只有当需要在字段中使用变量值时，才需要使用此转换。

> **提示**：如果需要引用先前管道的数据行字段，请使用"从结果获取行（Get rows from result）"转换。

## 用途说明

请注意，工作流/环境变量仅设置一次。管道需要启动才能获取任何新变量。正在运行、顺序或嵌套（嵌套管道在技术上与同一管道相同）的管道无法获取新的变量值。当管道为管道执行器中的每一行启动时，该管道被视为已启动。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 字段名 | 变量值对应的输出字段名 |
| 变量 | 要获取的变量（使用 `${变量名}` 或 `%%变量名%%` 格式） |

## 注意事项

- 变量值在管道启动时读取，运行中的管道无法获取新设置的变量值。
- 该转换仅在你需要将变量值作为字段使用时才必要；在 SQL、文件名等处可直接引用变量。
