# 伪造数据

伪造数据（Fake Data）转换允许你使用 Java Faker 库生成假数据。它可用于为开发、测试或项目展示生成逼真的数据。

例如，可以生成一些随机的宝可梦（Pokémon）数据。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 字段 | 要生成的假字段定义，每个字段指定 Java Faker 表达式 |
| Faker 表达式 | 使用 Java Faker 库的表达式（如 `name.fullName`、`address.city` 等） |

## 注意事项

- 该转换支持 Hop Engine、Spark、Flink、Dataflow 等多种引擎。
- 底层使用 Java Faker 库（[文档](https://github.com/DiUS/java-faker)）。
- 适用于生成开发、测试或演示用的逼真样本数据，如姓名、地址、公司名称等。
