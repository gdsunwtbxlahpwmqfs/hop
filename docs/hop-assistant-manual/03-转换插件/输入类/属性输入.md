# 属性输入

属性输入（Property Input）转换从 Java properties 文件中读取键/值对。

> **提示**：有关此文件格式的更多信息，请参见 https://en.wikipedia.org/wiki/.properties[Wikipedia]。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 文件 | 要读取的 properties 文件 |
| 字段 | 输出字段定义（键、值等） |

## 注意事项

- 该转换用于读取 Java properties 格式的配置文件，将键值对解析为数据行。
- 适用于读取应用程序配置或国际化资源文件。
