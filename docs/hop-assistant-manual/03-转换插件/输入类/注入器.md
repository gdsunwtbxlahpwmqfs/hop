# 注入器

注入器（Injector）转换用于特殊用途的管道，在这些管道中，你希望通过 Hop API 和 Java 将行"注入"管道，或从流式输入转换（如 Kafka Consumer 转换）注入行。

除此之外，你可以用它构建"无头"管道：即在设计时没有输入、不从文件或数据库读取数据的管道。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 字段名（Fieldname） | 指定要注入的行的字段名 |
| 类型（Type） | 指定数据类型 |

## 注意事项

- 该转换主要用于通过编程方式（Hop API/Java）或流式数据源（如 Kafka Consumer）向管道注入数据。
- 适用于构建在设计阶段不读取具体数据源的"无头"管道。
