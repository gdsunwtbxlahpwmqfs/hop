# 行生成器

行生成器（Row Generator / Generate Rows）转换输出指定数量的行，这些行可以为空或包含一组硬编码字段。默认情况下，行是空的，但也可以包含若干静态字段。

要使用字段，必须有一个数据行。某些转换（如表输入）会自动生成数据行（如果运行了预览）。如果没有数据行，可以添加"生成行"转换来创建行，以便使用字段或打印变量、参数等。

该转换主要用于测试目的。它对于生成固定数量的行非常有用，例如，你需要恰好 12 行来代表 12 个月。

有时你可以使用生成行转换生成一行作为管道的起始点。例如，生成一行包含两三个字段值，用于参数化 SQL，然后再生成真正的行。

## 示例

- 要生成样本数据行，可在管道中添加两个项：将"生成行"转换连接到"添加常量"。在"添加常量"中添加要生成的字段；在"生成行"中限制行数。
- 要循环 100 行，可在管道中使用"生成行"和"添加序列"转换并将它们连接。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 限制 | 要生成的行数 |
| 字段 | 生成的行中包含的静态字段定义 |

## 注意事项

- 该转换主要用于测试和生成固定数量的行。
- 生成的行默认为空，但可包含静态字段值。
