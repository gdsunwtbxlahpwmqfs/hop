# LDAP 输入

LDAP 输入（LDAP Input）转换允许你从 LDAP 服务器读取用户、角色和其他数据等信息。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 主机名 | LDAP 服务器主机名 |
| 端口 | LDAP 服务器端口 |
| 绑定 DN | 用于认证的绑定可分辨名称 |
| 搜索基础 | LDAP 搜索基础 DN |
| 搜索过滤器 | LDAP 搜索过滤器 |
| 字段 | 输出字段定义 |

## 注意事项

- 该转换用于从目录服务（如 Active Directory、OpenLDAP）中读取数据。
- 支持配置搜索基础和过滤器以精确检索所需的目录条目。
