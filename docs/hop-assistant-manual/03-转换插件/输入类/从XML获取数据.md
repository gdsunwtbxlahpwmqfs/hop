# 从 XML 获取数据

从 XML 获取数据（Get Data From XML）转换提供使用 XPath 规范从任何类型的 XML 文件中读取数据的能力。

该转换可以从 3 种来源（文件、流和 URL）以 2 种模式（静态或动态）读取数据。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 文件或目录 | 指定要读取的 XML 文件或目录 |
| 正则表达式 | 用于筛选文件的正则表达式 |
| 循环 XPath | 用于循环读取的 XPath 路径 |
| 字段（Fields） | 通过 XPath 表达式定义要提取的字段及其类型 |

## 注意事项

- 该转换基于 XPath 规范提取数据，适用于结构化 XML 文件。
- 对于超大或复杂的 XML 文件，可考虑使用 XML 流输入（StAX）转换以获得更好的性能。
- 可在 Samples 项目中查看示例：`transforms/get-data-from-xml.hpl`。
