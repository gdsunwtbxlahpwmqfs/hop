# 获取系统信息

获取系统信息（Get System Info）转换从 Hop 项目、或 Hop 运行所在的环境或系统中检索系统信息。

该转换包含一个表格，你可以在其中指定名称并将其分配给要检索的任何可用系统信息类型。此转换生成单行，其中包含所请求信息的字段。

它还可以接受任意数量的输入流，聚合该转换定义的任何字段，并将组合结果发送到输出流。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 字段名 | 为检索的系统信息指定的字段名 |
| 类型 | 系统信息类型（如系统日期、命令行参数、JVM 信息等） |

## 注意事项

- 该转换生成单行输出，包含所请求的系统信息。
- 可检索的信息包括系统日期、可用内存、命令行参数、Hop 版本等。
