## 选项

| 选项 | 描述 |
|---|---|
| Cassandra connection |  |
| 在此指定要使用的连接名称，可以是固定值或变量表达式。 |  |
| Transport max length |  |
| 设置可发送的最大对象大小（可选）。 |  |
| Execute query for each incoming row? |  |
| 选择此项将使查询重复执行：每个输入行执行一次。 |  |
| CQL |  |
| 您要使用的 Cassandra Query Language 语句。 |  |
