# 文本文件输入

文本文件输入转换用于从多种文本文件类型中读取数据，包括电子表格生成的格式和固定宽度的扁平文件。

该转换的功能允许你从文件或目录列表中读取数据，使用正则表达式形式的通配符，并接受来自先前转换的通用化文件名。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称，在单个管道中必须唯一 |
| 文件或目录 | 指定要读取的文件或目录 |
| 正则表达式 | 用于筛选文件的正则表达式通配符 |
| 选定文件 | 显示已选定的文件列表 |
| 内容分隔符 | 字段之间的分隔符 |
| 封闭符 | 文本字段的封闭符 |
| 表头/页眉行数 | 要跳过的表头或页眉行数 |
| 文件编码 | 文件的字符编码 |

## 注意事项

- 该转换支持 Hop Engine、Spark、Flink、Dataflow 等多种引擎。
- 相比"CSV 文件输入"，该转换提供更全面的选项，适用于更复杂的文本文件解析场景，包括固定宽度文件和多种电子表格格式。
