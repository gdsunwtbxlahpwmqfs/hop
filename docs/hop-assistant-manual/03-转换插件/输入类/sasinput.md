## Options

| Option | Description |
|---|---|
| Transform name |  |
| Transform 的名称，此名称在单个 Pipeline 中必须唯一。 |  |
| Field in the input to use as filename |  |
| 选择运行时将包含文件名的输入字段。 |  |
| Limit |  |
| 读取指定行数后停止。如果未指定或指定小于等于零的数字，将读取 SAS 文件中的所有行。 |  |
| The selected fields from the files |  |
| 如果使用 "Get Fields" 按钮，可以填充此数据表格。 |  |
