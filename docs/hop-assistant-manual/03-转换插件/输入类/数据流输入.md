# 数据流输入

数据流输入（Data Stream Input）转换从 IPC 数据流中读取数据，将配置的数据流（Data Stream）中的行读入 Hop 管道。

它支持所有可用的数据流实现（Arrow File Stream、Arrow Random Access File 和 Arrow Flight）。

## 主要选项

| 选项 | 说明 |
|------|------|
| 数据流（Data Stream） | 要读取的源数据流名称。这是一个下拉列表，显示项目中所有已定义的数据流 |

## 注意事项

- 需要先在项目中定义数据流元数据。
- 支持 Apache Arrow 格式的数据流，包括 Arrow File Stream、Arrow Random Access File 和 Arrow Flight。
