# 数据网格

数据网格（Data Grid）转换允许你在网格中输入静态的行列表，类似于管道内的电子表格。这通常用于测试、参考或演示目的。

将数据网格放置到数据编排网格上。在"Meta（元数据）"选项卡中添加字段名称、类型和格式。然后在"Data（数据）"选项卡中为每个数据行添加字段值。

## 主要选项

| 选项卡 | 说明 |
|--------|------|
| Meta（元数据）选项卡 | 指定数据的字段元数据（输出规范），包括字段名、类型、格式等 |
| Data（数据）选项卡 | 包含实际数据。所有内容均以字符串格式输入，因此需在元数据选项卡中使用正确的格式掩码 |

## 注意事项

- 数据选项卡中的所有值都以字符串形式输入，务必在元数据选项卡中设置正确的格式掩码，以确保数据正确解析为目标类型（如日期、数字等）。
- 该转换主要用于测试、参考或演示场景，不适合存储大量生产数据。
