# 阻塞直到转换完成

## 功能概述


阻塞直到转换完成（Blocking until transforms finish）转换简单地等待，直到对话框中指定的所有转换副本都已完成。
你可以使用它来避免管道转换副本之间存在的自然并发（并行性）。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 要等待的转换 | 需要等待完成的转换列表 |

## 注意事项

- 该转换等待指定转换的所有副本完全完成后再继续。
- 用于消除转换副本之间的并行并发，确保同步执行顺序。
- 适用于需要严格控制执行顺序的管道设计。
