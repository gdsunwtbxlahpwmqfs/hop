# 追加流

## 功能概述


追加流（Append Streams）转换从两个转换读取数据，仅在第一个流处理完成后才处理第二个流。
与往常一样，来自两个转换的输入数据的行布局必须相同：相同的行长度、相同的数据类型、相同字段索引位置的相同字段。
**重要**：如果你不关心输出行的顺序，可以使用任何转换来创建 2 个或更多数据流的并集。

## 注意事项

- 该转换仅支持 Qi 数据治理平台 Engine，不支持 Spark、Flink、Dataflow。
- 两个输入流的字段布局必须完全相同。
- 该转换保持顺序：先输出第一个流的所有行，再输出第二个流的所有行。
