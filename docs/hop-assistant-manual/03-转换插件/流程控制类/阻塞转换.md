# 阻塞转换

## 功能概述


阻塞转换（Blocking transform）阻塞所有输出，直到从先前转换接收到最后一行。
> **注意**：该转换阻塞直到获取最后一行。它*不会*阻塞直到先前转换 100% 完成，该功能在另一个转换"阻塞直到转换完成（Blocking until transforms finish）"中提供。
此时，最后一行被发送到下一个转换，或完整的输入被发送到下一个转换。
使用阻塞转换来触发插件、存储过程、Java 脚本等，或用于同步目的。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 传递所有行 | 是否传递所有行（而非仅最后一行） |

## 注意事项

- 该转换阻塞直到接收到最后一行数据。
- 适用于需要在所有输入数据到达后触发后续操作的场景。
- 与"阻塞直到转换完成"不同，它在收到最后一行时即解除阻塞。
