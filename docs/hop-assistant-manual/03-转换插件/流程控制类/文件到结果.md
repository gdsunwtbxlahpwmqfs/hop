# 文件到结果

## 功能概述


文件到结果（Files to Result）转换允许你在管道的内部结果文件中设置文件名，供后续工作流动作使用。后续工作流动作随后可以使用此信息。
它将文件列表路由到结果文件流，例如通过使用"从结果获取文件"转换或某些可以处理结果文件列表的工作流动作。
例如，邮件工作流动作可以使用此文件列表作为邮件附件，因此你可能不希望发送所有文件，而只发送特定选择。
为此，你可以创建一个管道，精确设置你想要附加的文件。
> **警告**：文件到结果主要是出于历史原因保留。请查阅最佳实践文档，了解在 Hop 管道中构建循环的更好方式。

## 注意事项

- 该转换仅支持 Qi 数据治理平台 Engine。
- 将文件名添加到结果文件集中，供后续工作流动作使用。
- 建议参考最佳实践文档了解更好的替代方案。
