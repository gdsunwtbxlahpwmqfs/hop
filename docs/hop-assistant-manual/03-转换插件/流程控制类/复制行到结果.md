# 复制行到结果

## 功能概述


复制行到结果（Copy Rows To Result）转换允许你通过内部结果行集，将数据行（在内存中）传递到工作流中的下一个管道（或工作流动作）。
请记住，不会复制值或变量，仅复制行数据。

## 用途

复制行到结果转换可与"从结果获取行（Get Rows from Result）"转换配对使用，以在管道中检索结果行集。在某些情况下，不需要"从结果获取行"转换即可检索结果集，例如工作流中的管道转换。数据行可以在工作流的管道之间传递，但无法在工作流中直接访问数据行。

## 注意事项

- 该转换仅支持 Qi 数据治理平台 Engine，不支持 Spark、Flink、Dataflow。
- 仅传递行数据，不传递变量。
- 与"从结果获取行"转换配合使用，实现管道间的数据传递。
