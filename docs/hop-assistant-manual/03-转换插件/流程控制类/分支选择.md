# 分支选择（Switch / Case）

分支选择（Switch / Case）转换实现了 Java 等流行编程语言中的 [Switch/Case 语句](https://en.wikipedia.org/wiki/Switch_statement)。

在我们的场景中，它根据某个字段中遇到的值，将数据行路由到一个或多个目标转换。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 字段名 | 用于分支选择的字段名 |
| 字段类型 | 用于比较的字段数据类型 |
| Case 值 | 各 case 值及其对应的目标转换 |
| 默认目标转换 | 未匹配任何 case 时数据行路由到的目标转换 |

## 注意事项

- 该转换根据指定字段的值将行路由到不同的目标转换。
- 可配置多个 case 值和一个默认目标。
- 适用于基于分类字段的数据分流。
