# 识别最后一行

## 功能概述


识别最后一行（Identify last row in a stream）管道转换生成一个布尔字段，最后一行填充为 true，其余行填充为 false。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 结果字段名 | 标识是否为最后一行的布尔字段名 |

## 注意事项

- 该转换仅支持 Qi 数据治理平台 Engine，不支持 Spark、Flink、Dataflow。
- 输出一个布尔字段，仅最后一行为 true。
- 适用于需要对最后一行进行特殊处理的场景。
