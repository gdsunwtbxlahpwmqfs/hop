# 蓄水池采样

蓄水池采样（Reservoir Sampling）转换允许你在预先不知道传入数据流总行数的情况下，从传入数据流中采样固定数量的行。

该转换使用均匀采样；所有传入行被选中的机会均等。

当与 ARFF 输出转换结合使用以生成适合 WEKA 使用的适当大小数据集时，此转换特别有用。

蓄水池采样转换使用 Jeffrey Vitter 的[算法 R（Algorithm R）](https://en.wikipedia.org/wiki/Reservoir_sampling)。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 样本大小 | 要采样的固定行数 |
| 随机种子 | 随机数生成的种子 |

## 注意事项

- 该转换适用于总行数未知时的均匀采样。
- 所有行被选中的概率相等。
- 基于 Algorithm R 实现，常用于大数据集的随机采样。
