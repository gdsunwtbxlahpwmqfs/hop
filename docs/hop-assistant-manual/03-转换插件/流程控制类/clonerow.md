## 选项

| 选项 | 描述 |
|---|---|
| Transform name | Transform 的名称。 |
| Nr clones | 在原始行之后要添加的克隆数量。 |
| Add clone flag a | 如果想在输出中添加一个布尔字段来指示该行是否为克隆行，请勾选此选项。 |
| Clone flag field | 克隆标志字段的名称。 |
