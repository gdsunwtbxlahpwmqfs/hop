# 过滤行

过滤行（Filter Rows）转换允许你基于条件和比较来过滤行。它通过将 True 或 False 结果/跳连接到另一个转换来影响工作流。有 3 个输出路径可用（True/False/主输出）。主输出仅遵循 True 路径。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 发送"True"数据到的转换 | 条件为真时发送数据的目标转换 |
| 发送"False"数据到的转换 | 条件为假时发送数据的目标转换 |
| 过滤条件 | 用于过滤行的条件表达式（支持 `${变量名}`） |

## 注意事项

- 该转换支持 Hop Engine、Spark、Flink、Dataflow 等多种引擎。
- 有三个输出路径：True、False 和主输出（主输出仅遵循 True 路径）。
- 条件中可使用 Hop 变量。
