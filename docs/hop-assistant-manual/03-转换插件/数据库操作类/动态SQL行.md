# 动态 SQL 行

动态 SQL 行（Dynamic SQL Row）转换允许你执行定义在数据库字段中的 SQL 语句。转换返回的查找值作为新字段添加到数据流中。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 数据库连接 | 用于执行的数据库连接 |
| SQL 字段名 | 包含 SQL 语句的字段名 |
| 每行返回的记录数 | 限制每个输入行返回的记录数 |
| 变量替换 | 是否替换 SQL 中的变量（如 `${变量名}`） |

## 注意事项

- SQL 语句来自输入字段，支持完全动态的查询。
- 返回的查找结果作为新字段添加到输出流。
- 支持变量替换以进一步参数化查询。
