# 数据库连接

数据库连接（Database Join）转换允许你使用从管道字段获得的数据作为参数，对数据库运行查询。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 数据库连接 | 用于查询的数据库连接 |
| SQL 查询 | 使用 `?` 占位符的 SQL 语句 |
| 参数 | 从输入字段映射到 SQL 占位符的参数 |
| 每行返回的记录数 | 限制每个输入行返回的记录数 |

## 注意事项

- 该转换使用输入字段值作为 SQL 查询参数，为每个输入行执行一次查询。
- 适用于需要根据输入数据动态查询数据库的场景。
- 通过限制返回记录数可控制输出行数。
