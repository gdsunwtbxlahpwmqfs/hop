# 调用数据库存储过程

调用数据库存储过程（Call DB Procedure）转换允许你执行数据库存储过程（或函数）并获取结果。

使用 MySQL 和 JDBC 时，无法检索存储过程的结果集。存储过程和函数只能通过其函数参数返回值，这些参数必须在 DB 过程调用配置的"参数"部分中定义。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 数据库连接 | 目标数据库连接 |
| 过程名称 | 要调用的存储过程或函数名称 |
| 参数 | 输入/输出/输入输出参数定义 |
| 结果字段名 | 存储返回结果的字段名 |

## 注意事项

- 该转换支持 Hop Engine、Spark、Flink、Dataflow 等多种引擎。
- 参数的方向（IN、OUT、INOUT）需正确定义。
- 某些数据库（如 MySQL via JDBC）不支持检索存储过程的结果集，仅能通过参数返回值。
