# 执行 SQL 脚本

执行 SQL 脚本（Execute SQL Script）转换运行 SQL 脚本，可在管道初始化阶段运行一次，或为转换接收的每个输入行运行一次，并将入站字段传递到其输出。

当用于每个输入行时，可将参数传递给转换中指定的 SQL 脚本。仅在此模式下，输入字段才会传递到下一个跳。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 数据库连接 | 用于执行脚本的数据库连接 |
| SQL 脚本 | 要执行的 SQL 脚本 |
| 对每行执行 | 是否为每个输入行执行脚本 |
| 参数 | 传递给 SQL 脚本的参数 |
| 变量替换 | 是否替换 SQL 中的变量（如 `${变量名}`） |

## 注意事项

- 可选择在初始化阶段执行一次或为每个输入行执行。
- 仅在每行执行模式下，输入字段才会传递到下一跳。
- 支持 SQL 脚本中的变量替换。
