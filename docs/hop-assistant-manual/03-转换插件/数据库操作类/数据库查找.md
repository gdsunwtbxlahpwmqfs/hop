# 数据库查找

数据库查找（Database Lookup）转换允许你在数据库表中查找值。查找的值作为新字段添加到数据流中。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 数据库连接 | 用于查找的数据库连接 |
| 查找表 | 要查找的数据库表 |
| 查找条件 | 用于查找的键字段和条件 |
| 返回字段 | 要返回的查找值字段 |
| 缓存查找数据 | 是否缓存查找结果以提升性能 |

## 注意事项

- 该转换支持 Hop Engine、Spark、Flink、Dataflow 等多种引擎。
- 启用缓存可显著提升重复查找的性能。
- 查找值作为新字段添加到输出流中。
