# 表是否存在

## 功能概述


表是否存在（Table Exists）转换验证数据库中是否存在某个表。该转换的结果是一个布尔标志字段。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 数据库连接 | 用于检查的数据库连接 |
| 表名字段 | 包含要检查表名的字段 |
| 结果字段名 | 输出布尔结果的字段名 |
