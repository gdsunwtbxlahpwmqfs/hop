## 选项

| 选项 | 描述 |
|---|---|
| Transform name | Transform 的名称。 |
| Connection | 要使用的数据库连接。 |
| Schema name | 要查找的表所在的 schema。 |
| Table name field | 将包含要查找的数据库名称的字段名称。 |
| Result fieldname | 将在此 transform 输出中包含布尔结果标志的字段名称。 |
