# 插入/更新

插入/更新（Insert/Update）转换首先使用一个或多个查找键在表中查找行。如果找不到指定键对应的表行，则插入新行。

如果能找到且要更新的字段相同，则不执行任何操作。

如果不完全相同，则更新表中的行。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 数据库连接 | 目标数据库连接 |
| 目标模式 | 目标表所在模式 |
| 目标表 | 要插入/更新的目标表名 |
| 查找键 | 用于查找行的键字段映射 |
| 更新字段 | 要更新的字段映射 |
| 提交记录数 | 每次提交的记录数量 |

## 注意事项

- 该转换根据查找键判断是插入还是更新。
- 适用于数据同步和增量加载场景。
- 注意提交记录数的设置以平衡性能与事务安全。
