# 表比较

表比较（Table Compare）转换比较两个数据库表中的数据。

该转换比较两个表（前提是它们具有相同的布局）中的数据，找出两个表数据之间的差异并记录。

差异摘要作为转换的正常输出提供。要获取找到的每个单独错误，请捕获错误处理输出（通过错误跳）。

> **提示**：你可以点击红色错误处理箭头来配置有关所发现差异的附加信息。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 数据库连接（参考） | 参考表的数据库连接 |
| 参考表 | 用作参考的表 |
| 数据库连接（比较） | 比较表的数据库连接 |
| 比较表 | 要与参考表比较的表 |
| 比较字段 | 用于比较的字段映射 |

## 注意事项

- 两个表必须具有相同的布局才能进行比较。
- 差异摘要通过正常输出提供，详细差异通过错误处理跳捕获。
