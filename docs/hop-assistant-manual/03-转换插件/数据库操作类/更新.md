# 更新

更新（Update）转换使用一个或多个查找键在表中查找行，并在需要时更新它。

如果找不到行，则不执行任何操作。

如果找到行，则更新表中的行。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 数据库连接 | 目标数据库连接 |
| 目标模式 | 目标表所在模式 |
| 目标表 | 要更新的目标表名 |
| 查找键 | 用于查找行的键字段映射 |
| 更新字段 | 要更新的字段映射 |
| 提交记录数 | 每次提交的记录数量 |

## 注意事项

- 该转换仅执行更新操作，不插入新行。
- 如需插入新行，请使用"插入/更新"转换。
- 找不到匹配行时不执行任何操作。
