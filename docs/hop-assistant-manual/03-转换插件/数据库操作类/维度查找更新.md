# 维度查找/更新

维度查找/更新（Dimension Lookup/Update）转换允许你实现 Ralph Kimball 的[缓慢变化维度（Slowly Changing Dimension）](https://en.wikipedia.org/wiki/Slowly_changing_dimension)，支持两种类型：类型 1（更新）和类型 2（插入），以及一些附加功能。

该转换可用于填充维度表，或在缓慢变化维度中查找值而不进行任何更改。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 数据库连接 | 目标数据库连接 |
| 目标表 | 维度表名 |
| 操作 | 更新维度或查找维度 |
| 业务键 | 用于查找的业务键字段 |
| 技术键 | 维度表的技术键字段 |
| 更新类型 | 类型 1（更新）或类型 2（插入） |

## 注意事项

- 类型 1（Type I）：直接更新维度记录，不保留历史。
- 类型 2（Type II）：插入新记录以保留历史，通常包含生效日期和过期日期字段。
- 该转换是数据仓库维度建模的核心组件。
