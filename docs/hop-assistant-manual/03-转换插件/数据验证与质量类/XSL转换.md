# XSL 转换（XSL Transformation）

XSL Transformation 转换对 XML 执行 XSL 转换。

XSLT 是 Extensible Stylesheet Language Transformations（可扩展样式表语言转换）的缩写。该转换通过应用 XSLT 样式表将 XML 文档转换为其他格式（如 HTML、纯文本或其他 XML 结构），适用于数据格式转换和文档处理场景。

## 主要选项

| 选项 | 说明 |
|------|------|
| XML 来源 | 指定要转换的 XML 数据来源（文件或输入字段） |
| XSL 文件 | 指定用于转换的 XSLT 样式表文件路径 |
| 输出字段 | 指定存放转换结果的输出字段 |
| 输出属性 | 配置转换输出的相关属性 |
