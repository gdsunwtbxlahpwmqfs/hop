# 流模式合并（Stream Schema Merge）

Stream Schema Merge 转换合并多个具有不同结构的输入，并创建字段的组合。

提示：如果要合并的数据流包含同名但类型不同的字段，这些字段将被合并为一个 `String` 类型的单一字段。如果您希望确保字段保留其原始数据类型，请确保将它们转换为相同的类型（例如通过 Select Values 转换的元数据选项卡）。

## 主要选项

| 选项 | 说明 |
|------|------|
| 输入流 | 指定要合并的多个输入数据流 |
| 字段合并方式 | 配置不同结构字段的合并规则（同名同类型字段合并，同名不同类型字段转为 String） |
