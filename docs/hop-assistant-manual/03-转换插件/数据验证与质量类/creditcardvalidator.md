## Options

- Transform name：Transform 的名称，在 Pipeline 中唯一
- Credit card field：执行期间包含信用卡号的输入字段名称
- Get only digits?

** Credit card type field：保存信用卡类型（发卡机构）的输出字段名称
** Not valid message：保存错误消息的输出字段名称。
