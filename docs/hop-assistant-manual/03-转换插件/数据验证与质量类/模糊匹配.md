# 模糊匹配（Fuzzy match）

Fuzzy Match 转换使用通过计算两个数据流相似度的重复检测算法，查找可能匹配的字符串。

该转换根据用户定义的最小值或最大值阈值，以分隔列表的形式返回匹配的值。它适用于数据清洗、去重、近似匹配等场景，能够识别拼写相近或存在细微差异的相似文本。

## 主要选项

| 选项 | 说明 |
|------|------|
| 匹配源字段 | 指定主数据流中用于匹配的字段 |
| 查找字段 | 指定参考数据流中用于比对的字段 |
| 算法 | 选择匹配算法（如 Levenshtein、Soundex、Double Metaphone、Jaro-Winkler 等） |
| 最小相似度 | 设置匹配所需的最小相似度阈值 |
| 最大相似度 | 设置匹配的最大相似度阈值 |
| 输出匹配值 | 指定匹配结果以分隔列表形式输出的字段 |
