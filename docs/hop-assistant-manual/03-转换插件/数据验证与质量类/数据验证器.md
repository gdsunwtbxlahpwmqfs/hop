# 数据验证器（Data Validator）

Data Validator 转换允许您为输入数据添加多种类型的数据验证。每项验证都可以对输入行中的各个字段进行校验。

当检测到某条输入行未通过该转换中定义的一项或多项验证时，验证器将抛出错误（异常）。要获取所有验证错误的列表，您可以从此转换定义一个错误处理（error handling）跳转。

## 主要选项

| 选项 | 说明 |
|------|------|
| 验证字段 | 选择要进行验证的输入字段 |
| 验证规则 | 为每个字段配置验证规则，如非空检查、数据类型、长度范围、枚举值、正则匹配等 |
| 错误处理 | 配置当验证失败时的处理方式，可将错误行导向错误处理分支 |
| 验证描述 | 为每条验证规则添加描述说明 |
