# 序列化到文件

序列化到文件（Serialize To File）转换以二进制格式将数据行存储到文件中。

二进制格式相比文本（扁平）文件具有优势，因为在读回时无需解析内容。元数据也存储在文件中。

序列化到文件转换支持一次性写入访问模式，不支持追加到现有文件。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 文件名 | 输出的二进制文件名 |

## 注意事项

- 二进制格式读取速度快，因为无需解析，元数据已包含在文件中。
- 仅支持一次性写入，不支持追加到已有文件。
- 可使用"从文件反序列化（De-serialize From File）"转换读取此转换生成的文件。
