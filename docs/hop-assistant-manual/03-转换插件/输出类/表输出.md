# 表输出

表输出（Table Output）转换将数据插入到关系型数据库表中。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 数据库连接 | 目标数据库连接 |
| 目标模式 | 目标表所在模式 |
| 目标表 | 要插入数据的目标表名 |
| 提交记录数 | 每次提交的记录数量 |
| 截断表 | 是否在插入前截断表 |
| 使用批处理 | 是否使用批处理插入 |

## 注意事项

- 该转换支持 Hop Engine、Spark、Flink、Dataflow 等多种引擎。
- 可通过设置提交记录数来优化性能。
- 启用批处理插入可显著提升写入性能。
- 该转换仅执行插入操作，如需更新或插入更新，请使用相应的转换。
