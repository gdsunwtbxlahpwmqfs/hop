# 数据流输出

数据流输出（Data Stream Output）转换将 Hop 管道中的行写入已配置的数据流（Data Stream）。

它支持所有可用的数据流实现（Arrow File Stream、Arrow Random Access File 和 Arrow Flight）。

## 主要选项

| 选项 | 说明 |
|------|------|
| 数据流（Data Stream） | 要写入的目标数据流名称。这是一个下拉列表，显示项目中所有已定义的数据流 |

## 注意事项

- 需要先在项目中定义数据流元数据。
- 支持 Apache Arrow 格式的数据流，包括 Arrow File Stream、Arrow Random Access File 和 Arrow Flight。
