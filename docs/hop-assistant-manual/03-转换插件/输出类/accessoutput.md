## 选项

| 选项 | 描述 |
|---|---|
| Transform name | Transform 的名称。 |
| The database filename | 要连接的数据库文件名。 |
| Create database | 启用此项可生成新的 Access 数据库文件。 |
| Target table | 指定要输出数据的目标表。 |
| Create table | 启用此项可在 Access 数据库中创建新表。 |
| Commit size | 定义输出数据时的提交大小。 |
