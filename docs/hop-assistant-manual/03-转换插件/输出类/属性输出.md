# 属性输出

属性输出（Properties Output）转换将一组数据行输出到 Java properties 文件。

数据需要以键/值格式构建，才能用于 properties 文件。

> **提示**：有关此文件格式的更多信息，请参见 https://en.wikipedia.org/wiki/.properties[Wikipedia]。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 文件名 | 输出的 properties 文件名 |
| 键字段 | 用作键的字段 |
| 值字段 | 用作值的字段 |

## 注意事项

- 输入数据必须以键/值对格式组织。
- 适用于生成 Java 应用程序配置文件或国际化资源文件。
