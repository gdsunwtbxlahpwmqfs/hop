# Microsoft Access 输出

Microsoft Access 输出（Microsoft Access Output）转换允许你在管道中创建一个新的 Access 数据库文件作为输出。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 文件名 | 输出的 Access 数据库文件名（.mdb 或 .accdb） |
| 目标表 | 要创建和写入的表名 |
| 字段 | 输出字段定义 |

## 注意事项

- 该转换创建新的 Access 数据库文件。
- 适用于将数据导出到 Microsoft Access 数据库格式。
