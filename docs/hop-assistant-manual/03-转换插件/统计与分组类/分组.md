# 分组

分组（Group By）转换根据指定字段或字段集合对来自源的行进行分组。每个组生成一个新行。

它还可以为各组生成一个或多个聚合值。

常见用途包括计算每个产品的平均销售额和统计库存中某物品的数量。

分组转换专为已排序的输入设计。如果你的输入未排序，则只有连续的重复行会被正确分组。如果你在 Hop 之外排序数据，字段中数据的大小写敏感性可能会产生意外的分组结果。

你可以使用内存分组（Memory Group By）转换来处理未排序的输入。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 分组字段 | 用于分组的字段 |
| 聚合 | 各组的聚合定义（字段名、Subject、类型如 SUM、AVG、COUNT 等） |
| 包含所有行 | 是否在输出中包含所有行 |
| 临时目录 | 临时文件目录（用于大数据集） |

## 注意事项

- 该转换要求输入数据已按分组字段排序，否则只有连续的重复行会被正确分组。
- 如输入未排序，请使用"内存分组"转换。
- 在 Hop 外部排序时需注意大小写敏感性对分组结果的影响。
