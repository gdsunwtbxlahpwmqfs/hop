# 唯一行

唯一行（Unique Rows）转换从（已排序的）输入流中删除重复行。

对于未排序的输入流，请查看唯一行（HashSet）（Unique Rows By Hashset）转换。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 用于比较的字段 | 用于判断重复的字段（留空表示使用所有字段） |
| 计数器字段 | 输出中包含重复计数的字段名（可选） |

## 注意事项

- 该转换仅支持 Hop Engine，不支持 Spark、Flink、Dataflow。
- 要求输入流已排序，因为仅比较连续行来检测重复。
- 对于未排序输入，请使用唯一行（HashSet）转换。
