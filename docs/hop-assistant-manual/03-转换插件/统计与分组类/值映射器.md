# 值映射器

值映射器（Value Mapper）转换将字符串值从一个值映射到另一个值。

映射通常通过将转换表存储在数据库中来解决。值映射器为你提供了一个简单的替代方案。

例如，如果你想替换语言代码：

```bash
Fieldname to use: LanguageCode
Target fieldname: LanguageDesc
Source/Target: EN/English, FR/French, NL/Dutch, ES/Spanish, DE/German, ...
```

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 使用的字段名 | 要映射的源字段名 |
| 目标字段名 | 输出中包含映射值的字段名 |
| 字段值 | 源值到目标值的映射列表 |
| 默认值 | 未匹配任何映射时的默认值 |

## 注意事项

- 该转换提供简单的值映射功能，无需数据库查询。
- 适用于代码到描述的转换，如语言代码、状态代码等。
- 可设置默认值处理未匹配的情况。
