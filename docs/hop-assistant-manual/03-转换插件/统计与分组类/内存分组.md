# 内存分组

## 功能概述


内存分组（Memory Group By）转换以分组方式构建聚合。
该转换在内存中处理所有行，因此不需要排序的输入。但是，它**确实**要求所有数据都能放入内存。
> **提示**：当行数太大无法放入内存时，请使用排序行（Sort Rows）和分组（Group By）转换的组合。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 分组字段 | 用于分组的字段 |
| 聚合 | 各组的聚合定义（字段名、Subject、类型如 SUM、AVG、COUNT 等） |

## 注意事项

- 该转换不需要输入数据排序，因为它在内存中处理所有行。
- 所有数据必须能放入内存；数据量过大时请改用排序行 + 分组的组合。
- 适用于中小规模数据的内存聚合。
