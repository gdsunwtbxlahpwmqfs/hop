# 行反规范化

行反规范化（Row Denormaliser）转换允许你通过查找键值对来反规范化数据，并可选择在过程中转换数据类型。

> **注意**：请务必查看 Beam 入门文档中关于此转换不受支持的说明。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 分组字段 | 用于分组的键字段 |
| 目标字段 | 包含目标字段名的字段 |
| 值字段 | 包含值的字段 |
| 反规范化字段 | 反规范化字段定义（数据类型、格式等） |

## 注意事项

- 该转换将规范化的行数据（键值对形式）转换为反规范化的列形式（透视）。
- 是行规范化（Row Normaliser）的逆操作。
- 在 Beam 引擎中使用时请注意其限制说明。
