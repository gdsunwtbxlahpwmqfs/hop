# 合并行（差异比较）

合并行（差异比较）（Merge Rows (Diff)）转换比较并合并两个数据流（称为**参考（Reference）**（先前数据）和**比较（Compare）**（新数据））之间的行，为每行添加一个标志。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 参考流（原数据） | 包含原始参考数据的输入流 |
| 比较流（新数据） | 包含新比较数据的输入流 |
| 键字段 | 用于匹配行的键字段 |
| 值字段 | 要比较的值字段 |
| 标志字段名 | 输出中标识比较结果的标志字段名 |

## 注意事项

- 该转换比较两个数据流，添加标志字段标识行状态。
- 标志值包括："identical"（相同）、"changed"（已更改）、"new"（新增）、"deleted"（已删除）。
- 输入流必须按键字段排序。
- 通常与"合并后同步（Synchronize after merge）"转换配合使用。
