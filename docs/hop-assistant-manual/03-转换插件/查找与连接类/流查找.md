# 流查找

流查找（Stream Lookup）转换允许你使用来自管道中其他转换的信息来查找数据。

来自源转换的数据首先被读入内存，然后用于从主流中查找数据。

> **提示**：由于此转换将查找数据加载到内存中，它可能是一种极快的数据查找方式。但是，整个查找数据集需要能够放入可用内存中。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 查找转换 | 提供查找数据的源转换 |
| 查找键 | 用于查找的键字段映射 |
| 输出字段 | 要返回的查找值字段 |

## 注意事项

- 该转换支持 Hop Engine、Spark、Flink、Dataflow 等多种引擎。
- 查找数据全部加载到内存中，速度极快，但整个数据集需能放入内存。
- 适用于中小规模数据的内存查找。
