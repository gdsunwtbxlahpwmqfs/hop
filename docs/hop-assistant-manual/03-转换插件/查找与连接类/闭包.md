# 闭包

## 功能概述


闭包（Closure）转换允许你为 Mondrian OLAP 引擎生成自反传递闭包表。
有关闭包表如何帮助 Mondrian 提升性能的更多信息，请参见相关技术文档。从技术上讲，此转换将所有输入行读入内存，并计算所有可能的父子关系。它附加了从父到子的距离（以层级计）。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 父字段名 | 包含父级 ID 的字段 |
| 子字段名 | 包含子级 ID 的字段 |
| 距离字段名 | 输出中包含父子距离的字段名 |
| 根节点字段名 | 标识是否为根节点的字段名 |

## 注意事项

- 该转换将所有输入数据读入内存计算，适用于中小规模层级数据。
- 生成的闭包表用于优化 Mondrian OLAP 引擎的层级查询性能。
- 计算所有父子关系及其层级距离。
