# 合并后同步

合并后同步（Synchronize After Merge）转换可与合并行（差异比较）管道转换结合使用，根据合并行（差异比较）的输出对数据库字段执行插入/更新/删除操作。

合并行（差异比较）管道转换为每行附加一个标志列，值为"identical"（相同）、"changed"（已更改）、"new"（新增）或"deleted"（已删除）。

然后，合并后同步管道转换使用此标志列对连接表执行更新/插入/删除操作。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 数据库连接 | 目标数据库连接 |
| 目标表 | 要同步的目标表名 |
| 操作字段 | 定义操作的标志字段 |
| 键字段 | 用于匹配行的键字段 |
| 值字段 | 要同步的值字段 |

## 注意事项

- 该转换通常与"合并行（差异比较）"转换配合使用。
- 根据标志列的值自动执行相应的插入、更新或删除操作。
- 适用于数据同步和增量更新场景。
