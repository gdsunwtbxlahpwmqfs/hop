# 合并连接

合并连接（Merge Join）转换在来自两个不同输入转换的数据集之间执行经典的合并连接。

该转换假设你的数据已按连接键排序。如有必要，在传入流上使用排序行（Sort Rows）转换来强制排序。

连接选项包括 INNER（内连接）、LEFT OUTER（左外连接）、RIGHT OUTER（右外连接）和 FULL OUTER（全外连接）。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 第一个输入 | 第一个输入流 |
| 第二个输入 | 第二个输入流 |
| 连接类型 | INNER、LEFT OUTER、RIGHT OUTER、FULL OUTER |
| 键字段 | 两个流的连接键字段映射 |

## 注意事项

- 该转换要求输入数据已按连接键排序。
- 支持四种标准连接类型：内连接、左外、右外、全外。
- 如输入未排序，需在合并连接前使用排序行转换。
