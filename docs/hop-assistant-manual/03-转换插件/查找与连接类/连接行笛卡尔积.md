# 连接行（笛卡尔积）

## 功能概述


连接行（笛卡尔积）（Join Rows / cartesian product）转换允许你组合/连接多个输入流（[笛卡尔积](https://en.wikipedia.org/wiki/Cartesian_product)），而无需按键连接。它最适合每个流一行的情况。你可以添加条件，仅在满足条件时才连接。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 输入流 | 要连接的输入流 |
| 连接条件 | 仅在满足条件时才连接的条件表达式 |

## 注意事项

- 该转换生成笛卡尔积，行数可能呈指数增长。
- 最适合每个输入流仅一行的情况。
- 可添加连接条件以限制组合数量。
- 适用于需要生成所有可能组合的场景。
