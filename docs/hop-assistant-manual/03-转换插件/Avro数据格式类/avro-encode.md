## Options

| Option | Description |
|---|---|
| Transform name |  |
| Transform 的名称。 |  |
| Output field name |  |
| 为 Avro Record 输出字段选择一个名称。 |  |
| Schema name |  |
| 输出 Avro record 中包含的 schema 名称 |  |
| Namespace |  |
| 可选的 schema 命名空间 |  |
| Documentation |  |
| 可选的 schema 文档（doc）元素 |  |
| The fields to encode in a generic Avro record: |  |
| 可以指定要包含在输出 Avro record 字段中的 Hop 输入字段名称。 |  |
| Get fields button |  |
| 可以使用 "Get fields" 按钮检索要包含在 Avro record 中的字段。 |  |
