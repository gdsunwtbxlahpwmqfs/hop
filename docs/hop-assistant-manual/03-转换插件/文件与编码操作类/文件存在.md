# 文件存在（File exists）

文件存在转换用于验证文件是否存在。要检查的文件名从管道字段中读取。

该转换输出一个布尔标志字段，指示文件是否存在。

## 主要选项

| 选项 | 说明 |
|------|------|
| 文件名字段（Filename field） | 包含要检查的文件名的输入字段 |
| 结果字段名（Result fieldname） | 输出布尔结果（true/false）的字段名称，指示文件是否存在 |
| 添加文件名（Add filename to result） | 是否将检查的文件名添加到结果文件列表中 |

## 注意事项

- 支持 Hop Engine 引擎；Spark、Flink、Dataflow 引擎支持情况视具体实现而定。
- 适用于在数据处理前验证源文件是否存在。
