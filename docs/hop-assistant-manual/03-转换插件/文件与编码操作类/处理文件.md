# 处理文件（Process Files）

处理文件转换通过在数据流中提供源文件名和目标文件名，来复制、移动或删除文件。

## 主要选项

| 选项 | 说明 |
|------|------|
| 源文件名字段（Source filename field） | 包含源文件名的输入字段 |
| 目标文件名字段（Target filename field） | 包含目标文件名的输入字段（复制或移动操作时使用） |
| 操作类型（Process action） | 要执行的操作：复制（Copy）、移动（Move）或删除（Delete） |
| 如果目标文件存在（If file exists） | 当目标文件已存在时的处理方式 |
| 结果字段名（Result fieldname） | 存放操作结果的输出字段名称 |

## 注意事项

- 支持 Hop Engine 引擎；Spark、Flink、Dataflow 引擎支持情况视具体实现而定。
- 适用于在数据处理过程中对文件进行批量复制、移动或删除操作。
