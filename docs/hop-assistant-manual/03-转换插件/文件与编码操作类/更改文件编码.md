# 更改文件编码（Change file encoding）

## 功能概述


更改文件编码转换用于将文本文件从一种编码更改为另一种编码。

## 主要选项

| 选项 | 说明 |
|------|------|
| 源文件名字段（Source filename field） | 包含源文件名的输入字段 |
| 目标文件名字段（Target filename field） | 包含目标文件名的输入字段 |
| 源编码（Source encoding） | 源文件的字符编码（如 UTF-8、GBK 等） |
| 目标编码（Target encoding） | 目标文件的字符编码（如 UTF-8、GBK 等） |
| 如果目标文件存在（If target file exists） | 当目标文件已存在时的处理方式 |

## 注意事项

- 支持 Qi 数据治理平台 Engine 引擎；Spark、Flink、Dataflow 引擎支持情况视具体实现而定。
- 适用于处理不同字符编码的文件转换场景，例如将 GBK 编码文件转换为 UTF-8 编码。
