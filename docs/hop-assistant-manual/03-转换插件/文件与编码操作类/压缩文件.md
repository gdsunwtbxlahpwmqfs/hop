# 压缩文件（Zip file）

## 功能概述


压缩文件转换使用您在对话框中指定的选项创建标准的 ZIP 归档文件。

## 主要选项

| 选项 | 说明 |
|------|------|
| 源文件名字段（Source filename field） | 包含要压缩的源文件名的输入字段 |
| 目标 ZIP 文件名（Zip file） | 要创建的 ZIP 归档文件名 |
| 如果目标文件存在（If zip file exists） | 当 ZIP 文件已存在时的处理方式 |
| 压缩比率（Compression rate） | 压缩比率设置 |
| 结果字段名（Result fieldname） | 存放操作结果的输出字段名称 |

## 注意事项

- 支持 Qi 数据治理平台 Engine 引擎；Spark、Flink、Dataflow 引擎支持情况视具体实现而定。
- 适用于在数据处理过程中将文件打包为 ZIP 归档。
