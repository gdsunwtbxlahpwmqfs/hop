# 检查文件是否锁定（Check if file is locked）

## 功能概述


检查文件是否锁定转换通过尝试将文件移动到自身，来判断文件是否被其他进程锁定。

## 主要选项

| 选项 | 说明 |
|------|------|
| 文件名字段（Filename field） | 包含要检查的文件名的输入字段 |
| 结果字段名（Result fieldname） | 输出布尔结果（true/false）的字段名称，指示文件是否被锁定 |

## 注意事项

- 该转换通过尝试将文件移动到自身来检测锁定状态。
- 支持 Qi 数据治理平台 Engine 引擎；Spark、Flink、Dataflow 引擎支持情况视具体实现而定。
- 适用于在处理文件前确认文件未被其他进程占用。
