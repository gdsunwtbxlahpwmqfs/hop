# 服务器状态（Get Server Status）

## 功能概述


获取服务器状态转换用于联系 Hop 服务器，服务器名称从输入字段中读取。
它返回关于服务器的高层信息。您指定名称的每个输出字段都将被添加到输出字段列表中。

## 主要选项

| 选项 | 说明 |
|------|------|
| 服务器字段（Server field） | 包含要查询的服务器名称的输入字段 |
| 服务器状态字段（Server status field） | 输出服务器状态信息的字段名称 |
| 其他输出字段（Output fields） | 指定要获取的其他服务器信息字段名称 |

## 注意事项

- 支持 Qi 数据治理平台 Engine 引擎；Spark、Flink、Dataflow 引擎支持情况视具体实现而定。
- 适用于监控和管理 Hop 服务器的运行状态。
