# 简单映射（Simple Mapping）

## 功能概述


简单映射转换允许您以"可重用管道"（也称为"映射" Mapping）的形式重用一系列转换。
可以将其想象为编程中的可重用函数——您将一些数据传入函数，函数会返回数据。

## 主要选项

| 选项 | 说明 |
|------|------|
| 映射管道（Mapping pipeline） | 要作为映射调用的子管道（Mapping） |
| 输入字段映射（Input mapping） | 将父管道的字段映射到映射管道的输入字段 |
| 输出字段映射（Output mapping） | 将映射管道的输出字段映射回父管道的字段 |
| 参数（Parameters） | 传递给映射管道的参数 |

## 注意事项

- 支持 Qi 数据治理平台 Engine、Spark、Flink、Dataflow 全部引擎。
- 适用于将通用数据处理逻辑封装为可重用的子管道。
