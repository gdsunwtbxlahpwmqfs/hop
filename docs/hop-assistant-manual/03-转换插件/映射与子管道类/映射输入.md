# 映射输入（Mapping Input）

映射输入转换用于表示和描述映射管道（Mapping）的输入。

此输入的数据将由父管道（调用映射的管道）提供。映射输入转换通常放置在映射管道的开头，作为数据流的入口点。

## 主要选项

| 选项 | 说明 |
|------|------|
| 字段（Field name） | 映射管道中定义的输入字段名称 |
| 类型（Type） | 输入字段的数据类型 |
| 长度（Length） | 输入字段的长度 |
| 精度（Precision） | 输入字段的精度 |

## 注意事项

- 支持 Hop Engine、Spark、Flink、Dataflow 全部引擎。
- 此转换仅在作为映射（Mapping）的子管道中使用，由父管道提供输入数据。
