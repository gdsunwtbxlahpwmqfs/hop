# 元数据注入（Metadata Injection）

ETL 元数据注入转换允许在运行时将来自各种数据源的数据插入到管道中。

元数据注入允许 Apache Hop 用户在运行时为管道提供所需的元数据。这可以显著减少开发基本相同但略有差异的管道版本的需求。

## 主要选项

| 选项 | 说明 |
|------|------|
| 目标管道（Target pipeline） | 接收注入元数据的目标管道（模板管道） |
| 字段映射（Fields to inject） | 定义源数据字段与目标管道元数据项之间的映射关系 |
| 输出字段（Output fields） | 注入结果的输出字段 |

## 注意事项

- 支持 Hop Engine 引擎；不支持 Spark、Flink、Dataflow 引擎。
- 适用于构建通用化的模板管道，通过运行时注入元数据来处理不同的数据结构，避免重复开发。
