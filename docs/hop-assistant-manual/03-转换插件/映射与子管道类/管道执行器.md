# 管道执行器（Pipeline Executor）

管道执行器转换允许您在管道中执行另一个 Hop 管道。它类似于工作流执行器（Workflow Executor）转换，但用于执行管道。

## 主要选项

| 选项 | 说明 |
|------|------|
| 运行的管道（Running pipeline） | 要执行的子管道 |
| 输入字段映射（Input fields） | 传递给子管道的输入行字段 |
| 参数（Parameters） | 传递给子管道的参数 |
| 变量（Variables） | 传递给子管道的变量 |
| 输出字段（Output fields） | 子管道执行结果的输出字段 |

## 注意事项

- 支持 Hop Engine 引擎；不支持 Spark 引擎；Flink、Dataflow 引擎支持情况视具体实现而定。
- 适用于在管道中嵌套执行其他管道，实现复杂的处理逻辑复用。
