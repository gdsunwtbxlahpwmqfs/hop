# 映射输出（Mapping Output）

## 功能概述


映射输出转换用于描述映射管道向父管道（调用映射的管道）的输出。
进入此转换的数据将被流式传输到父管道。映射输出转换通常放置在映射管道的末尾，作为数据流的出口点。

## 主要选项

| 选项 | 说明 |
|------|------|
| 字段（Field name） | 映射管道中定义的输出字段名称 |
| 类型（Type） | 输出字段的数据类型 |
| 长度（Length） | 输出字段的长度 |
| 精度（Precision） | 输出字段的精度 |

## 注意事项

- 支持 Qi 数据治理平台 Engine、Spark、Flink、Dataflow 全部引擎。
- 此转换仅在作为映射（Mapping）的子管道中使用，将处理结果输出到父管道。
