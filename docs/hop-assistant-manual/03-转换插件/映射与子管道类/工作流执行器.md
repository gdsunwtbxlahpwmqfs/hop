# 工作流执行器（Workflow Executor）

工作流执行器转换允许您在管道中执行一个 Hop 工作流（workflow）。

## 主要选项

| 选项 | 说明 |
|------|------|
| 运行的工作流（Running workflow） | 要执行的子工作流 |
| 输入字段映射（Input fields） | 传递给子工作流的输入行字段 |
| 参数（Parameters） | 传递给子工作流的参数 |
| 变量（Variables） | 传递给子工作流的变量 |
| 执行结果字段（Execution results field） | 存放工作流执行结果状态（如成功/失败）的输出字段 |

## 注意事项

- 支持 Hop Engine 引擎；Spark、Flink、Dataflow 引擎支持情况视具体实现而定。
- 适用于在管道数据处理过程中触发工作流编排逻辑。
