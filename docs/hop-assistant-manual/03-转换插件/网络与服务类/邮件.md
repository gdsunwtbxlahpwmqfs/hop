# 邮件（Mail）

## 功能概述


邮件转换使用 SMTP 服务器，利用管道字段中的数据发送电子邮件。

## 支持的引擎

| 引擎 | 是否支持 |
|------|---------|
| Qi 数据治理平台 Engine | ✅ 支持 |
| Spark | ❓ 可能支持 |
| Flink | ❓ 可能支持 |
| Dataflow | ❓ 可能支持 |
