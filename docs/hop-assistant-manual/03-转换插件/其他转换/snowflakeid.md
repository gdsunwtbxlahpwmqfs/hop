## 选项

| 选项 | 描述 |
|---|---|
| Transform name | 此 transform 在 Pipeline 工作区中显示的名称。 |
| Name of value | 添加到流中的新 Snowflake 值的名称。 |
| Data Center No | 数据中心或逻辑组的标识符。有效范围：0–31（默认：1）。 |
| Machine No | 生成 ID 的机器标识符。有效范围：0–31（默认：1）。 |
