## 选项

| Option | Description |
|---|---|
| Transform name | 唯一标识该 Transform 的名称。 |
| Parent ID field | 包含父子关系中父 ID 的字段名。 |
| Child ID field | 包含父子关系中子 ID 的字段名。 |
| Distance field name | 将添加到输出的距离字段的名称 |
| Root is zero | 如果父子树的根不是空（null）而是零（0），请勾选此框 |
