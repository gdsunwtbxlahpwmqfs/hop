## 选项

| 选项 | 描述 |
|---|---|
| Transform Name | Transform 的名称；此名称在单个 Pipeline 中必须唯一。 |
| Filename | 要生成的 Hop cube 文件的名称。 |
| Limit | 允许您限制写入 |
| Size | cube 文件的行数。 |
