## 选项

| Option | Description |
|---|---|
| Transform name | Transform 的名称。 |
| Fieldname | 指定要注入的行的字段名。 |
| Type | 指定数据类型。 |
| Length | 对于 Number：数字中的有效数字总位数；对于 String：字符串总长度；对于 Date：字符串打印输出的长度。 |
| Precision | 对于 Number：浮点位数；对于 String、Date、Boolean：不使用。 |
