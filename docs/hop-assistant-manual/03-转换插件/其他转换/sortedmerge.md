## 选项

| 选项 | 描述 |
|---|---|
| Transform name | Transform 的名称，此名称在单个 Pipeline 中必须唯一。 |
| Fields table | 指定字段名称和排序方向（升序/降序）；点击 Get Fields 从输入流中检索字段列表。 |
