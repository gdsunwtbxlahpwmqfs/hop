## 选项

| 选项 | 描述 |
|---|---|
| Transform name | Transform 的名称，此名称在单个 Pipeline 中必须唯一。 |
| Sample size | 选择从输入流中采样的行数。 |
| Random seed | 为随机数生成器选择一个种子。 |
