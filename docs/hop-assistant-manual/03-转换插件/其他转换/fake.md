## 选项

| Option | Description |
|---|---|
| Transform name | Transform 的名称。 |
| Locale | 可使用 locale 使生成的数据更具体地对应某一地区。 |
| Output field name | 生成字段的名称。 |
| Type of fake data | 数据类别。 |
| Topic to generate | 在数据类别中要生成的主题。 |
