## 选项

| Option | Description |
|---|---|
| Transform name | Transform 的名称。 |
| Output row count | 在输出中包含此 Transform 消费的行数 |
| Field for row count | 包含消费行数的字段名称 |
| Include position field? | 在 Transform 的输出中包含字段位置 |
| Field for position | 包含位置信息的字段名称 |
| Include field name field? | 在 Transform 的输出中包含字段名称 |
| Field for field name | 包含字段名称的字段名称 |
| Include comment field? | 在 Transform 的输出中包含字段注释 |
| Field for comments | 包含注释的字段名称 |
| Include type field? | 在 Transform 的输出中包含字段数据类型 |
| Field for type | 包含数据类型的字段名称 |
| Include format mask field? | 在 Transform 的输出中包含用于数据的格式掩码字段 |
| Field for format mask | 包含格式掩码的字段名称 |
| Include length field? | 在 Transform 的输出中包含字段长度 |
| Field for length | 包含数据长度的字段名称 |
| Include precision field? | 在 Transform 的输出中包含字段精度 |
| Field for precision | 包含精度（如果值为数值类型）的字段名称 |
| Include origin field? | 在 Transform 的输出中包含字段来源 |
| Field for origin | 包含字段来源的字段名称 |
