## 选项

| 选项 | 描述 |
|---|---|
| Transform name | 此 Transform 在 Pipeline 中的名称 |
| startOfPipelineDelta | 给定 Pipeline 的日期范围开始时间 |
| endOfPipelineDelta | 给定 Pipeline 的数据范围结束时间 |
| startOfWorkflowDelta | 给定 Workflow 的日期范围开始时间 |
| endOfWorkflowDelta | 给定 Workflow 的日期范围结束时间 |
