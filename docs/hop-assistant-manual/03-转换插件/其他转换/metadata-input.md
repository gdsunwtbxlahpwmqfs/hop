## 选项

您可以指定所需信息的字段名称，也可以不指定。

您还可以指定一个或多个类型过滤器来限制 Transform 的输出。

例如，您可以选择过滤器 'rdbms' 以在输出中仅查看关系数据库连接 metadata。
