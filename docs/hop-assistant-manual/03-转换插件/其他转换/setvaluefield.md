## 选项

| 选项 | 描述 |
|---|---|
| Transform name | Transform 的名称，此名称在单个 Pipeline 中必须唯一。 |
| Field name | 选择要设置的字段（要替换的值） |
| Replace by value from field | 选择要从中复制值的字段 |
