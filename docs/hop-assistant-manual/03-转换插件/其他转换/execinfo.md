## 选项

| Option | Description |
|---|---|
| Transform name |  |
| Transform 的名称；该名称在单个 Pipeline 中必须唯一 |  |
| Execution Information Location |  |
| 选择你要使用的 [位置](metadata-types/execution-information-location.md)。 |  |
| Operation type |  |
| 选择你要执行的操作。 |  |
| Execution ID field |  |
| 将包含执行 ID 的输入字段 |  |
| Execution parent ID field |  |
| 将包含执行父 ID 的输入字段 |  |
| Execution name field |  |
| 将包含要查找的执行名称的输入字段 |  |
| Execution type field |  |
| 将包含要查找的执行类型的输入字段 |  |
| Include children field |  |
| 将包含布尔值的输入字段，用于指示是否在搜索中包含子项 |  |
| Limit field |  |
| 将包含搜索限制的输入字段 |  |
