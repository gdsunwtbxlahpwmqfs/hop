## 选项

| Option | Description |
|---|---|
| Transform name | Transform 的名称。 |
| URL field | 指定数据流中的 URL 字段名。 |
| Connect timeout (ms) | 连接超时时间（毫秒）。 |
| Read timeout (ms) | 连接后，Transform 尝试读取数据的超时时间。 |
| Result fieldname | 如果在给定超时时间内连接成功并且可以读取数据，则返回布尔值结果 'true'，否则返回 'false' |
