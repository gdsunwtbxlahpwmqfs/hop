## 选项

| 选项 | 描述 |
|---|---|
| Transform name | Transform 的名称。 |
| GPG location | GPG 密钥位置。 |
| Browse... | 浏览文件系统查找 GPG 密钥位置。 |
| Passphrase | 密钥的密码短语。 |
| Read passphrase from field | 是否从字段中读取密码短语。 |
| Passphrase fieldname | 密码短语字段。 |
| Data fieldname | 要解密的字段。 |
| Result fieldname | 解密后的结果字段。 |
