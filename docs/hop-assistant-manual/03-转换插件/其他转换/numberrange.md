## 选项

| 选项 | 描述 |
|---|---|
| Transform name | Transform 的名称，此名称在单个 Pipeline 中必须唯一。 |
| Input field | 指定要检查下限和上限边界的字段。 |
| Output field | 指定根据输入字段值填充的输出字段名称。 |
| Default value | 当指定范围内没有匹配项时返回的值。 |
| Ranges | 指定范围的下限和上限。 |
| Lower Bound | 指定范围的最小值。 |
| Upper Bound | 指定范围的上限值。 |
| Value | 指定值的名称。 |
