## 选项

| Option | Description |
|---|---|
| Transform name | 此 Transform 的名称 |
| Test name input field | 用于获取单元测试名称的字段名，以确定要执行哪些 Transform。 |
| Type of tests to run | 开发测试或单元测试 |
| Pipeline output field name | 被测试的 Pipeline |
| Unit test output field name | 被执行的单元测试 |
| Data set output field name | 用于此测试的（黄金）数据集 |
| transform output field name | 此测试中使用的 Transform |
| Error output field name | 错误状态（Y 或 N） |
| Comment output field name | 测试返回的注释或错误消息 |
