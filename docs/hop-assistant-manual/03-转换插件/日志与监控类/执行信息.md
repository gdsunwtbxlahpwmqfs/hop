# 执行信息（Execution Information）

## 功能概述


Execution Information 转换允许您从执行信息位置（Execution Information Location）读取或删除信息。
根据所选的操作不同，会启用不同的输入选项，转换的输出也会有所不同。请注意，该转换始终需要输入行才能产生输出。
例如，要删除 200 个执行 ID：首先使用 Generate rows（生成行）创建所需的字段和值（children: Boolean = true）和（limit: Integer = 200），然后使用第一个转换获取 ID（操作：Get execution IDs），再连接第二个转换执行删除（操作：Delete execution）。您也可以按日期和 ID 等其他字段进行删除。查询执行信息可能耗时较长，请关注 Duration（耗时）列。

## 主要选项

| 选项 | 说明 |
|------|------|
| 执行信息位置 | 指定要读取或删除信息的 Execution Information Location 元数据项 |
| 操作 | 选择要执行的操作：获取执行 ID、获取执行信息、删除执行等 |
| 限制 | 指定返回或删除的记录数量上限 |
| 是否包含子项 | 是否包含子执行（如子管道、子工作流）的信息 |
