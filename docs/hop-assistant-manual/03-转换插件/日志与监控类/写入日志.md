# 写入日志（Write to log）

Write To Log 转换用于将信息写入 Hop 的日志系统。

典型的使用场景是将特定字段值或自定义的日志消息记录到 Hop 日志中。该转换便于在管道执行过程中进行调试、追踪或记录关键节点的信息，是排查问题和监控执行流程的常用工具。

## 主要选项

| 选项 | 说明 |
|------|------|
| 日志级别 | 设置日志消息的级别（如 Error、Minimal、Basic、Detailed、Debug、Errorlevel 等） |
| 日志消息 | 指定要写入日志的消息内容，支持使用变量和字段引用 |
| 打印标题 | 是否在日志中打印字段标题 |
| 限制输出 | 限制写入日志的行数 |
