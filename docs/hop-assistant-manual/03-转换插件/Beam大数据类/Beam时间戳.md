# Beam 时间戳（Beam Timestamp）

Beam 时间戳转换使用 Beam 执行引擎为数据添加自定义时间戳。该转换专为在 Beam 引擎上运行的管道设计。

## 支持的引擎

| 引擎 | 是否支持 |
|------|---------|
| Hop Engine | ❌ 不支持 |
| Spark | ✅ 支持 |
| Flink | ✅ 支持 |
| Dataflow | ✅ 支持 |

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称（Transform name） | 转换的名称，在单个管道中必须唯一。 |
| 使用的时间字段（为空时使用系统时间） | 包含时间戳的字段，为空时使用系统时间。 |
| 是否从流中获取时间戳？（Get timestamp from stream?） | 如果在流中定义了事件时间字段，则返回该事件时间字段。 |
