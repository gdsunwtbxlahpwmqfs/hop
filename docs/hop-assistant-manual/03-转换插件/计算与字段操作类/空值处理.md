# 空值处理（Null If）

空值处理转换用于在字段值等于本转换中指定的值时，将该字段的值设置为空值（null）。

可以使用"获取字段（Get Fields）"按钮添加输入流中的所有字段。

## 主要选项

| 选项 | 说明 |
|------|------|
| 字段名（Field name） | 需要进行空值判断的输入字段名称 |
| 值（Value） | 当字段值等于此指定值时，将该字段值设置为 null |

## 注意事项

- 支持 Hop Engine、Spark、Flink、Dataflow 引擎。
- 适用于数据清洗场景，例如将表示空值的占位符（如 "N/A"、"NULL"、0 等）转换为真正的空值。
