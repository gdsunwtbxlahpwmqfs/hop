# 计算器（Calculator）

计算器转换提供了一组预定义函数，可对输入字段值执行计算。除了参数（Field A、Field B 和 Field C）之外，还必须指定函数的返回类型。

可以选择使用 `Remove` 选项在所有值计算完成后从结果（输出）中移除字段。这在使用了不需要出现在最终管道字段中的临时值时非常有用。

## 主要选项

| 列名 | 说明 |
|------|------|
| 新字段（New field） | 计算结果的输出字段。对于计算后续字段所需但最终结果中不需要的临时字段，将 `Remove` 设为 `Y` |
| 计算（Calculation） | 要执行的计算类型 |
| 字段 A/B/C（Field A/B/C） | 计算的输入参数 |
| 值类型（Value type） | 函数的返回类型 |
| 长度/精度（Length/Precision） | 结果字段的长度和精度 |
| 移除（Remove） | 设为 Y 可从输出中移除该临时字段 |

## 注意事项

- 计算器的执行速度远优于自定义脚本（JavaScript）提供的速度。
- 支持 Hop Engine、Spark、Flink、Dataflow 全部引擎。
