# 添加序列（Add Sequence）

添加序列转换向 Hop 数据流添加一个序列。序列是一个不断变化的整数值，具有特定的起始值和增量值。

可以使用数据库序列（如受支持的 Oracle、PostgreSQL）来确定序列的值，也可以由 Hop 生成。Hop 生成的序列值在同一管道中使用时唯一，每次运行管道时返回相同的起始值。Hop 序列仅在同一管道中使用时唯一，且不会被存储，因此每次启动管道时值都会回到相同的起始值。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称（Transform name） | 转换在管道工作区中显示的名称，在单个管道中必须唯一 |
| 值名称（Name of value） | 添加到数据流的新序列值的名称 |
| 使用数据库生成序列（Use DB to generate the sequence） | 启用以使用数据库序列驱动，然后设置：连接名、模式名（可选）、序列名 |
| 连接名（Connection name） | 数据库序列所在的连接名称 |
| 使用计数器（Use counter） | 启用以使用 Hop 内置计数器 |
| 起始值（Start at value） | 序列的起始值 |
| 增量（Increment by） | 每行的增量值 |
| 最大值（Maximum value） | 序列的最大值 |

## 注意事项

- 仅 Hop Engine 支持，Spark、Flink、Dataflow 引擎不支持。
- Hop 序列值不会被持久化存储，每次运行管道都从起始值重新开始。
