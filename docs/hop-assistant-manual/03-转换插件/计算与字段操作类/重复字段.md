# 重复字段（Repeat Fields）

## 功能概述


重复字段转换用于解决以下常见问题：
- 当当前行的值为空时，从前一行重复某个字段值：这种情况通常发生在读取电子表格中合并单元格的数据时。在该场景下，合并单元格仅第一行被填充，需要从单元格的首行重复数据。
- 当指示符匹配时取当前行的字段值，否则取前一行的字段值。这可用于在拥有共同分组键时，合并来自多个数据源的数据。

## 主要选项

| 选项 | 说明 |
|------|------|
| 分组字段（Group fields） | 用于分组的字段，转换将在分组内重复字段值 |
| 重复字段（Repeat fields） | 需要在当前行为空时从前一行重复的字段 |
| 指示符字段（Indicator field） | 用于判断是否取当前行值的指示符字段 |

## 注意事项

- 重要：如果输入数据未按指定的分组字段排序，输出结果可能不正确。建议在管道内对输入数据进行排序。
- 支持 Qi 数据治理平台 Engine 引擎。
