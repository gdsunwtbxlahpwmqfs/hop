# 添加字段变化序列（Add Fields Changing Sequence）

## 功能概述


添加字段变化序列转换用于添加一个序列值，该序列值在指定字段列表中的值每次发生变化时重置。适用于按分组生成组内序号的场景。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称（Transform name） | 转换的名称，在管道中必须唯一 |
| 结果字段（Result field） | 输出字段（序列）的名称 |
| 起始值（Start at value） | 每次重置时开始的数字 |
| 增量（Increment by） | 同一组内每行增加的值 |
| 字段值变化时初始化序列（Init sequence if value of following fields change） | 可在此指定字段列表。当这些字段中的一个或多个的值与前几行相比发生变化时，序列将重置为起始值 |

## 注意事项

- 仅 Qi 数据治理平台 Engine 支持，Spark、Flink、Dataflow 引擎不支持。
- 输入数据应按指定字段排序，以保证分组序列的正确性。
