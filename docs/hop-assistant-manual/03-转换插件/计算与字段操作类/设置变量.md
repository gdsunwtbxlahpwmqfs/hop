# 设置变量（Set Variables）

设置变量转换允许您在工作流（workflow）或虚拟机（JVM）中设置变量。可以通过字段值来设置变量，点击"获取字段（Get Fields）"按钮会自动填充数据行中的所有字段，并以全大写形式生成变量名称。

## 主要选项

| 选项 | 说明 |
|------|------|
| 字段名（Field name） | 提供变量值的字段名称 |
| 变量名（Variable name） | 要设置的变量名称，建议使用全大写命名 |
| 作用范围（Variable scope type） | 变量的有效范围，包括：JVM（虚拟机级别）、Parent workflow（父工作流）、Grand parent workflow（祖父工作流）、Root workflow（根工作流） |
| 默认值（Default value） | 变量的默认值 |

## 注意事项

- 此转换仅接受一行（且仅一行）数据来设置变量的值，因此需要为此转换隔离出单行数据，一种简单的方法是使用"阻塞转换（Blocking transform）"。
- 支持 Hop Engine 引擎；Spark、Flink、Dataflow 引擎支持情况视具体实现而定。
