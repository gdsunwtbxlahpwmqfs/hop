# 空值判断（If Null）

## 功能概述


空值判断转换用于将空值（null）替换为指定的值。可以通过以下三种方式进行处理：
1. 处理包含所有字段的完整行
2. 处理完整行，但仅针对特定字段类型（Number、String、Date 等）
3. 处理完整行，但仅针对按名称指定的特定字段

## 主要选项

| 选项 | 说明 |
|------|------|
| 字段类型（Select field type） | 选择要处理的字段类型（Number、String、Date 等），用于按类型替换空值 |
| 字段（Fields） | 按名称指定要处理的字段，用于针对特定字段替换空值 |
| 替换值（Replace by value） | 用于替换空值的目标值 |
| 替换值的类型（Replace by value type） | 替换值的数据类型 |

## 注意事项

- 支持 Qi 数据治理平台 Engine、Spark、Flink、Dataflow 引擎。
- 适用于数据补全场景，例如将缺失的数值字段填充为 0 或将缺失的字符串字段填充为默认值。
