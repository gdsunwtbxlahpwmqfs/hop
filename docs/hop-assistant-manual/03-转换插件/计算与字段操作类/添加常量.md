# 添加常量（Add Constants）

## 功能概述


添加常量值转换是一种简单且高性能的方式，用于向管道数据流添加常量值。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称（Transform name） | 转换的名称。注意：该名称在单个管道中必须唯一 |
| 字段（Fields） | 以字符串形式指定名称、类型和值，然后指定将值转换为所选数据类型的格式 |
