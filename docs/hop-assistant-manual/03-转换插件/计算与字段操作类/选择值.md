# 选择值（Select Values）

选择值转换用于从输入流中选择、移除、重命名字段，更改数据类型，以及配置字段的长度和精度。它是管道中最常用的字段处理转换之一。

这些操作被组织为以下不同类别：

- **选择与修改（Select & Alter）** —— 指定字段在输出行中排列的确切顺序和名称
- **移除（Remove）** —— 指定需要从输出行中移除的字段
- **元数据（Meta-data）** —— 更改一个或多个字段的名称、类型、长度和精度（即元数据）

## 主要选项

### 选择与修改（Select & Alter）选项卡

| 选项 | 说明 |
|------|------|
| 字段名（Fieldname） | 来自输入流的字段名称 |
| 重命名为（Rename to） | 字段的新名称，若不想重命名则留空 |
| 长度（Length） | 字段长度 |
| 精度（Precision） | 精度选项取决于字段类型，仅 Number 类型支持，表示小数位数 |
| 包含未指定的字段（Include unspecified fields, ordered by name） | 启用后将隐式选择输入流中未在 Fields 区域显式选择的所有其他字段 |

### 元数据（Meta-data）选项卡

| 选项 | 说明 |
|------|------|
| 字段名（Fieldname） | 输入字段的名称 |
| 重命名为（Rename to） | 若要重命名字段，在此填写新名称 |
| 类型（Type） | 字段的数据类型 |
| 长度（Length） | 字段长度 |
| 精度（Precision） | 字段精度 |
| 二进制转普通（Binary to Normal?） | 在适当时将字符串转换为数值数据类型 |
| 格式（Format） | 格式掩码（数值类型或日期格式） |
| 日期格式宽松（Date Format Lenient?） | 决定日期解析器是否严格。设为 N 仅接受严格有效的日期值；设为 Y 解析器会尝试推断并修正错误日期 |
| 日期区域（Date Locale） | 用于日期转换和计算的日期区域设置 |

## 注意事项

- 若要复制字段，只需多次指定同一字段名并使用不同的重命名值。
- 支持 Hop Engine、Spark、Flink、Dataflow 引擎。
