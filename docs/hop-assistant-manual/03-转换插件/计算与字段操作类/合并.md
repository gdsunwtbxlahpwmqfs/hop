# 合并（Coalesce）

合并转换允许您列出多个字段，并返回其中第一个非空（non-null）的值。

这与 SQL 中的 COALESCE 函数功能类似，常用于从多个可能为空的字段中获取有效值。

## 主要选项

| 选项 | 说明 |
|------|------|
| 输出字段名（Output field name） | 合并结果输出的字段名称 |
| 值类型（Value type） | 输出字段的数据类型 |
| 字段列表（Fields） | 按顺序列出参与合并的多个字段，转换将返回其中第一个非空的值 |

## 注意事项

- 支持 Hop Engine 引擎；Spark、Flink、Dataflow 引擎支持情况视具体实现而定。
- 适用于数据合并场景，例如从多个数据源中优先选取有效值。
