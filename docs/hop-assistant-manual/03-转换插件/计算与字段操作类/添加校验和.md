# 添加校验和（Add a Checksum）

添加校验和转换用于为输入流中的一个或多个字段计算校验和（Checksum），并将计算结果作为新字段添加到输出流中。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称（Transform name） | 转换的名称，在单个管道中必须唯一 |
| 校验和类型（Type） | 需要计算的校验和类型，包括：CRC32（32 位循环冗余校验）、ADLER 32（Mark Adler 校验算法）、MD5（消息摘要算法 5）、SHA-1（安全散列算法 1）、SHA-256（安全散列算法 2） |
| 结果类型（Result Type） | 部分校验和类型支持设置不同的结果类型：字符串（String）、十六进制（Hexadecimal）、二进制（Binary） |
| 结果字段（Result field） | 包含校验和计算结果的新字段名称 |
| 参与计算的字段（Fields used in the checksum） | 参与校验和计算的字段名称，可使用"获取字段（Get Fields）"按钮插入前序转换的所有输入字段 |

## 注意事项

- 支持 Hop Engine、Spark、Flink、Dataflow 引擎。
