## 选项

| 选项 | 描述 |
|---|---|
| Transform name | Transform 的名称。 |
| Type a | 需要计算的校验和类型。 |
| Result Type | 某些校验和类型允许设置不同的结果类型：String、Hexadecimal 和 Binary。 |
| Result field | 包含校验和的结果字段名称。 |
| Fields used in the checksum | 参与校验和计算的字段名称。 |
