# 字段拼接（Concat Fields）

## 功能概述


字段拼接转换将多个字段拼接成一个目标字段。可以指定目标字段名称（始终创建新字段），并可选择在字段之间使用分隔符和封装符，按顺序添加要拼接的字段或自由文本。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称（Transform name） | 转换的名称 |
| 目标字段名（Target Field Name） | 目标字段名称（字符串类型） |
| 目标字段长度（Length of Target Field） | 字符串类型的长度（设置字符串类型的元数据，Fast Data Dump 选项也用于性能优化） |
| 分隔符（Separator） | 指定分隔字段的字符，通常为 `;` 或制表符 |
| 封装符（Enclosure） | 一对字符串可封装某些字段，允许字段中包含分隔符。封装字符串为可选 |
| 强制封装？（Force enclosure?） | 强制在添加到拼接字符串的每个字段前后添加封装字符 |
| 移除已拼接字段？（Remove concatenated fields from output?） | 勾选后，用于创建拼接字段（在 Target Field Name 中指定）的字段将从输出中移除，后续转换不可用 |

      <method>none</method>
      <schema_name/>
    </partitioning>
    <fields>
      <field>
        <case_sensitive>N</case_sensitive>
        <in_stream_name>value_variable</in_stream_name>
        <is_unicode>N</is_unicode>

        <replace_string>backslash</replace_string>
        <set_empty_string>N</set_empty_string>
        <use_regex>N</use_regex>
        <whole_word>N</whole_word>
      </field>
    </fields>
    <attributes/>
    <GUI>
      <xloc>288</xloc>

    </GUI>
  </transform>
  <transform>
    <name>Verify</name>

    <copies>1</copies>
    <partitioning>
      <method>none</method>
      <schema_name/>

      <yloc>96</yloc>

    <type>DataGrid</type>
