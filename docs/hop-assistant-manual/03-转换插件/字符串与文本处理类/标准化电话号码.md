# 标准化电话号码（Standardize Phone Number）

## 功能概述


标准化电话号码转换以标准化和一致的方式对电话号码进行规范化处理。该转换使用 [Google libphonenumber](https://github.com/google/libphonenumber) 库来解析和格式化电话号码。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称（Transform name） | 转换的名称 |
| 输入字段（Input field） | 输入数据中保存电话号码的字段 |
| 输出字段（Output field） | 留空则更新输入流中的字段；否则将向输出数据添加新字段（字符串类型） |
| 国家字段（Country field） | 输入数据中保存国家代码的字段。国家代码需为 ISO alpha-2 格式 |
| 默认国家（DefaultCountry） | 当国家字段输入数据为空或无效时使用的默认国家代码 |
| 格式（Format） | 用于标准化电话号码的格式，可选：E164（默认）、INTERNATIONAL 等 |

## 注意事项

- 在 Spark、Flink、Dataflow 引擎上可能受支持（Maybe Supported），Qi 数据治理平台 Engine 完全支持。
- 国家代码必须使用 ISO alpha-2 格式（如 CN、US）。
