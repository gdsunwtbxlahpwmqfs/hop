# 字符串操作（String Operations）

## 功能概述


字符串操作转换可以对传入的 Hop 字段执行修剪（trim）、大小写转换、填充（padding）以及其他字符串操作。该转换支持对单个输入字段应用多种字符串处理操作，常用于数据清洗和格式标准化场景。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称（Transform name） | 转换的名称 |
| 输入流字段（In stream field） | 指定要处理的字段 |
| 输出流字段（Out stream field） | 指定要创建的新字段名称 |
| 修剪类型（Trim type） | 修剪类型：无（none）、左（left）、右（right）、两端（both） |
| 大小写（Lower/Upper） | 指定转换为大写或小写 |
| 填充（Padding） | 指定左填充或右填充 |
| 填充字符（Pad char） | 指定填充使用的字符 |

## 支持的操作

- 修剪（去除前导和/或尾随空格）
- 转换为大写或小写
- 填充（添加前导或尾随额外字符）
- 转换为首字母大写
- 忽略转义字符
- 移除或仅保留数字字符
- 移除特殊字符
