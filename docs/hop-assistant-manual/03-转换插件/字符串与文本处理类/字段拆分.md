# 字段拆分（Split Fields）

字段拆分转换根据指定的分隔符将一个字段拆分为多个字段。当源数据中某个字段包含由分隔符连接的多个值时，可使用此转换将其分离为独立的字段。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称（Transform name） | 转换的名称，在管道内必须唯一 |
| 待拆分字段（Field to split） | 要拆分的字段名称 |
| 分隔符（Delimiter） | 决定字段的分隔符。特殊字符（如 CHAR ASCII HEX01）可用 `$[value]` 格式设置，例如 `$[01]` 或 `$[6F,FF,00,1F]` |
| 封装符（Enclosure） | 可指定封装字符串，放置在值周围时允许分隔符出现在值中。例如用逗号分隔时，`"aaaa,\"123,456\",ccc"` 可用封装符 `"` 解析为 3 个字段 |
| 转义字符串（Escape string） | 为在值中包含分隔符，可使用转义字符串如反斜杠。例如用逗号分隔时，`aaaa,123\\,456,ccc` 可用转义符 `\\` 解析为 3 个字段 |
| 字段表（Fields table） | 定义拆分创建的每个新字段的属性，包括字段名、数据类型等 |

## 注意事项

- 在 Spark、Flink、Dataflow 引擎上可能受支持（Maybe Supported），Hop Engine 完全支持。
