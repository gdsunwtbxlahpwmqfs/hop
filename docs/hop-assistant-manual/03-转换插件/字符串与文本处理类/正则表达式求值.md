# 正则表达式求值（Regex Evaluation）

## 功能概述


正则表达式求值转换将输入字段的字符串与通过正则表达式（regex）定义的文本模式进行匹配。该转换使用 `java.util.regex` 包，正则表达式的语法定义在 `java.util.regex.Pattern` 的 javadoc 中。
可以利用此转换解析复杂的文本字符串，并通过捕获组（由圆括号定义）从输入字段中创建新字段。

## 使用说明

### 模式匹配

该转换的主要用途是检查输入字段是否匹配给定模式：向数据流添加一个布尔字段，指示是否匹配。

模式旨在匹配整个输入字段，而不仅是其中一部分。例如输入 `"Author, Ann" - 53 posts`，正则表达式 `\d* posts` 不会匹配，即使输入的一部分（`53 posts`）确实匹配该模式。要获得实际匹配，需要在模式中添加 `.*`。

## 注意事项

- 在 Spark、Flink、Dataflow 引擎上可能受支持（Maybe Supported），Qi 数据治理平台 Engine 完全支持。
- 默认情况下模式匹配整个输入字段，需用 `.*` 包裹以匹配部分内容。
