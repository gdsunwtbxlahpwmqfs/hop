# 公式（Formula）

## 功能概述


公式转换允许在管道中对字段应用类 Excel 公式和函数（参考 [Apache POI](https://poi.apache.org/components/spreadsheet/eval-devguide.html)）。注意：此处使用的是 Excel 公式语法，而非 Java 或 JavaScript 公式。
使用公式时，不局限于像计算器（Calculator）那样仅比较三个字段，还可以设置和比较日期与时间。

## 使用说明

- 可以选择创建新字段（使用 New field 列）或设置字段值（使用 Replace value 列）。
- 支持的公式列表请查阅 Apache POI 文档页面：https://poi.apache.org/components/spreadsheet/eval-devguide.html#What+functions+are+supported

## XML代码模板

```xml
<transform>
    <name>information formulas</name>
    <type>Formula</type>
    <description/>
    <distribute>Y</distribute>
    <custom_distribution/>
    <copies>1</copies>
    <partitioning>
      <method>none</method>
      <schema_name/>
    </partitioning>
    <formulas>
      <formula>
        <field_name>is_even</field_name>
        <formula>ISEVEN([three])</formula>
        <replace_field/>
        <value_length>-1</value_length>
        <value_precision>-1</value_precision>
        <value_type>4</value_type>
      </formula>
      <formula>
        <field_name>is_odd</field_name>
        <formula>ISODD([twenty_four])</formula>
        <replace_field/>
        <value_length>-1</value_length>
        <value_precision>-1</value_precision>
        <value_type>4</value_type>
      </formula>
      <formula>
        <field_name>dollar</field_name>
        <formula>DOLLAR([two_and_a_half],2)</formula>
        <replace_field/>
        <value_length>-1</value_length>
        <value_precision>-1</value_precision>
        <value_type>2</value_type>
      </formula>
    </formulas>
    <attributes/>
    <GUI>
      <xloc>272</xloc>
      <yloc>96</yloc>
    </GUI>
  </transform>
```

## 注意事项

- 在 Spark、Flink、Dataflow 引擎上可能受支持（Maybe Supported），Qi 数据治理平台 Engine 完全支持。
- 公式不会评估预先不存在的字段。可先执行管道创建缺失字段，或通过 Replace value 下拉菜单查看字段是否存在。当管道属性参数未设置时，公式编辑器可能无法打开。
