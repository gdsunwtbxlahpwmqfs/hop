# 拆分字段为多行（Split Fields To Rows）

## 功能概述


拆分字段为多行转换将包含分隔字段的行拆分为多个新行，每个拆分值对应一行。适用于将单个字段中存储的多个值展开为独立数据行的场景。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称（Transform name） | 转换的名称，在管道内必须唯一 |
| 待拆分字段 | 要拆分的字段（必须为字符串字段） |
| 新字段名 | 必须指定的新字段名称 |
| 分隔符 | 用于拆分的分隔符，支持正则表达式 |

## 使用说明

- 原始行不会传递到后续转换。
- 原始字段保留在每个新行中，以便于调试或错误处理；如不需要可通过 Select Values 转换移除。
- 如果未指定新行的值类型/格式，新字段将使用与原字段相同的类型/格式。
- 分隔符支持正则表达式，例如按逗号、空格或逗号+空格拆分可使用：`[,\s]\s*`
