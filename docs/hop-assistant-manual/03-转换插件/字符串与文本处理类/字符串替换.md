# 字符串替换（Replace In String）

字符串替换转换对字符串字段执行简单的搜索与替换操作。它还支持正则表达式和分组引用（group references），分组引用在替换字符串中以 `$n` 形式表示，其中 n 为分组编号。也可以使用十六进制替换来处理特殊字符。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称（Transform name） | 转换的名称，在插件内必须唯一 |
| 输入流字段（In stream field） | 要执行替换的字段 |
| 输出流字段（Out stream field） | 结果字段。留空则替换输入字段值；否则输出到新字段 |
| 使用正则（use RegEx） | 指定是否使用正则表达式指定要替换的值，必须为 Y 或 N |
| 搜索（Search） | 搜索特定模式。可使用通配符（*）匹配多个字符；使用正则时在此指定模式 |
| 替换为（Replace with） | 指定替换搜索模式的值；使用正则时在此指定替换部分 |
| 设置空字符串？（Set empty string?） | 指定是否将字段值替换为空字符串，必须为 Y 或 N |
| 替换字段（Replace with field） | 用指定字段的值替换搜索模式 |

## 注意事项

- 支持 Hop Engine、Spark、Flink、Dataflow 全部引擎。
- 分组引用使用 `$n` 语法，n 为正则分组的编号。
