# 信用卡验证（Credit Card Validator）

信用卡验证转换用于检查信用卡号码的有效性以及识别信用卡发卡机构。它使用 LUHN10（MOD-10）算法验证号码有效性，并能识别发卡机构类型，如 VISA、MasterCard、Diners Club、EnRoute、American Express（AMEX）等。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称（Transform name） | 转换名称，在管道中唯一 |
| 信用卡字段（Credit card field） | 执行期间包含信用卡号码的输入字段名 |
| 仅获取数字？（Get only digits?） | 启用此选项可从（字符串）输入字段中剥离所有非数字字符 |
| 结果字段名（Result fieldname） | 指示号码有效性的（布尔型）输出字段名 |
| 信用卡类型字段（Credit card type field） | 保存信用卡类型（发卡机构）的输出字段名 |
| 无效消息（Not valid message） | 保存错误消息的输出字段名 |

## 注意事项

- 支持 Hop Engine、Spark、Flink、Dataflow 全部引擎。
- 有效性校验基于 LUHN10（MOD-10）算法。
