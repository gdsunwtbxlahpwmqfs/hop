# 令牌替换（Token Replacement）

令牌替换转换用于替换输入字符串或文件中的令牌（token）。该转换可以将结果数据输出到文件或数据流中的字段。

一个令牌包含起始字符串、名称和结束字符串。例如 `${my_token}` 可以是一个令牌。起始字符串和结束字符串是可配置的，可以是任意字符序列。

## 使用说明

- 当替换文件中的令牌时，此转换会读取整个文件，而非逐行操作。
- 替换文件中的令牌时，最佳实践是同时输出到文件，以避免 Hop 将整个文件读入内存。

## 注意事项

- 在 Spark、Flink、Dataflow 引擎上可能受支持（Maybe Supported），Hop Engine 完全支持。
- 令牌的起始和结束字符串可自定义。
