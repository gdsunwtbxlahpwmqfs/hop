# 语言检测（Detect Language）

语言检测转换通过检查文本来识别其所属语言。基于 [Lingua](https://github.com/pemistahl/lingua) 库实现，支持多种语言的自动识别。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称（Transform name） | 转换的名称，在整个管道中必须唯一 |

## 注意事项

- 该转换的依赖项未包含在应用程序中，需将以下文件放入 `/plugins/transforms/detectlanguage/lib` 文件夹：
  - [Lingua](https://repo1.maven.org/maven2/com/github/pemistahl/lingua/1.2.2/lingua-1.2.2.jar)
  - [Fastutil](https://repo1.maven.org/maven2/it/unimi/dsi/fastutil/8.5.15/fastutil-8.5.15.jar)
- 支持的语言列表参见 Lingua 项目文档。
- 在 Spark、Flink、Dataflow 引擎上可能受支持（Maybe Supported），Hop Engine 完全支持。
