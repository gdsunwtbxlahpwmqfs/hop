# 字符串剪切（Strings Cut）

## 功能概述


字符串剪切转换用于截取字符串的一部分（即子字符串）。如果指定的字段超出范围，则返回空白。该转换通过指定起止位置来提取子串。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称（Transform name） | 转换的名称 |
| 输入流字段（In stream field） | 要截取子串的字段名称 |
| 输出流字段（Out stream field） | 给定时将创建包含子串的新字段；否则用子串替换原字段 |
| 截取起始（Cut from） | 指定子串的起始位置。该值从 0 开始，即字符串的第一个字符为 0 |
| 截取结束（Cut to） | 指定子串的结束位置。该值为排他性（基于 1），新字符串的最大长度为 Cut to 减去 Cut from |

## 注意事项

- Cut from 基于 0 索引，Cut to 基于 1 的排他索引。
- 支持使用正数和负数长度引用来定位字符。
