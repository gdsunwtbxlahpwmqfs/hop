# Snowflake 批量加载器

Snowflake 批量加载器（Snowflake Bulk Loader）转换利用 Snowflake Copy 命令加载数据，而不是通过表输出转换发送单独的插入语句。

它通过三步过程执行此批量加载：

1. 将数据写入本地临时文件；
2. 运行 put 语句将本地文件复制到 Snowflake 暂存区（stage）；
3. 运行 copy 命令将数据从 Snowflake 暂存区批量加载到表中。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 数据库连接 | 目标 Snowflake 数据库连接 |
| 目标表 | 要加载的目标表名 |
| 工作目录 | 本地临时文件的工作目录 |
| Snowflake 暂存区 | Snowflake 暂存区名称 |
| 字段映射 | 输入字段到表列的映射 |

## 注意事项
- 该转换通过三步过程（写入临时文件 → PUT 到暂存区 → COPY 到表）实现批量加载。
- 性能优于通过表输出转换逐行插入。
