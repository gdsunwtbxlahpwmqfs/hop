# Doris 批量加载器

Doris 批量加载器（Doris Bulk Loader）转换允许你向 DorisDB 批量插入数据。

Apache Doris 是一款现代的 MPP 分析型数据库产品，可提供亚秒级查询和高效的实时数据分析。凭借其分布式架构，可支持高达 10PB 级别的数据集，且易于操作。

Apache Doris 能满足各种数据分析需求，包括历史数据报表、实时数据分析、交互式数据分析和探索性数据分析。

Doris 批量加载器允许你以高速、大批量向 Apache Doris 插入数据，比使用传统数据库插入语句更快。

## 主要选项

| 选项 | 说明 |
|------|------|
| 转换名称 | 转换的名称 |
| 数据库连接 | 目标 Doris 数据库连接 |
| 目标表 | 要加载的目标表名 |
| 字段映射 | 输入字段到表列的映射 |

## 注意事项

- 该转换通过流式加载实现批量插入，性能优于逐行插入。
- 适用于向 Apache Doris 大批量加载分析数据。
