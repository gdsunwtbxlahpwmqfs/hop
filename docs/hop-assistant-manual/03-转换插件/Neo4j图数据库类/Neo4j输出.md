# Neo4j 输出（Neo4j Output）

Neo4j Output 转换允许您对一个节点、两个节点或两个节点及其之间关系进行高性能更新操作。

该转换会自动生成所需的 Cypher 语句及配套的参数，从而实现高效的批量写入。它适用于需要将数据快速写入 Neo4j 图数据库的场景，支持节点和关系的创建与更新。

## 主要选项

| 选项 | 说明 |
|------|------|
| 连接 | 指定 Neo4j 数据库连接 |
| 更新模式 | 选择更新的目标：一个节点、两个节点或两个节点及一个关系 |
| 节点标签 | 为节点指定标签（Label） |
| 节点属性映射 | 将输入字段映射到节点的属性 |
| 主键属性 | 指定用于匹配和合并节点的主键属性 |
| 关系类型 | 当更新关系时，指定关系的类型 |
| 批处理大小 | 设置每批次处理的记录数量以提升写入性能 |
