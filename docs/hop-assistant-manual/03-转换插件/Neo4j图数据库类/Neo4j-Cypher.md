# Neo4j Cypher（Neo4j Cypher）

Neo4j Cypher 转换允许您通过 Neo4j Bolt 协议，使用 Java 驱动程序执行任意 Cypher 语句。

该转换支持读取、写入或同时进行读写操作，您还可以调用存储过程并获取其输出结果。它为与 Neo4j 图数据库交互提供了灵活的方式，适用于执行查询、更新数据或调用数据库内置过程等场景。

## 主要选项

| 选项 | 说明 |
|------|------|
| 连接 | 指定用于连接 Neo4j 数据库的连接配置（基于 Bolt 协议的 Java 驱动） |
| Cypher 语句 | 要执行的 Cypher 查询语句，可包含参数占位符 |
| 参数映射 | 将输入流字段绑定到 Cypher 语句中的参数 |
| 返回字段 | 配置从查询结果中读取并输出到下游的字段 |
| 批处理大小 | 控制写入操作时的批处理数量，以提升性能 |
