# LDAP 输出（LDAP Output）

LDAP Output 转换允许您将用户、角色及其他数据等信息写入 LDAP 服务器。

该转换提供对 LDAP 目录服务的写入能力，支持创建、更新和删除目录中的条目。适用于将组织架构、用户账户、权限角色等信息同步到 LDAP 目录服务器的场景。

## 主要选项

| 选项 | 说明 |
|------|------|
| 连接 | 指定 LDAP 服务器连接配置（包括主机、端口、认证方式等） |
| 操作类型 | 选择对 LDAP 条目执行的操作：新增、更新、删除或 upsert |
| 基础 DN | 指定操作的基准可分辨名称（Base DN） |
| 条目 DN | 指定目标条目的可分辨名称（DN） |
| 属性映射 | 将输入字段映射到 LDAP 条目的属性 |
