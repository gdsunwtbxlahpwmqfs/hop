# 从 MySQL 批量加载到文件

## 描述

`Bulk load from MySQL into file` action 将数据从 MySQL 表批量加载到文件中。

## 选项

| 选项 | 描述 |
|---|---|
| Action 名称 | workflow action 的名称。 |
| 连接 | 数据库连接。 |
| 目标 schema | 目标 schema。 |
| 目标文件名 | 目标文件。 |
| 高优先级 | 加载文件的优先级。 |
| 类型 | OUTFILE、DUMPFILE |
| 字段分隔符 | 使用的字段分隔符。 |
| 字段封闭符 | 使用的字段封闭符。 |
| 可选封闭 |  |
| 行终止符 | 使用的行终止字符串。 |
| 列名 | 要导出的逗号分隔列列表 |
| 限制前...行 | 仅导出前 x 行 |
| 文件已存在时 | 目标文件已存在时采取的操作： |
| 将文件添加到结果文件名 | 将目标文件添加到结果文件名中 |
