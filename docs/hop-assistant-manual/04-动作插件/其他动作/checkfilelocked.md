# 检查文件是否被锁定

## 描述

`Check files locked` action 遍历文件列表，检查它们是否被其他进程锁定。

验证方式是将文件重命名为相同的名称。

在某些操作系统（特别是 MS Windows 系统）上，如果文件被锁定，此操作将会失败。

## 选项

| 选项 | 描述 |
|---|---|
| Action 名称 | workflow action 的名称。 |
| 设置：包含子文件夹 | 启用此选项可同时检查指定文件夹子文件夹中的文件 |
| 设置：将上一步结果复制到参数？ | 将上一步 workflow action 执行的结果行传递到此 action 的文件列表中。 |
| 文件/文件夹 | 要检查的文件或文件夹。 |
| 通配符 | 通配符为 Java 正则表达式。 |
