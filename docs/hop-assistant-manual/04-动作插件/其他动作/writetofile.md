# 写入文件

## 描述

`Write to file` action 将字段中的自定义文本写入文件。

## 选项

| 选项 | 描述 |
|---|---|
| Action name | Workflow action 的名称。 |
| File name | 文件目录和名称。 |
| Browse... | 浏览文件系统以查找文件位置。 |
| Create parent folder | 是否创建父文件夹。 |
| Append file | 如果不勾选将覆盖之前的文件，如果勾选则将新文本追加到之前的文件。 |
| Encoding | 写入文件时使用的编码。 |
| Text | 要写入文件的文本。 |
