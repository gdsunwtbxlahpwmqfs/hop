# 构建文档（Build documentation）

## 功能概述


此动作为您的 Hop 文件或项目构建文档。

## 主要选项

- 您希望项目文档输出到的目标文件夹。
- 是否包含备注？
- 是否包含参数？
- 是否包含元数据？
- 是否生成 HTML？
- 生成 HTML 后是否删除 MarkDown？

## 从 MarkDown 生成 HTML

索引中的链接当前设置为 .md.html 文件。可以在目标文件夹中运行以下命令来生成 HTML 文件。
