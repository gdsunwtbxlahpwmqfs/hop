# 评估表中的行数

## 描述

`Evaluate rows numbers in a table` action 评估表中的行数，并允许您定义成功条件，用于将 workflow 分支为成功/错误流。

您可以仅测试目标表中的行数，也可以编写自定义 SQL 语句。

对于成功条件，您可以检查行数是否：

- 等于
- 不等于
- 小于
- 小于或等于
- 大于
- 大于或等于

## 选项

| 属性 | 描述 |
|---|---|
| Action 名称 | 此 workflow action 使用的名称 |
| 连接 | 要使用的[关系型数据库连接](metadata-types/rdbms-connection.md) |
| 目标 schema | 数据库 schema 名称 |
| 目标表名 | 数据库表名 |
| 行数成功条件 |  |
| 限制 | 与上方选择的选项结合使用来评估的行数。 |
| 自定义 SQL | 当您要在下方 `SQL Script` 字段中指定自定义 SQL 语句时，请勾选此选项。 |
| 使用变量替换 | SQL 语句中的变量将被替换。 |
| 清除结果行列表 | 在执行前清除内部结果行列表。 |
| 将行添加到结果 | 将 SQL 语句返回的所有行添加到内部结果行列表中。 |
| SQL 脚本 | 要使用的自定义 SQL 脚本。需要先勾选上方的 `Custom SQL` 选项才能启用此字段。 |
