# 邮件验证器

## 描述

`Mail validator` action 检查电子邮件地址是否有效。

当检查通过时，将执行成功的跳转；否则将执行失败的跳转。

## 选项

| 选项 | 描述 |
|---|---|
| Action 名称 | workflow action 的名称。 |
| 电子邮件地址 | 要检查的电子邮件地址或地址列表（以空格分隔）。 |
| SMTP 检查？ | 默认情况下，Qi Hop 仅检查电子邮件地址的结构（使用正则表达式）。 |
| 超时时间 | 为了执行 SMTP 检查，Hop 将在目标 SMTP 主机上打开一个套接字。 |
| 邮件发件人 | 如果您选择了"SMTP check?"选项，则此字段为必填。 |
| 默认 SMTP 服务器 | 如果您知道要查询哪个 SMTP 服务器，请在此指定，Hop 将仅查询该服务器。 |
