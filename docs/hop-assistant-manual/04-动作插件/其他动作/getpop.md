# 获取邮件（POP3/IMAP）

## 描述

`Get mails (POP3/IMAP)` action 从 POP3 或 IMAP 服务器检索邮件，并将其存储到一个或多个文件中。

## 选项

### 常规选项卡

| 选项 | 描述 |  |  |
|---|---|---|---|
| Mail Connection | 如果在此处选择了邮件服务器连接，则忽略此选项卡的其余部分。服务器配置将从 [Mail Server Connection](../../06-元数据类型/mail-server-connection.md) metadata 项中获取。 |  |  |
| Workflow action 名称 | workflow action 的名称。 |  |  |
| 源主机 | POP 服务器的主机名或 IP 地址。 |  |  |
| 使用 SSL | 如果勾选，将使用安全连接与服务器通信。 |  |  |
| 使用 XOAUTH2 | 如果勾选，将使用 XOAuth2 与服务器进行身份验证。 |  |  |
| 端口 | 运行 SSL POP3 或 IMAP 连接的端口。 |  |  |
| 用户名 | 用于登录的用户名。 |  |  |
| 密码 | 用于登录的密码。 |  |  |
| 使用代理 | 勾选以启用代理身份验证 |  |  |
| 代理用户名 | 代理授权用户（仅在勾选 `Use proxy` 时可用）|  |  |
| 协议 | 选择 |  |  |
| 测试连接 | 点击此按钮可测试提供的设置 |  |  |
| 目标目录 | 您希望存放邮件文件的目录。 |  |  |
| 创建文件夹 | 勾选以在指定目标目录不存在时创建它 |  |  |
| 目标文件名模式 | 文件名模式（文件名开头）。默认模式为 `name_{SYS\ | hhmmss_MMddyyyy\ | }_#IdFile#.mail` |
| 保存消息内容 | 勾选以保存检索到的电子邮件的消息正文 |  |  |
| 获取邮件附件 | 勾选以保存检索到的电子邮件的附件 |  |  |
| 附件使用不同文件夹 | 勾选以将检索到的电子邮件的附件保存到与消息正文不同的文件夹中 |  |  |
| 附件文件夹 | 存储检索到的电子邮件附件的文件夹（仅在勾选 `Different folder for attachment` 时可用）|  |  |
| 附件通配符 | 用于保存检索到的电子邮件附件的通配符（无 = 全部）|  |  |

### 设置选项卡

| 选项 | 描述 |
|---|---|
| 操作 | 此下拉框允许您指定对检索到的电子邮件执行的操作： |
| 检索 |  |
| 检索前.. 封邮件 | 如果选择了"前 ... 封邮件"，允许您指定要检索的邮件数量。 |
| 检索后删除邮件 | 勾选以在检索后删除 POP 服务器上的邮件。 |
| 选项 | 描述 |
| IMAP 文件夹 | 要从中检索消息的 IMAP 服务器上的文件夹 |
| 包含子文件夹 | 勾选以从选定的 IMAP 文件夹的子文件夹中选择消息 |
| 检索 | 选择以下之一 |
| 检索前...封邮件 | 指定要检索的消息数量 |
| 检索后 | 指定检索后对电子邮件执行的操作： |
| 移动到文件夹 | 将检索到的消息移动到 IMAP 服务器上的不同文件夹（如果选择了 `Move message to folder`）|
| 创建文件夹 | 在 IMAP 服务器上创建一个文件夹，用于移动消息（如果不存在）。 |

### 筛选器选项卡

| 选项 | 描述 |
|---|---|
| 发件人（FROM）| 消息将按发件人筛选 |
| 收件人（TO）| 消息将按收件人筛选 |
| 主题 | 消息将按主题筛选 |
| 选项 | 描述 |
| 正文 | 消息将按电子邮件正文中指定的文本筛选 |
| 选项 | 描述 |
| 条件 | 选择日期或日期范围。可用选项包括 |
