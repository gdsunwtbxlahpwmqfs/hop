# 比较文件夹

## 描述

`Compare folders` action 比较两个文件夹以确定其内容是否相同；结果为 true 或 false。

## 选项

| 选项 | 描述 |
|---|---|
| Workflow action 名称 | workflow action 的名称。 |
| 包含子文件夹 | 同时比较子文件夹的内容。 |
| 比较内容 | 指定要比较的内容：全部、仅文件、仅文件夹，或让我指定... |
| 通配符 | 如果您在上一个选项中选择了"让我指定"，则可以指定要比较的文件的正则表达式。 |
| 比较文件大小 | 勾选此项可比较文件大小，而不仅仅是比较文件夹名称。 |
| 比较文件内容 | 检查文件是否具有相同的内容。 |
| 文件/文件夹名称 1 | 要比较的第一个文件或文件夹 |
| 文件/文件夹名称 2 | 要比较的第二个文件或文件夹 |
