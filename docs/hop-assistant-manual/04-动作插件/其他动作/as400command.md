# AS/400 Command

## 描述

`AS/400 Command` action 用于执行 AS/400 CL 命令。

## 选项

| 选项 | 描述 |
|---|---|
| Action 名称 | workflow action 的名称。 |
| 服务器 | 主机名或 IP 地址。 |
| 用户名 | 与 AS/400 服务器账户关联的用户名。 |
| 密码 | 与 AS/400 服务器账户关联的密码。 |
| 代理主机 | 代理服务器的主机名或 IP 地址。 |
| 代理端口 | 代理服务器的端口号。 |
| 要调用的命令 | 在 AS/400 上运行的命令。如果命令未指定库限定名，将使用库列表来查找命令。 |

> **📝 注意:** 此 plugin 会尝试从 as400 数据库 plugin 加载 as400 库。
