# XSD 验证器

## 描述

`XSD validator` 根据内部（在 XML 文件中定义）或外部（在单独的 xsd 文件中定义）的 XML schema 验证 XML 文件。

XSD 是链接：[XML Schema](http://en.wikipedia.org/wiki/XML_Schema_(W3C)) Definition 的缩写。

因此，这里有两个实体在工作：

- 您要验证布局的 XML
- 描述 XML 文件应是什么样子的 XSD（Schema）文件。

另请参见 [XSD Validator pipeline transform](xsdvalidator.md)。
