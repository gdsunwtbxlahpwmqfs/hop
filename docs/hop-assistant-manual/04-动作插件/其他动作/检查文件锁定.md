# 检查文件锁定（Check Files Locked）

`Check files locked` 动作循环遍历文件列表，检查它们是否被其他进程锁定。

验证方式是将文件重命名为与之前相同的名称。

在某些操作系统（特别是 MS Windows 系统）上，如果文件被锁定，此操作将失败。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 设置：包含子文件夹（Settings: Include subfolders） | 启用此选项可同时检查指定文件夹子文件夹中的文件。 |
| 设置：将先前结果复制到参数？（Settings: Copy previous results to args?） | 将先前工作流动作执行的结果行传递给此动作的文件列表。 |
| 文件/文件夹（Files/Folders） | 要检查的文件或文件夹。 |
| 通配符（Wildcard） | 通配符是 Java 正则表达式。 |
