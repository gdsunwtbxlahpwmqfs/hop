# 截断表

## 描述

`Truncate tables` action 截断一个或多个表。

## 选项

| 选项 | 描述 |
|---|---|
| Action name | Workflow action 的名称。 |
| Connection | 表所在的数据库名称。 |
| Previous results to args? | 如果要以动态方式从先前 workflow action 的结果中获取要截断的表，请勾选此选项。 |
| Click here to return tables name list | 此按钮将返回该连接上定义的所有表。 |
| Selected tables grid | 在此指定要截断的表（手动输入或使用上方的按钮）。 |
