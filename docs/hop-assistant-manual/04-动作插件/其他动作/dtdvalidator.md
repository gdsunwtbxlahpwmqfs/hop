# DTD 验证器

## 描述

`DTD validator` action 根据文档类型定义（DTD）验证 XML 文件是否符合特定结构或格式，详见 link:[DTD](http://en.wikipedia.org/wiki/Document_Type_Definition)。

## 选项

| 选项 | 描述 |
|---|---|
| Action 名称 | workflow action 的名称。 |
| XML 文件名 | 要验证的 XML 文件名称 |
| 内部 DTD | 如果 DTD 包含在 XML 文件中，请勾选此项 |
| DTD 文件名 | 如果 DTD 不是内部的，请指定用于验证的 DTD 文件。 |
