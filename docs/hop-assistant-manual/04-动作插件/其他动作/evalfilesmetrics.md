# 评估文件指标

## 描述

`Evaluate files metrics` action 可用于评估文件的大小或数量。

## 选项

### 常规选项卡

| 选项 | 描述 |
|---|---|
| Action 名称 | workflow action 的名称。 |
| 源文件 | 源类型。 |
| 通配符 | 正则表达式通配符。 |
| 文件结果字段 |  |
| 通配符结果字段 |  |
| 包含子文件夹结果字段 |  |
| 评估 | 评估类型 |
| 文件/文件夹来源 | 文件或文件夹路径。 |
| 通配符 | 正则表达式通配符。 |
| 添加 | 将源添加到文件/文件夹列表。 |
| 文件... | 浏览文件系统以选择文件。 |
| 文件夹... | 浏览文件系统以选择文件夹。 |
| 文件/文件夹 | 文件和/或文件夹列表。 |
| 删除 | 从列表中删除一个源。 |
| 编辑 | 编辑一个源。 |

指定文件夹和通配符（正则表达式）后，使用 `Add` 按钮将您的文件/文件夹选择添加到表中。使用 `Delete` 或 `Edit` 按钮可删除或编辑表中选定的行。
