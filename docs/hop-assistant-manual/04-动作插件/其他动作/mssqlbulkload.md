# 批量加载到 MSSQL

## 描述

`Bulk load into MSSQL` action 将数据从文件批量加载到 MSSQL 中。

与 transform 批量加载器不同，此 action 需要一个预先存在的文件来批量加载到 MS SQL。

| 截断表 | 是否在开始批量加载前截断表。 |
