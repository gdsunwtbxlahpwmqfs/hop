# FTP上传文件（Put a file with FTP）

Put A File With FTP 动作将文件上传到 FTP 服务器。

## 主要选项

### 常规选项卡

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| FTP 服务器名称/IP 地址（FTP server name/IP address） | 服务器名称或 IP 地址。 |
| 端口（Port） | FTP 服务器连接的端口号。 |
| 用户名（Username） | 与 FTP 服务器账户关联的用户名。 |
| 密码（Password） | 与 FTP 服务器账户关联的密码。 |
| 代理主机（Proxy host） | 代理服务器主机名。 |
