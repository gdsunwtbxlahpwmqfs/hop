# AS400命令（AS/400 Command）

`AS/400 Command` 动作执行 AS/400 CL 命令。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 服务器（Server） | 主机名或 IP 地址。 |
| 用户名（User name） | 与 AS/400 服务器账户关联的用户名。 |
| 密码（Password） | 与 AS/400 服务器账户关联的密码。 |
| 代理主机（Proxy host） | 代理服务器主机名或 IP 地址。 |
| 代理端口（Proxy port） | 代理服务器端口号。 |
| 要调用的命令（Command to call） | 要在 AS/400 上运行的命令。如果命令未限定库，将使用库列表来查找命令。 |
