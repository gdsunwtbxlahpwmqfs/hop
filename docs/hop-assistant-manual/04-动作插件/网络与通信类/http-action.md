# HTTP

## 描述

`HTTP` action 使用 HTTP 协议从 Web 服务器检索文件。

此 action 可用于访问合作伙伴 Web 站点上的数据。

例如，每日数据导出或每日客户列表位于指定的 Web 站点。

此外，SaaS 提供商可能会给您一个 URL 来定位报告。

您可以调用该 URL 来检索包含数据的 Excel 文件或 zip 文件。

Salesforce 要求您使用 SOAP API 来检索数据。

如果您公司环境中的 HTTP 流量过大，您可以选择使用带有 HTTP 身份验证的代理服务器。

## 选项

### 常规选项卡

| 选项 | 描述 |
|---|---|
| Action 名称 | workflow action 的名称。 |
| URL | 要检索文件的 HTTP URL，或存储上传文件的目录名。 |
| 为每行结果运行？ | 如果您希望为上一步 pipeline 生成的每一行运行此 action，请勾选此项。 |
| 包含 URL 的输入字段 | 如果选中"Run for every result row?"选项，此处指定的字段将确定每行的文件 URL |
| 包含上传文件名的输入字段 | 如果选中"Run for every result row?"选项，此处指定的字段将确定将上传到与"Input field which contains URL"字段关联的 URL 的本地文件。 |
| 包含目标文件名的输入字段 | 如果选中"Run for every result row?"选项，此处指定的字段将确定从与"Input field which contains URL"字段关联的 URL 下载的结果保存到的本地文件。 |
| 用户名 | 如果站点需要身份验证，使用此用户名登录 |
| 密码 | 如果定义了用户名，这是对应的密码 |
| 上传代理服务器 | 您想要通过其连接到 HTTP URL 的代理服务器 URL |
| 代理端口 | 如果定义了代理服务器，这是它监听的端口号 |
| 忽略主机的代理 | 代理重定向例外的正则表达式列表。 |
| 上传文件 | 如果您要上传文件，这将是其在远程服务器上的名称 |
| 目标文件 | 如果您要下载文件，这是其在本地文件系统上的名称 |
| 追加到指定目标文件？ | 如果选中，且目标文件已存在，Hop 会将所有新数据追加到文件末尾 |
| 在文件名中添加日期和时间？ | 如果选中，HTTP 请求的日期和时间（yyyMMdd_HHmmss 格式）将添加到目标文件名中 |
| 目标文件扩展名 | 如果选中上一个选项，此字段指定目标文件名的扩展名（点后的字母） |
| 将文件名添加到结果文件名 | 所有复制的文件将作为此 action 的结果显示；显示此 action 中复制的文件列表。 |

### Headers 选项卡

| 选项 | 描述 |
|---|---|
| # | header 的处理顺序 |
| 名称 | 此 HTTP header 的名称 |
| 值 | 要传递的实际 header 值 |
