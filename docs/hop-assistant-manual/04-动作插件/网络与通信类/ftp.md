# 通过 FTP 获取文件

## 描述

`Get a file with FTP` action 从 FTP 服务器检索一个或多个文件。

此 workflow action 不会"爬取"系统。

例如，它不会访问远程目录并进入其他目录来查找匹配通配符的文件。

此 workflow 仅从一个目录检索文件。

## 选项

### 常规选项卡

| 选项 | 描述 |
|---|---|
| Action 名称 | workflow action 的名称。 |
| FTP 服务器名称/IP 地址 | 服务器名称或 IP 地址。 |
| 服务器端口 | FTP 服务器的端口号。 |
| 用户名 | 与 FTP 服务器账户关联的用户名。 |
| 密码 | 与 FTP 服务器账户关联的密码。 |
| 代理主机 | 代理服务器主机名。 |
| 代理端口 | 代理服务器端口号。 |
| 代理用户名 | 代理服务器账户用户名。 |
| 代理密码 | 代理服务器账户密码。 |
| 二进制模式 | 如果文件必须以二进制模式传输，请启用。 |
| 超时 | FTP 服务器超时时间（秒）。 |
| 使用主动 FTP 连接 | 如果使用主动模式连接 FTP 服务器，请启用；您必须设置防火墙以接受到 FTP 客户端将打开的端口的连接。 |
| 控制编码 | 检索包含特殊字符的文件名时，编码很重要。 |

### Files 选项卡

| 选项 | 描述 |
|---|---|
| 远程目录 | FTP 服务器上从中获取文件的远程目录。 |
| 通配符（正则表达式） | 要选择多个文件时的正则表达式。 |
| 检索后删除文件 | 仅在所有选定文件成功传输后，才删除 FTP 服务器上的文件。 |
| 检索后移动文件 | 仅在所有选定文件成功传输后，才将 FTP 服务器上的文件移动到下方指定的文件夹。 |
| 移动到文件夹 | 将文件移动到服务器上的指定文件夹。 |
| 创建文件夹 | 创建将包含文件的文件夹。 |
| 目标目录 | 您希望放置检索到的文件的目录。 |
| 在文件名中包含日期 | 将系统日期添加到文件名中（_20101231）。 |
| 在文件名中包含时间 | 将系统时间添加到文件名中（_235959）。 |
| 指定日期时间格式 | 启用以提供您自己的日期/时间格式；默认为 yyyyMMdd'_'HHmmss |
| 日期时间格式 | 选择日期时间格式。 |
| 在扩展名前添加日期 | 在文件名中的扩展名之前添加日期。 |
| 不覆盖文件 | 启用以在目标目录中已存在同名文件时跳过、重命名或失败。 |
| 如果文件存在 | 当目标目录中已存在同名文件时要采取的操作。 |
| 将文件名添加到结果 | 启用以将读取的文件名添加到此 workflow 的结果中。 |

|---|---|
