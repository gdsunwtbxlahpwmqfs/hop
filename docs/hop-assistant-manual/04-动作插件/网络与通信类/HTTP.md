# HTTP（HTTP）

## 功能概述

`HTTP` 动作使用 HTTP 协议从 Web 服务器检索文件。

此动作可用于访问合作伙伴网站上的数据。例如，每日数据导出或每日客户列表位于指定网站上。此外，SaaS 提供商可能会给您一个 URL 来定位报告。您可以调用该 URL 来检索包含数据的 Excel 文件或 zip 文件。

Salesforce 要求您使用 SOAP API 来检索数据。

如果您的企业环境中 HTTP 流量过大，您可以选择使用带有 HTTP 身份验证的代理服务器。
