# Shell脚本（Shell）

## 功能概述

`Shell` 动作在运行工作流的主机上执行 shell 脚本。

例如，假设您有一个程序读取五个数据表并以指定格式创建文件。

### 脚本或可执行文件

Shell 允许您在 Qi 数据治理平台 中完成部分工作，但根据需要重用读取数据表的程序。

Shell 工作流动作是平台无关的；您可以使用批处理文件、UNIX 等。

当您使用 Shell 工作流动作时，Hop 进行 Java 调用以在指定位置执行程序。
