# 获取邮件POP3-IMAP（Get mails (POP3/IMAP)）

`Get mails (POP3/IMAP)` 动作从 POP3 或 IMAP 服务器检索邮件，并将其存储到一个或多个文件中。

## 主要选项

### 常规选项卡

| 选项 | 说明 |
|------|------|
| 邮件连接（Mail Connection） | 如果在此选择邮件服务器连接，则忽略此选项卡的其余部分。服务器配置将从邮件服务器连接元数据项中获取。 |
| 工作流动作名称（Workflow action name） | 工作流动作的名称。 |
| 源主机（Source Host） | POP 服务器的主机名或 IP 地址。 |
| 使用 SSL（Use SSL） | 如果勾选，将使用安全连接与服务器通信。 |
| 使用 XOAUTH2（Use XOAUTH2） | 如果勾选，将使用 XOAuth2 与服务器进行身份验证。 |
