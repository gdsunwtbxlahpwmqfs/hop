# FTP 删除

## 描述

`FTP delete` action 删除 FTP 服务器上的文件。

## 选项

### 常规选项卡

| 选项 | 描述 |
|---|---|
| Action 名称 | workflow action 的名称。 |
| 协议 a |  |
| FTP 服务器名称/IP 地址 | 服务器名称或 IP 地址。 |
| 服务器端口 | FTP 服务器的端口号。 |
| 用户名 | 与 FTP 服务器账户关联的用户名。 |
| 密码 | 与 FTP 服务器账户关联的密码。 |
| 连接类型 a | FTPS 连接类型。 |
| 使用代理 | 是否使用代理。 |
| 代理主机 | 代理服务器主机名。 |
| 代理端口 | 代理服务器端口号。 |
| 代理用户名 | 代理服务器账户用户名。 |
| 代理密码 | 代理服务器账户密码。 |
| 使用公钥 | 是否在 SSH 连接中使用公钥。 |
| 密钥文件 | 公钥文件。 |
| 浏览... | 浏览文件系统以选择公钥文件。 |
| 密钥口令 | 公钥口令。 |
| 测试连接 | 测试已配置的连接。 |

### Files 选项卡

| 选项 | 描述 |
|---|---|
| 超时 | 超时期限。 |
| 使用主动 FTP 连接 | 如果使用主动模式连接 FTP 服务器，请启用；您必须设置防火墙以接受到 FTP 客户端将打开的端口的连接。 |
| 将上一步结果复制到参数？ |  |
| 远程目录 | 远程 FTP 服务器上的目录。 |
| 检查文件夹 | 检查目录是否存在。 |
| 通配符（正则表达式） |  |
| 成功条件 a | 成功条件 |
| 限制文件数 | 设置删除文件数量的限制（默认 10）。 |

### Socks 代理选项卡

| 选项 | 描述 |
|---|---|
| 主机 | Socks 代理主机名。 |
| 端口 | Socks 代理端口号。 |
| 用户名 | 与 Socks 代理账户关联的用户名。 |
| 密码 | 与 Socks 代理账户关联的密码。 |
