# 邮件验证器（Mail Validator）

## 功能概述


`Mail validator` 动作检查电子邮件地址是否有效。
当检查通过时将走成功跳，否则走失败跳。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 电子邮件地址（Email address） | 要检查的电子邮件地址或列表（以空格分隔）。 |
| SMTP 检查？（SMTP check?） | 默认情况下，Qi 数据治理平台 仅检查电子邮件地址结构（使用正则表达式）。如果要执行 SMTP 检查，请选择此选项并填写电子邮件发件人。 |
