# 删除文件夹（Delete Folders）

`Delete Folders` 动作删除一组文件夹及其内容。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 设置（Settings） | 将先前结果复制到参数：如果要删除的文件夹来自工作流中前一个动作的结果文件，请启用此选项。 |
| 成功条件（Success on） | 选择以下之一：无错误报告时（一切正常删除）；至少处理 x 个文件夹时（在下方限制字段中指定 x）；错误数量少于时（在下方限制字段中指定最大错误数）。 |
| 限制（Limit） | 要删除的文件夹数量或错误数量，取决于上方选择的成功条件选项。 |
| 文件夹（Folders） | 所有要删除的文件夹列表。 |
