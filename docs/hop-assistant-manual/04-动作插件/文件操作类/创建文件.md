# 创建文件（Create File）

`Create File` 动作创建一个空文件。这对于从工作流中创建触发器文件非常有用。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 文件名（File name） | 要创建空文件的名称和路径。 |
| 文件存在时失败（Fail if file exists） | 当要创建的文件已存在（无论是否为空）且此选项开启时，工作流动作将走失败输出跳。默认为开启。 |
| 添加文件名到结果（Add filename to result） | 将此动作中创建的文件名添加到结果中。 |
