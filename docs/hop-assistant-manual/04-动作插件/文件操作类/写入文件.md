# 写入文件（Write to File）

`Write to file` 动作将字段中的自定义文本写入文件。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 文件名（File name） | 文件目录和名称。 |
| 浏览...（Browse...） | 浏览文件系统以查找文件位置。 |
| 创建父文件夹（Create parent folder） | 是否创建父文件夹。 |
| 追加文件（Append file） | 如果未勾选将覆盖先前文件，如果勾选则将新文本追加到先前文件。 |
| 编码（Encoding） | 写入文件时使用的编码。 |
| 文本（Text） | 要写入文件的文本。 |
