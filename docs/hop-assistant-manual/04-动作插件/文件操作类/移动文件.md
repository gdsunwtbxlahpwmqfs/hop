# 移动文件（Move Files）

`Move files` 动作将多个文件和/或文件夹移动到文件系统上的其他位置。

此功能也可通过 VFS 在远程文件系统上工作，例如 FTP。

在 FTP 服务器上移动（重命名）文件的示例（源和目标都必须提供用户名/密码）：

* 文件源：\ftp://username:password@ftpserver/dir/testfile_partly.txt
* 文件目标：\ftp://username:password@ftpserver/dir/testfile.txt
* 在"目标文件"选项卡中勾选"目标是文件"。

## 主要选项

### 常规选项卡

在此选项卡中，您可以指定要移动的文件和/或文件夹。
