# 文件是否存在

## 描述

`File exists` action 验证指定文件是否存在，以及运行 Qi Hop 的机器是否可以访问该文件。

您必须提供文件名。

Hop 根据文件是否存在返回 True 或 False 值。

File Exists action 可以作为与其他系统的简单集成点。

例如，假设您有一个三部分的数据仓库加载流程。

第一部分在 PERL 中运行。

您有批处理脚本从远程位置访问数据，执行第一级行处理，并将数据输出到指定目录。

您不希望在此完成之前启动 workflow，因此将 workflow 放入调度程序中。

任务完成后，文件会被放置在一个已知位置，即"文件存在"。这就是启动 workflow 进行最终处理的信号。

## 选项

| 选项 | 描述 |
|---|---|
| Action 名称 | workflow action 的名称。 |
| 文件名字段 | 指定要验证的文件的文件名和路径。 |
