# 从结果删除文件名（Delete filenames from result）

`Delete filenames from result` 动作删除上一个工作流动作的结果文件列表中的所有文件名。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 限制动作为（Limit action to） | 如果要将删除限制为结果文件列表中的某些文件名，请启用此功能。 |
| 通配符（Wildcard） | 用于限制要删除文件的正则表达式。 |
| 排除通配符（Exclude wildcard） | 用于排除某些文件不被删除的正则表达式。 |
