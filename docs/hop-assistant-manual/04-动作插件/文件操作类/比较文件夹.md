# 比较文件夹（Compare folders）

`Compare folders` 动作比较两个文件夹以确定内容是否相同；结果将为 true 或 false。

## 主要选项

| 选项 | 说明 |
|------|------|
| 工作流动作名称（Workflow action name） | 工作流动作的名称。该名称在单个工作流中必须唯一。 |
| 包含子文件夹（Include Subfolders） | 同时比较子文件夹的内容。 |
| 比较（Compare） | 指定比较内容：全部、仅文件、仅文件夹，或让我指定... |
| 通配符（Wildcard） | 如果在前一选项中选择了"让我指定"，可以指定要比较文件的正则表达式。 |
| 比较文件大小（Compare file size） | 勾选此项以比较文件大小，而不是仅比较文件夹名称。 |
| 比较文件内容（Compare file content） | 检查文件是否具有相同内容。注意：这可能导致处理速度变慢。 |
| 文件/文件夹名称 1（File/Folder name 1） | 要比较的第一个文件或文件夹。 |
