# DOS与Unix格式转换（Convert file between Windows and Unix）

`Convert file between Windows and Unix` 动作将文件从 Windows（DOS）格式（行尾为 CR/LF）转换为 Linux/Unix 格式（行尾为 LF），反之亦然。

当格式未知时，此动作还具有猜测功能，并转换为另一种格式。

## 主要选项

### 常规选项卡

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 设置（Settings） | 包含子文件夹：同时处理子文件夹。将先前结果复制到参数：使用先前工作流动作的结果文件列表代替下方的静态文件列表。 |
| 文件/文件夹（Files / Folders） | 要处理的文件和文件夹列表。 |
