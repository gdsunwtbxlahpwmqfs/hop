# 解压文件（Unzip file）

`Unzip file` 动作将一个或多个文件解压到指定的目标位置。

## 主要选项

### 常规选项卡

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 从上一个获取参数（Get args from previous） | 如果要使用（来自前一个工作流动作的）结果文件列表作为要解压的文件列表，请勾选此选项。 |
| Zip 文件名（Zip file name） | zip 文件的名称，或如果要使用通配符则为文件夹。 |
| 源通配符（Source wildcard） | 如果前一选项是文件夹，您可以在此输入正则表达式通配符。例如：要匹配 `/folder/test.zip`，正则表达式 `te.\.zip` 将无结果。 |
