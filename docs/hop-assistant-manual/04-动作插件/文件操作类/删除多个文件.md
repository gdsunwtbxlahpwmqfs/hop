# 删除多个文件（Delete files）

`Delete files` 动作删除一组指定的文件，可选择包含子文件夹。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 包含子文件夹（Include Subfolders） | 同时删除文件夹选择中的子文件夹。 |
| 将先前结果复制到参数？（Copy previous results to args?） | 如果文件集来自先前的工作流动作（作为结果文件的一部分），请启用此项。 |
| 文件/文件夹（File/Folder） | 要删除的文件或文件夹。 |
| 通配符（Wildcard） | 如果前一选项是文件夹，则匹配要删除文件的正则表达式。例如，要删除所有以 .dat 结尾的文件，正则表达式为 `.*\.dat$`。 |
| 文件/文件夹列表（Files/Folders） | 要删除的文件/文件夹完整列表。 |
