# 将文件名添加到结果

## 描述

`Add Filenames to result` action 将一组文件或文件夹添加到 workflow action 的结果列表中。

此文件名列表随后可在各个 workflow action 中使用。

## 选项

| 选项 | 描述 |
|---|---|
| Action 名称 | workflow action 的名称。 |
| 包含子文件夹 | 包含所选文件夹的子文件夹 |
| 将上一步结果复制到参数？ | 将上一步 action 的结果传递到此 action 的参数中。 |
| 清除结果文件名？ | 此选项在创建新列表之前清除结果文件列表（来自之前的 workflow action）。 |
| 文件/文件夹 | 在此网格中指定带有通配符（正则表达式）的文件或文件夹列表。 |
| 通配符 (RegExp) | 用于在文件名列表中包含文件模式的正则表达式通配符。 |

指定文件夹和通配符（正则表达式）后，使用 `Add` 按钮将您的文件/文件夹选择添加到表中。使用 `Delete` 或 `Edit` 按钮可删除或编辑表中选定的行。
