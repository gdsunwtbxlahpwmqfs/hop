# 删除文件

## 描述

`Delete files` action 删除一组指定文件，可选择包含子文件夹。

## 选项

| 选项 | 描述 |
|---|---|
| Action 名称 | workflow action 的名称。 |
| 包含子文件夹 | 同时删除文件夹选择中的子文件夹 |
| 将上一步结果复制到参数？ | 如果文件集来自上一个 workflow action（作为结果文件的一部分），请启用此选项 |
| 文件/文件夹 | 要删除的文件或文件夹 |
| 通配符 | 如果上一个选项是文件夹，则匹配要删除文件的正则表达式。 |
| 文件/文件夹 | 要删除的文件/文件夹完整列表 |
