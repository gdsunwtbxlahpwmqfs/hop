# 检查文件是否存在（Checks if files exist）

`Checks if files exist` 动作验证指定文件是否存在。如果列出的*所有*文件都存在，该动作返回 `true`（成功）；如果*任何*文件不存在，返回 `false`（失败）。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。该名称在单个工作流中必须唯一。 |
| 文件/文件夹（Files/Folders） | 指定要验证的文件和文件夹完整路径，每行一个。您可以在路径中使用如 `${PROJECT_HOME}` 这样的变量，它们将被解析。 |
