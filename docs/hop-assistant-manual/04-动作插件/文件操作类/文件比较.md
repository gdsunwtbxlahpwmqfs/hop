# 文件比较（File compare）

`File Compare` 动作比较两个文件的内容。如果文件相同则返回 true，不同则返回 false。

当文件内容相同时将走成功输出跳，否则走失败输出跳。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 文件名 1（File name 1） | 要比较的第一个文件的名称和路径。 |
| 文件名 2（File name 2） | 要比较的第二个文件的名称和路径。 |
| 添加文件名到结果（Add filename to result） | 将两个文件的文件名添加到结果中。 |
