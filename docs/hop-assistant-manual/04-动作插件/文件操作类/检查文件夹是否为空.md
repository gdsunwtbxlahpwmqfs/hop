# 检查文件夹是否为空（Check if a folder is empty）

`Check if a folder is empty` 动作验证文件夹是否为空，即文件夹中没有常规文件或子文件夹。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 文件夹名称（Folder name） | 要验证是否为空的文件夹名称。 |
| 包含子文件夹（Include subfolders） | 检查要检查的文件夹中是否有子文件夹。 |
| 限制搜索范围（Limit search to） | 将文件搜索限制为具有特定通配符（正则表达式）的文件。 |
| 通配符（Wildcard） | 用于限制要查找的文件的通配符（正则表达式），例如：`.*\.txt$`。 |
