# 移动文件

## 描述

`Move files` action 将多个文件和/或文件夹移动到文件系统上的其他位置。

这也可以通过 [VFS](../../14-虚拟文件系统/vfs.md) 在远程文件系统上工作，例如 FTP。

以下是在 FTP 服务器上移动（重命名）文件的示例（源和目标都必须提供用户名/密码）。

- 文件源：\ftp://username:password@ftpserver/dir/testfile_partly.txt
- 文件目标：\ftp://username:password@ftpserver/dir/testfile.txt
- 在 "Destination File" 选项卡中勾选 "Destination is a file"。

## 选项

### 常规选项卡

在此选项卡中，您可以指定要移动的文件和/或文件夹。

| 选项 | 描述 |
|---|---|
| Action 名称 | workflow action 的名称。 |
| 包含子文件夹 | 同时移动子文件夹的内容 |
| 移动空文件夹 | 同时移动空文件夹 |
| 模拟 | 不实际移动任何内容，仅查看是否可行。 |
| 将上一步结果复制到参数 | 使用上一个 workflow action 的结果文件作为要移动的文件和/或文件夹。 |
| 文件/文件夹 | 您可以在此网格中指定带有目标和通配符的文件或文件夹列表。 |

### 目标文件选项卡

在此选项卡中，您可以指定文件和/或文件夹的目标及其新文件名。

| 选项 | 描述 |
|---|---|
| 创建目标文件夹 | 使用此选项创建目标文件夹 |
| 目标是文件 | 如果要移动文件，请勾选此选项 |
| 不保留文件夹结构 | 如果使用了子文件夹，则展平子文件夹结构 |
| 在目标文件名中添加日期 | 在目标文件名中添加日期（yyyyMMdd 或 20091231） |
| 在目标文件名中添加时间 | 在目标文件名中添加时间（HHmmss 或 235959） |
| 指定日期时间格式 a | 允许您指定自己的日期时间格式。 |
| 在扩展名前添加日期 | 在文件扩展名之前添加日期部分。 |
| 如果目标文件存在 a | 选择以下选项之一： |
| 目标文件夹 | 要移动到的目标文件夹 |
| 创建文件夹 |  |

| 指定格式 a | 允许您指定自己的日期时间格式。 |

### 高级选项卡

| 选项 | 描述 |
|---|---|
| 成功条件 a | 选择以下选项之一： |
