# 复制文件（Copy Files）

Copy Files 动作复制一个或多个文件或文件夹。

## 文件选项卡

| 选项 | 说明 |
|------|------|
| 动作名称（Action Name） | 动作的名称。 |
| 源环境（Source Environment） | 指示可以找到要输入项的文件系统或特定集群。选项有 Local 和 `<Static>`。Local 表示指定的文件/文件夹位于 Hop 本地的文件系统中。`<Static>` 表示使用字段中的路径名，当您已知文件路径并想直接复制粘贴时使用。 |
| 源文件/文件夹（Source File/Folder） | 要从中复制的文件或目录。 |
