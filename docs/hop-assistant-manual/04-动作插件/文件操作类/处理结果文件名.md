# 处理结果文件名（Process result filenames）

`Process result filenames` 动作允许您复制、移动或删除多个文件和/或文件夹到文件系统上的其他位置。

## 主要选项

| 选项 | 说明 |
|------|------|
| 工作流动作名称（Workflow action name） | 工作流动作的名称。 |
| 动作（Action） | 选择以下之一：复制（默认）、移动、删除。 |
| 目标文件夹（Destination folder） | 定义目标文件夹。 |
| 创建目标文件夹（Create destination folder） | 当目标文件夹不存在时，使用此选项创建它。 |
| 替换现有文件（Replace existing file） | 使用此选项覆盖现有文件。 |
