# 添加文件名到结果（Add filenames to result）

`Add Filenames to result` 动作将一组文件或文件夹添加到工作流动作的结果列表中。

此文件名列表随后可以在各个工作流动作中使用。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 包含子文件夹（Include subfolders） | 包含所选文件夹的子文件夹。 |
| 将先前结果复制到参数？（Copy previous results to args?） | 将前一个动作的结果传递给此动作的参数。 |
| 清除结果文件名？（Clear result filenames?） | 此选项在创建新列表之前清除（先前工作流动作的）结果文件列表。 |
| 文件/文件夹（File/Folder） | 在此网格中指定带有通配符（正则表达式）的文件或文件夹列表。每行可以添加不同的源/目标。 |
| 通配符（RegExp）（Wildcard） | 用于在文件名列表中包含文件模式的正则表达式通配符。 |
