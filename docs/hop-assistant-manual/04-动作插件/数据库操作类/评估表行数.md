# 评估表行数（Evaluate rows numbers in a table）

## 功能概述

`Evaluate rows numbers in a table` 动作评估表中的行数，并允许您定义可用于将工作流分支为成功/错误流的成功条件。

您可以仅测试目标表中的行数，或编写自定义 SQL 语句。

对于成功条件，您可以检查行数是否：

* 等于（Equal to）
* 不等于（Different from）
* 小于（Smaller than）
* 小于或等于（Smaller or equal to）
* 大于（Greater than）
* 大于或等于（Greater or equal to）
