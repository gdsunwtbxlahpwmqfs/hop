# MongoDB-QL（MongoDB QL）

## 功能概述


`MongoDB QL` 动作从该动作的配置或文件中执行一个或多个 JSON 指令。
此动作可用于在 Mongo 数据库上执行管理指令。
您可以执行多个 JSON 指令。
任何可以在 MongoDB 的 runCommand() 中使用的 JSON 指令，都可以由此动作执行。
MongoDB 文档：https://www.mongodb.com/docs/manual/reference/command/

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
