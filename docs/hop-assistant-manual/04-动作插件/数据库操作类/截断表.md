# 截断表（Truncate Tables）

## 功能概述


`Truncate tables` 动作截断一个或多个表。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 连接（Connection） | 表所在的数据库名称。 |
| 先前结果到参数？（Previous results to args?） | 如果要以动态方式从先前工作流动作的结果中获取要截断的表，请勾选此选项。 |
| 点击此处返回表名列表（Click here to return tables name list） | 此按钮将返回连接上定义的所有表。选择表（一个或多个）并按"OK"。表名将自动添加到"已选表"网格中。 |
| 已选表网格（Selected tables grid） | 在此指定要截断的表（手动插入或使用前一个按钮）。如果表位于非默认架构中，请指定架构名称。 |
