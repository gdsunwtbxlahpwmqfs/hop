# SQL（SQL）

`SQL` 动作从该动作的配置或文件中执行一个或多个 SQL 脚本。

您可以执行多个 SQL 语句，只要它们用分号分隔。

SQL 工作流动作很灵活；您可以执行过程调用、创建和分析表等。

与 SQL 工作流动作相关的常见用途包括截断表、删除索引、分区加载、刷新物化视图、禁用约束、禁用统计等。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 数据库连接（Database Connection） | 要使用的数据库连接。 |
| 从文件加载SQL（SQL from file） | 启用此选项以从 SQL 文件名指定的文件中加载 SQL 语句。 |
