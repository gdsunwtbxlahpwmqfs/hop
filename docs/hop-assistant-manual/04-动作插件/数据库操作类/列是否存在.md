# 列是否存在（Columns exist in a table）

`Columns exist in a table` 动作验证数据库表中是否存在一个或多个列。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 连接（Connection） | 要使用的关系数据库连接。 |
| 架构名称（Schema name） | 要使用的表架构。 |
| 表名（Table name） | 要使用的表名称。 |
| 列（Columns） | 要验证的列名列表（一个或多个）。 |

使用 `Browse` 按钮可以从所选连接中选择数据库架构或表。
