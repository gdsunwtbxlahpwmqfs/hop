# 管道（Pipeline）

`Pipeline` 动作在工作流中运行先前定义的管道。

此动作是从您的工作流访问实际数据处理活动（管道）的入口点。

## 使用说明

一个常见的工作流示例包括：获取 FTP 文件、检查必要目标数据库表是否存在、运行填充该表的管道，以及如果管道失败则发送错误日志邮件。

在此示例中，管道动作定义了要运行哪个管道来填充表。

另请参阅：

* 工作流动作（Workflow action）：从工作流执行子工作流。
* 工作流执行器转换（Workflow Executor transform）：从管道执行工作流。
* 管道执行器转换（Pipeline Executor transform）：从管道执行子管道。
