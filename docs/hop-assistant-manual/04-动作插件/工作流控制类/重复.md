# 重复（Repeat）

## 功能概述


`Repeat` 动作会重复执行您选择的管道或工作流，直到满足特定条件。条件为以下之一：
* 工作流中的某个变量被设置或被设置为特定值
* 在指定的重复工作流中执行了结束重复（End Repeat）动作
除了管道和工作流执行器转换之外，重复（Repeat）和结束重复（End Repeat）动作允许您从工作流构建循环。它需要一个工作流或管道以及一个运行配置来使用。该动作将继续执行指定的管道或工作流，直到满足条件：某个变量被设置，或某个变量被设置为特定的可选值，或在子工作流中触发了结束重复动作。
> 提示：如果您使用变量名作为要重复的文件，请确保设置了带有文件扩展名的默认参数，以便 Hop 知道它是管道文件还是工作流文件。
> 提示：在管道中以"在 JVM 中有效"范围设置的变量，在重复管道的上游将无法访问。

## 示例

示例项目示例：`/loops/repeat-action.hwf` 和 `child-check-set-counter-value.hpl` 运行一个管道，该管道每次运行时递增 `${COUNTER}` 变量。如果变量值超过 10，则设置变量 `${END_LOOP}`。
