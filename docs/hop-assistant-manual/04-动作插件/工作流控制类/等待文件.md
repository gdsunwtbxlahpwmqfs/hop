# 等待文件（Wait for file）

`Wait for file` 动作会等待并定期检查文件是否存在。

此工作流动作将休眠并定期检查指定文件是否存在，然后流程将继续。

该动作可以无限期等待文件，也可以在一定时间后超时。

> 警告：此动作需要定义单个文件。可以使用变量，但此动作不支持通配符，因此无法根据文件名模式等待多个文件的存在。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 文件名（File name） | 要等待的文件名和路径。 |
| 最大超时时间（Maximum timeout） | 最大超时时间（秒数），或设为 0 表示无限期等待。 |
