# 检查XML格式良好（Check if XML file is well-formed）

`Check if XML file is well-formed` 动作验证一个或多个文件是否包含格式良好（合法）的 XML 内容。

## 主要选项

### 常规选项卡

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| 包含子文件夹（Include Subfolders） | 移动子文件夹内容的选项。 |
| 将先前结果复制到参数（Copy previous results to args） | 勾选此项可将前一个动作的结果传递给此动作的参数。 |
| 文件/文件夹源（File/Folder source） | 输入或选择（浏览按钮）文件或文件夹。 |
| 通配符（RegExp）（Wildcard） | 文件名模式的正则表达式通配符。 |
| 文件/文件夹（Files/Folders） | 在此网格中指定带有通配符（正则表达式）的文件或文件夹列表。每行可以添加不同的源/目标。 |
