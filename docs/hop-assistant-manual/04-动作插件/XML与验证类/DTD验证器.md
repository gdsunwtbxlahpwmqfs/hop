# DTD验证器（DTD validator）

`DTD validator` 动作根据文档类型定义（[DTD](http://en.wikipedia.org/wiki/Document_Type_Definition)）验证 XML 文件是否对应于特定结构或格式。

## 主要选项

| 选项 | 说明 |
|------|------|
| 动作名称（Action name） | 工作流动作的名称。 |
| XML 文件名（XML File name） | 要验证的 XML 文件名。 |
| 内部 DTD（DTD Intern） | 如果 DTD 包含在 XML 文件中，请勾选此项。 |
| DTD 文件名（DTD File name） | 如果 DTD 不是内部的，指定用于验证的 DTD 文件。 |
