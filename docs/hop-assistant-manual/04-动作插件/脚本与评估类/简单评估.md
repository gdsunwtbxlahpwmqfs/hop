# 简单评估（Simple evaluation）

`Simple evaluation` 动作评估结果流中变量或字段的内容或值。

用简单评估的输出连接两个动作。

如果条件成功评估，将调用绿色（成功）跳；如果条件未成功评估，将调用红色（失败）跳。

## 使用说明

* Null 值与空字符串相同。例如，要检测任何数据类型的 null，请将类型设置为 String，成功条件设置为"如果值等于"，并将值文本框留空。

* 简单评估动作功能有限。对于更复杂的条件，请参见：https://hop.apache.org/manual/latest/workflow/actions/eval.html
