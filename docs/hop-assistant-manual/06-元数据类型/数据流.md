# 数据流（Data Stream）

## 核心功能

**数据流**元数据类型提供了一种统一的、高性能的方式，用于在 Hop 管道和外部系统（尤其是通过 PyHop 的 Python）之间流式传输数据行，而无需总是将数据落地到磁盘。

它被设计为可插拔的，以便随着时间的推移可以添加不同的流式传输技术。

## 目标

- 实现 Hop 与 Python（以及其他 Arrow 兼容工具）之间的快速、低延迟数据交换
- 提供背压（backpressure）和正确的流式语义
- 保持一致的用户体验（在输入/输出转换中按名称选择数据流）

## 可用的实现

- Apache Arrow 随机访问文件（Apache Arrow Random Access File）
- Apache Arrow 文件流（Apache Arrow File Stream）
- Apache Arrow Flight
