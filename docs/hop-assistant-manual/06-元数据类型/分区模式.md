# 分区模式（Partition Schema）

## 核心功能

描述一个分区模式。分区模式定义了数据行流将被拆分为多少路。分区的名称可以使用您喜欢的任何名称。

## 主要参数

| 参数 | 默认值 | 说明 |
| --- | --- | --- |
| Partition schema name（分区模式名称） | | 此分区模式使用的名称 |
| Dynamically create the schema definition（动态创建模式定义） | ? | 是否动态创建模式定义 |
| Number of partitions?（分区数量） | 4 | 分区的数量 |
| Partitions（分区） | | 分区 ID 列表 |

## 相关插件

无特定限制，适用于所有插件。

## 示例（集成测试）

包含静态分区、动态分区、重新分区、流查找分区等多种集成测试用例。
