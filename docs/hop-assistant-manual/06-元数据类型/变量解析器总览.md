# 变量解析器总览（Variable Resolver）

## 核心功能

您可以定义变量解析器来帮助您解析以下格式的表达式：

```
#{name:key:element}
```

- `name`：用于解析变量的元数据元素名称
- `key`：用于检索值的键或路径
- `element`：如果检索到的值是 JSON 格式，我们可以从中选取其中一个值（可选）

**重要提示**：解析变量时的错误不会导致管道或工作流级别的失败。

**注意**：出于可用性考虑，错误不会立即在用户界面中显示。否则将无法输入任何表达式。请在 Hop GUI 执行面板的日志中查看解析器抛回的任何错误。

**提示**：如果您发现在表格视图（表格网格）中输入变量表达式太慢，请在配置视图中取消选中在工具提示中解析变量的选项。

## 嵌套解析

变量解析器支持嵌套解析，允许构建更复杂的解析逻辑。

## 可用的变量解析器类型

- Azure Key Vault 变量解析器
- Google Secret Manager 变量解析器
- Hashicorp Vault 变量解析器
- 管道变量解析器
