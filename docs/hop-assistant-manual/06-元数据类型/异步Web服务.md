# 异步 Web 服务（Asynchronous Web Service）

## 核心功能

此 Web 服务变体用于执行长时间运行的工作流。

与通过 Web 服务调用从管道立即获取结果不同，调用后唯一返回的是正在执行的工作流的唯一 ID。使用该唯一 ID，您可以查询工作流的状态。您可以指定附加变量，以便在查询异步运行工作流状态时回报。

## 主要参数

| 参数 | 说明 |
| --- | --- |
| Name（名称） | 此异步 Web 服务使用的名称 |
| Enabled（是否启用） | 布尔标志，指示此 Web 服务是否启用 |
| Filename（文件名） | 此 Web 服务使用的工作流，包含打开所选工作流（"打开"）、创建新工作流（"新建"）或浏览选择现有工作流（"浏览"）的选项 |
| Status variables (, separated)（状态变量，逗号分隔） | 列出查询异步状态服务时回报的变量 |
| Content variable（内容变量） | 将包含服务调用内容主体的变量名称 |
