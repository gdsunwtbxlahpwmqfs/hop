# 管道运行配置（Pipeline Run Configuration）

## 核心功能

描述管道以何种方式、使用哪种引擎来执行。管道可以在多种运行时引擎上执行。

## 主要参数

| 参数 | 说明 |
| --- | --- |
| Name（名称） | 此管道运行配置的名称 |
| Description（描述） | 此管道运行配置的描述 |
| Execution information location（执行信息位置） | 此管道运行配置所使用的执行信息位置 |
| Execution data profile（执行数据画像） | 此管道运行配置所使用的执行数据画像 |
| Engine Type（引擎类型） | 可用的管道引擎，详细配置取决于所选引擎 |

## 可用引擎类型

- Beam DataFlow 管道引擎
- Beam Direct 管道引擎
- Beam Flink 管道引擎
- Beam Spark 管道引擎
- Hop 本地管道引擎（Native Local）
- Hop 远程管道引擎（Native Remote）

## 说明

Hop 安装时会自带一个默认的 local（本地）管道运行配置。
