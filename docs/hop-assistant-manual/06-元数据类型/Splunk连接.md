# Splunk 连接（Splunk Connection）

## 核心功能

描述一个 Splunk 连接。

## 主要参数

| 参数 | 说明 |
| --- | --- |
| Connection name（连接名称） | 此 Splunk 连接使用的名称 |
| Hostname or IP address（主机名或 IP 地址） | Splunk 服务器的主机名或 IP 地址 |
| Port（端口） | Splunk 服务器的端口号 |
| Username（用户名） | 登录用户名 |
| Password（密码） | 登录密码 |

## 相关插件
