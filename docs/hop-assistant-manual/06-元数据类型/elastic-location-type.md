# Elastic Location Type

## 描述

使用此 location type，你可以将执行信息存储在 Elastic 数据库中。

## 选项

- *Elastic base server URL*：Elastic 服务器的基 URL，包括端口。例如：`http://localhost:9200`
- *The Elastic API Key to use*：Elastic 服务器的 API 密钥
- *The index name to use*：用于存储执行信息的索引名称。
- *Button 'Create and configure Elastic index'*：执行一系列命令来创建和配置指定索引，使 Hop 能够在 Elastic 中存储和检索信息。
