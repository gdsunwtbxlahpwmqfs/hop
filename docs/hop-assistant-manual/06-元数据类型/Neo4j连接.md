# Neo4j 连接（Neo4j Connection）

## 核心功能

可以通过点击图标（点击它进行编辑）以及下拉菜单（箭头向下）来创建和编辑连接，您可以在各种 Neo4j 对话框中每行连接的右侧找到它们。

它们也可以通过 Hop GUI 中的元数据视图进行管理。

Neo4j 连接以 JSON 格式序列化存储在 Hop 元数据文件夹下：

`metadata/neo4j-connection/`

## 用户界面

在大多数情况下，Neo4j 驱动程序可以自动判断协议和安全选项，让您可以专注于最基本的选项：服务器、端口、用户名和密码。

一如既往，我们建议您永远不要硬编码任何设置，而是使用可以在环境配置文件中定义的环境变量。这样，同一个 Neo4j 连接就可以在多个环境中使用，而无需更改管道或工作流。

## 基本选项

Neo4j 连接的基本选项包括服务器地址、端口、用户名、密码、协议以及安全（加密）相关设置。
