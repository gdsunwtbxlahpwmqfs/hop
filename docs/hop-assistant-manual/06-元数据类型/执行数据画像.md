# 执行数据画像（Execution Data Profile）

## 核心功能

Qi 数据治理平台 执行数据画像在数据流经管道时构建数据画像。可以选择并配置多个数据采样器，以精细调整被画像数据的类型和细节。

## 主要参数

| 参数 | 说明 |
| --- | --- |
| Name（名称） | 此执行数据画像使用的名称 |
| Description（描述） | 此执行数据画像使用的描述 |
| Data Samplers to use（使用的数据采样器） | 此执行数据画像使用的一个或多个数据采样器。详见下文。 |

## 数据采样器（Data Samplers）

| 数据采样器 | 说明 | 选项 |
| --- | --- | --- |
| Data profile output rows（数据画像输出行） | 对转换输出行执行一些基础数据画像 | 采样大小（默认 25）、仅最后转换（默认 true）、最小值（默认 true）、最大值（默认 true）、统计空值（默认 true）、统计非空值（默认 true）、最小长度（默认 true）、最大长度（默认 true） |
| First output rows（首批输出行） | 采样转换输出的前几行 | 采样大小（默认 100） |
| Last output rows（末批输出行） | 采样转换输出的最后几行 | 采样大小（默认 100） |
| Random output rows（随机输出行） | 对转换输出行执行蓄水池采样 | 采样大小（默认 100） |
