# 数据集（Data Set）

## 核心功能

数据集是一个静态的、预定义的数据行集合，用于单元测试。在单元测试中，数据集被用作替代输入数据。

该输入数据集会在单元测试管道中被处理。处理完成后，将管道的结果与被指定为该测试"黄金数据集（golden data set）"的数据集进行比较。

## 主要参数

| 参数 | 说明 |
| --- | --- |
| Name（名称） | 此数据集使用的名称 |
| Description（描述） | 此数据集使用的描述 |
| Set Folder（设置文件夹，或使用 HOP_DATASETS_FOLDER） | 可以找到并保存数据集的项目文件夹 |
| Base file name（基础文件名） | 数据集的默认名称 |
| 数据集字段及其在文件中的列名 | 字段名、类型、格式、长度和精度的列表。描述此数据集的文件布局 |

## 相关插件

无特定限制，适用于所有插件。
