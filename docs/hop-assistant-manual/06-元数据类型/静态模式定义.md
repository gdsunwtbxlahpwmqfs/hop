# 静态模式定义（Static Schema Definition）

## 核心功能

静态模式定义描述了一种可以应用到选定输入/输出转换集合上的数据流布局。模式定义是一种定义可复用、重复出现的流布局的方式，可在多个管道中重复使用。

能够在多个管道中使用相同的模式定义，简化了开发过程，并降低了手动指定字段或布局时出错的风险。

## 相关插件（转换）

- Text File Input（文本文件输入）
- Text File Output（文本文件输出）
- CSV Input（CSV 输入）
- Excel Input（Excel 输入）
- Excel Writer（Excel 写入）
- Schema Mapping（模式映射）
