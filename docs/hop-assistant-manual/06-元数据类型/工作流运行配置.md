# 工作流运行配置（Workflow Run Configuration）

## 核心功能

描述如何运行一个工作流。工作流可以在本地或远程执行。

## 主要参数

| 参数 | 说明 |
| --- | --- |
| Name（名称） | 此工作流运行配置的名称 |
| Description（描述） | 此工作流运行配置的描述 |
| Execution information location（执行信息位置） | 此工作流运行配置所使用的执行信息位置 |
| Workflow engine type（工作流引擎类型） | Hop 本地工作流引擎或 Hop 远程工作流引擎 |
| Safe mode（安全模式） | 默认：false。在安全模式下，Hop 在开始执行工作流之前会执行一系列数据类型及其他校验 |

## 说明

Hop 安装时会自带一个默认的 local（本地）工作流运行配置。
