# 关系型数据库连接（Relational Database Connection）

## 核心功能

描述连接到关系型数据库所需的全部信息。

Hop 开箱即支持数十种关系型数据库。如果您的特定数据库没有内置支持，通常仍可通过 Generic（通用）数据库连接方式进行连接。

## 相关插件

可被多种 Action（工作流动作）和 Transform（管道转换）使用，例如检查数据库连接、列是否存在检查、MS SQL 批量加载、各类数据库相关操作等。

## 说明

该元数据类型是 Hop 中最基础、使用最广泛的连接定义之一，定义一次后即可在管道和工作流中反复引用，避免在多处重复填写连接信息。
