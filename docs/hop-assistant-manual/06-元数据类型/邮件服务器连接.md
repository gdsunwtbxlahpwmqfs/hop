# 邮件服务器连接（Mail Server Connection）

## 核心功能

邮件服务器连接是一个到邮件服务器的连接，可以从诸如"邮件（Mail）"动作等重复使用。后续版本将添加对其他电子邮件相关动作和转换的支持。

## 主要参数

| 参数 | 说明 |
| --- | --- |
| Connection name（连接名称） | 此邮件服务器连接使用的名称 |
| Connection protocol（连接协议） | 此连接使用的连接协议。支持的选项有 SMTP、IMAP 和 POP。目前不支持 MBOX。 |
| Server host（服务器主机） | 此连接使用的邮件服务器主机名 |
| Server port（服务器端口） | 此连接使用的邮件服务器端口 |
| Use authentication?（是否使用认证） | 此连接是否使用认证 |
| User XOAuth2?（是否使用 XOAuth2） | 此邮件服务器连接是否使用 XOAuth2 认证 |
| Username（用户名） | 此连接使用的用户名 |
| Password（密码） | 此连接使用的密码 |
| Use secure authentication?（是否使用安全认证） | 此连接是否使用安全认证 |
| Secure authentication type（安全认证类型） | 此连接使用的安全认证类型。支持的选项有 SSL、TLS、TLS1.2 |
| Check server identity?（是否检查服务器身份） | 是否检查服务器身份 |
| Trusted hosts（可信主机） | 以空格分隔的可信主机列表，例如："host1 host2 host3" |
| Use proxy?（是否使用代理） | 此邮件服务器连接是否使用代理服务器 |
| Proxy username（代理用户名） | 此连接使用的代理用户名 |
| Proxy password（代理密码） | 此连接使用的代理密码 |
