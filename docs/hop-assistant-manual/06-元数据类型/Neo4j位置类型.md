# Neo4j 位置类型（Neo4j Location Type）

## 核心功能

使用此位置类型，您可以将执行信息存储在 Neo4j 图数据库中。

## 主要选项

- **Neo4j 连接**：用于存储信息的 Neo4j 连接的名称。
- **"显示索引 DDL"按钮**：显示您希望在 Neo4j 连接上执行的 CREATE INDEX 语句，以确保在写入和读取信息时保持良好的性能。
- **"复制 Neo4j 索引动作到剪贴板"按钮**：将 Neo4j 索引动作复制到剪贴板，以便您可以直接将其粘贴到工作流中。它将确保创建正确的索引（如果需要），以在使用执行信息位置时获得出色的性能。
