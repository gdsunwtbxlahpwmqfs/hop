# 工具栏

您在[创建 Pipeline](create-pipeline.md) 中学习了如何创建 pipeline。您将在 pipeline 画布上执行许多操作，但 pipeline 编辑器的主工具栏还提供了许多额外功能。

还有两个重要的工具栏用于[单元测试](pipeline-unit-testing.md)和[项目与环境](../index.md)。请查看相应页面以了解有关管理项目和环境以及如何为 pipeline 编写单元测试的更多信息。

让我们来浏览顶部工具栏：

![Hop - Pipeline Toolbar,align="left"](../assets/images/getting-started/getting-started-pipeline-toolbar.png)

| 操作 | 图标 | 说明 |
|---|---|---|
| run | ![Run,25px,align="bottom"](../assets/images/getting-started/icons/run.svg) | 开始执行 pipeline |
| pause | ![Pause,25px,align="bottom"](../assets/images/getting-started/icons/pause.svg) | 暂停 pipeline 的执行 |
| stop | ![Stop,25px,align="bottom"](../assets/images/getting-started/icons/stop.svg) | 停止 pipeline 的执行 |
| preview & debug | ![Preview,25px,align="bottom"](../assets/images/getting-started/icons/view.svg) | 预览 pipeline，并可选择在满足行条件时暂停（同一对话框） |
| print | ![Print,25px,align="bottom"](../assets/images/getting-started/icons/print.png) | 打印 pipeline |
| undo | ![Undo,25px,align="bottom"](../assets/images/getting-started/icons/Antu_edit-undo.svg) | 撤销操作 |
| redo | ![Redo,25px,align="bottom"](../assets/images/getting-started/icons/Antu_edit-redo.svg) | 重做操作 |
| align | ![Snap To Grid,25px,align="bottom"](../assets/images/getting-started/icons/snap-to-grid.svg) | 将指定的（选中的）transform 对齐到指定的网格大小 |
| align left | ![Align Left,25px,align="bottom"](../assets/images/getting-started/icons/align-left.svg) | 将选中的 transform 与选区中最左的 transform 左对齐 |
| align right | ![Align Right,25px,align="bottom"](../assets/images/getting-started/icons/align-right.svg) | 将选中的 transform 与选区中最右的 transform 右对齐 |
| align top | ![Align Top,25px,align="bottom"](../assets/images/getting-started/icons/align-top.svg) | 将选中的 transform 与选区中最上的 transform 顶对齐 |
| align bottom | ![Align Bottom,25px,align="bottom"](../assets/images/getting-started/icons/align-bottom.svg) | 将选中的 transform 与选区中最下的 transform 底对齐 |
| distribute horizontally | ![Distribute Horizontally,25px,align="bottom"](../assets/images/getting-started/icons/distribute-horizontally.svg) | 将选中的 transform 在选区中最左和最右的 transform 之间均匀分布 |
| distribute vertically | ![Distribute Vertically,25px,align="bottom"](../assets/images/getting-started/icons/distribute-vertically.svg) | 将选中的 transform 在选区中最上和最下的 transform 之间均匀分布 |
