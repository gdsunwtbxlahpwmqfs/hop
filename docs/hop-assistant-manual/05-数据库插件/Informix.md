# Informix（IBM Informix）

## 概述

IBM Informix 是 IBM 提供的关系型数据库管理系统，以其高性能和低管理开销著称。使用此数据库需要单独下载 JDBC 驱动程序。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | https://download.oracle.com/otndocs/jcp/jdbc-4.0-fr-eval-oth-JSpec/ |
| 内置版本 | 无 |
| Hop 依赖 | 无 |
| 文档 | https://www.ibm.com/support/knowledgecenter/en/SSGU8G_12.1.0/com.ibm.jdbc_pg.doc/jdbc.htm |
| JDBC 连接串 | jdbc:informix-sqli://hostname:1533/databaseName |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
