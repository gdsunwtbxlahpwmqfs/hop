# Gupta SQLBase（Gupta SQLBase）

## 概述

Gupta SQLBase（现为 OpenText 旗下产品）是一个关系型数据库管理系统。使用此数据库需要单独下载 JDBC 驱动程序。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | https://knowledge.opentext.com/knowledge/cs.dll/kcs/kb |
| 内置版本 | 无 |
| Hop 依赖 | 无 |
| 文档 | https://knowledge.opentext.com/knowledge/cs.dll/kcs/kb |
| JDBC 连接串 | jdbc:sqlbase://hostname:port/databaseName |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
