# PostgreSQL（PostgreSQL）

## 概述

PostgreSQL 是一个功能强大的开源关系型数据库管理系统，以其可靠性、数据完整性和正确性著称。其 JDBC 驱动已内置在 hfxt data process 中，无需额外下载。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | 内置（Included） |
| 内置版本 | 42.7.10 |
| Hop 依赖 | 无 |
| 文档 | https://jdbc.postgresql.org/documentation/head/index.html |
| JDBC 连接串 | jdbc:postgresql://host:port/database |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
