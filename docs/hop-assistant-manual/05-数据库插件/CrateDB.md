# CrateDB（CrateDB）

## 概述

CrateDB 是一个分布式的 SQL 数据库，基于关系型架构设计，适用于物联网（IoT）和机器数据等场景。其 JDBC 驱动已内置在 Qi 数据治理平台 中。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | 内置（Included） |
| 内置版本 | 2.7.0 |
| Hop 依赖 | 无 |
| 文档 | https://cratedb.com/docs/jdbc/en/latest/index.html |
| JDBC 连接串 | jdbc:crate://host:port/database |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
