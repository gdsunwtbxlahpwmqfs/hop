# Vertica（Micro Focus Vertica）

## 概述

Vertica 是一个面向列式存储的分析型关系数据库，专为数据仓库和大规模分析场景优化。使用此数据库需要单独下载 JDBC 驱动程序。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | https://www.vertica.com/download/vertica/client-drivers/ |
| 内置版本 | 无 |
| Hop 依赖 | 无 |
| 文档 | https://www.vertica.com/docs/9.2.x/HTML/Content/Authoring/ConnectingToVertica/ClientJDBC/JDBCConnectionProperties.htm |
| JDBC 连接串 | jdbc:vertica://VerticaHost:portNumber/databaseName |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
