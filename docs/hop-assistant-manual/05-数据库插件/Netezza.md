# Netezza（IBM Netezza）

## 概述

IBM Netezza 是一个专为高性能分析而设计的数据仓库设备。其 JDBC 驱动包含在数据库安装包中。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | 包含在数据库安装中 |
| 内置版本 | 无 |
| Hop 依赖 | 无 |
| 文档 | https://www.ibm.com/support/knowledgecenter/SSULQD_7.2.1/com.ibm.nz.datacon.doc/c_datacon_installing_configuring_jdbc.html |
| JDBC 连接串 | jdbc:netezza://hostname:5490/databaseName |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
