# Apache Derby

| 选项 | 信息 |
|---|---|
| 类型 | Relational |
| 驱动 | 已内置 |
| 内置版本 | 10.17.1.0 |
| Hop 依赖 | 无 |
| 文档 | [文档链接](https://db.apache.org/derby/derby_downloads) |
| JDBC Url | jdbc:derby:<host>[:<port>]/<database> |
| JDBC Url（嵌入式） | jdbc:derby:<database> |
| 驱动文件夹 | <Hop Installation>/lib/jdbc |

> **📝 注意:** Derby JDBC 驱动有两种不同模式，如果您想使用 Derby 嵌入式模式，请将 _derbyclient.jar_ 文件替换为 _derby.jar_
