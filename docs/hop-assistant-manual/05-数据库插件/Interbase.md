# Interbase（Embarcadero InterBase）

## 概述

InterBase 是 Embarcadero 公司提供的关系型数据库管理系统，以轻量级和高性能著称。其 JDBC 驱动包含在数据库安装包中。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | 包含在数据库安装中 |
| 内置版本 | 无 |
| Hop 依赖 | 无 |
| 文档 | http://docwiki.embarcadero.com/InterBase/2020/en/Programming_with_JDBC |
| JDBC 连接串 | jdbc:interbase://hostname:3050/path/to/database.ib |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
