# IBM Universe（IBM Universe）

## 概述

IBM Universe（UniVerse）是 IBM 提供的多值（MultiValue）数据库管理系统，属于 IBM U2 产品系列。其 JDBC 驱动包含在数据库安装包中。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | 包含在数据库安装中 |
| 内置版本 | 无 |
| Hop 依赖 | 无 |
| 文档 | https://www-05.ibm.com/e-business/linkweb/publications/servlet/pbi.wss?CTY=US&FNC=SRX&PBL=G251-1210-00 |
| JDBC 连接串 | jdbc:ibmu2://localhost |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
