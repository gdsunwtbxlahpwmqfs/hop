# DB2（IBM DB2）

## 概述

IBM DB2 是 IBM 提供的关系型数据库管理系统。使用此数据库需要单独下载 JDBC 驱动程序。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | https://www.ibm.com/support/pages/db2-jdbc-driver-versions-and-downloads |
| 内置版本 | 无 |
| Hop 依赖 | 无 |
| 文档 | https://www.ibm.com/support/knowledgecenter/SSCQGF_7.2.0.1/com.ibm.IBMDI.doc_7.2.0.1/rg_conn_jdbc.html |
| JDBC 连接串 | jdbc:db2://hostname:port/dbname |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
