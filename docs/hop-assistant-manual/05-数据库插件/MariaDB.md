# MariaDB（MariaDB）

## 概述

MariaDB 是 MySQL 的一个分支，由原 MySQL 开发团队维护，是一款开源的关系型数据库管理系统。此插件依赖 MySQL 数据库插件，使用此数据库需要单独下载 JDBC 驱动程序。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | https://mariadb.com/kb/en/about-mariadb-connector-j/ |
| 内置版本 | 无 |
| Hop 依赖 | MySQL 数据库插件 |
| 文档 | https://mariadb.com/kb/en/about-mariadb-connector-j/ |
| JDBC 连接串 | jdbc:mariadb://hostname:port/databaseName |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
