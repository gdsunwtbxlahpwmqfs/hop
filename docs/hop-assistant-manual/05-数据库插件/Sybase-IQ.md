# Sybase IQ（SAP Sybase IQ）

## 概述

Sybase IQ（现为 SAP IQ）是一个面向列式存储（Column Oriented）的分析型数据库，专为数据仓库和商业智能场景优化。其 JDBC 驱动包含在数据库安装包中。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 列式存储（Column Oriented） |
| 驱动 | 包含在数据库安装中 |
| 内置版本 | 无 |
| Hop 依赖 | 无 |
| 文档 | http://infocenter.sybase.com/help/index.jsp?topic=/com.sybase.infocenter.dc01776.1600/doc/html/san1357754910584.html |
| JDBC 连接串 | jdbc:sybase:Tds:localhost:2638 |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
