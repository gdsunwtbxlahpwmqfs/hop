# Teradata（Teradata）

## 概述

Teradata 是一个专为大规模数据仓库和分析而设计的关系型数据库管理系统。使用此数据库需要单独下载 JDBC 驱动程序。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | https://downloads.teradata.com/download/connectivity/jdbc-driver |
| 内置版本 | 无 |
| Hop 依赖 | 无 |
| 文档 | https://teradata-docs.s3.amazonaws.com/doc/connectivity/jdbc/reference/current/frameset.html |
| JDBC 连接串 | jdbc:teradata://Hostname |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
