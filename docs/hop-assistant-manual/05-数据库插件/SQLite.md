# SQLite（SQLite）

## 概述

SQLite 是一个轻量级的嵌入式关系型数据库引擎，不需要独立的服务器进程。其 JDBC 驱动已内置在 Apache Hop 中。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | 内置（Included） |
| 内置版本 | 3.51.2.0 |
| Hop 依赖 | 无 |
| 文档 | https://www.sqlitetutorial.net/sqlite-java/sqlite-jdbc-driver/ |
| JDBC 连接串 | jdbc:sqlite:sample.db |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
