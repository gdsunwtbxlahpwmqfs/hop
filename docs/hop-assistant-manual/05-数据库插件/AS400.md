# AS400（AS400）

## 概述

AS400（即 IBM i 系列）是 IBM 提供的关系型数据库系统。其 JDBC 驱动已内置在 Qi 数据治理平台 中，无需额外下载。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | 内置（Included） |
| 内置版本 | 21.0.0 |
| Hop 依赖 | 无 |
| 文档 | https://www.ibm.com/support/knowledgecenter/ssw_ibm_i_71/rzahh/javadoc/com/ibm/as400/access/doc-files/JDBCProperties.html |
| JDBC 连接串 | jdbc:as400://hostname/default-schema |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
