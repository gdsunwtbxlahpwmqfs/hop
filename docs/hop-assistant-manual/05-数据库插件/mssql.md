# MS SqlServer

此数据库类型使用 JTDS 驱动。
创建新的数据库连接时，建议使用原生连接类型。

| 选项 | 信息 |
|---|---|
| 类型 | Relational |
| 驱动 | [驱动链接](https://sourceforge.net/projects/jtds/files/jtds/) |
| 内置版本 | 无 |
| Hop 依赖 | 无 |
| 文档 | [文档链接](http://jtds.sourceforge.net/faq) |
| JDBC Url | jdbc:jtds:sqlserver://<server>[:<port>][/<database>][;<property>=<value>[;...]] |
| 驱动文件夹 | <Hop Installation>/lib/jdbc |
