# Exasol（Exasol）

## 概述

Exasol 是一款高性能的内存分析型关系数据库，专为数据分析（OLAP）场景优化。使用此数据库需要单独下载 JDBC 驱动程序。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | https://docs.exasol.com/db/latest/connect_exasol/drivers/jdbc.htm#JDBCDriver |
| 内置版本 | 无 |
| Hop 依赖 | 无 |
| 文档 | https://docs.exasol.com/connect_exasol/drivers/jdbc.htm |
| JDBC 连接串 | jdbc:exa:&lt;server&gt;:&lt;port8563&gt; |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
