# MonetDB（MonetDB）

## 概述

MonetDB 是一个开源的列式存储（Column Store）数据库管理系统，专为数据仓库和数据分析场景设计。其 JDBC 驱动已内置在 Qi 数据治理平台 中。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 列式存储（Column Store） |
| 驱动 | 内置（Included） |
| 内置版本 | 12.0 |
| Hop 依赖 | 无 |
| 文档 | https://www.monetdb.org/Documentation/Manuals/SQLreference/Programming/JDBC |
| JDBC 连接串 | jdbc:monetdb://hostname/databaseName |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
