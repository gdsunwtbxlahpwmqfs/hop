# Firebird（Firebird）

## 概述

Firebird 是一个开源的关系型数据库管理系统，提供了良好的 SQL 标准兼容性。使用此数据库需要单独下载 JDBC 驱动程序。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | https://firebirdsql.org/en/jdbc-driver/ |
| 内置版本 | 无 |
| Hop 依赖 | 无 |
| 文档 | https://firebirdsql.github.io/jaybird-manual/jaybird_manual.html |
| JDBC 连接串 | jdbc:firebirdsql://localhost:3050/c:/database/example.fdb |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
