# Kingbase ES（人大金仓 KingbaseES）

## 概述

Kingbase ES（人大金仓）是一款国产关系型数据库管理系统，兼容 PostgreSQL 协议。使用此数据库需要单独获取 JDBC 驱动程序。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | （需单独获取） |
| 内置版本 | 无 |
| Hop 依赖 | 无 |
| 文档 | https://www.kingbase.com.cn/zhuanti/kes/html/jdbc.html |
| JDBC 连接串 | jdbc:kingbase8://host:port/database |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
