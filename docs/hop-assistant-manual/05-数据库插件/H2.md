# H2（H2 Database）

## 概述

H2 是一个开源的轻量级关系型数据库，用 Java 编写，支持嵌入式和服务器模式。其 JDBC 驱动已内置在 Apache Hop 中。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | 内置（Included） |
| 内置版本 | 2.3.232 |
| Hop 依赖 | 无 |
| 文档 | https://jdbc.postgresql.org/documentation/head/index.html |
| JDBC 连接串 | jdbc:h2:[file:][&lt;path&gt;]&lt;databaseName&gt; |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
