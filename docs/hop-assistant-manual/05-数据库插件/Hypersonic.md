# Hypersonic（Hypersonic / HSQLDB）

## 概述

Hypersonic（即 HSQLDB，HyperSQL）是一个开源的关系型数据库管理系统，用 Java 编写，支持内存模式和持久化模式。其 JDBC 驱动已内置在 Apache Hop 中。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | 内置（Included） |
| 内置版本 | 2.7.4 |
| Hop 依赖 | 无 |
| 文档 | http://hsqldb.org/doc/2.0/guide/dbproperties-chapt.html |
| JDBC 连接串 | jdbc:hsqldb:hsql//hostname |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
