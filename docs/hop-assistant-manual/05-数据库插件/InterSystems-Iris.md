# InterSystems Iris（InterSystems IRIS）

## 概述

InterSystems IRIS 是 InterSystems 公司推出的新一代数据平台，集成了数据库管理系统、互操作性和分析能力。使用此数据库需要单独下载 JDBC 驱动程序。

## 主要连接参数

| 参数 | 信息 |
|------|------|
| 类型 | 关系型（Relational） |
| 驱动 | https://intersystems-community.github.io/iris-driver-distribution/ |
| 内置版本 | 无 |
| Hop 依赖 | 无 |
| 文档 | https://docs.intersystems.com/iris20223/csp/docbook/Doc.View.cls?KEY=BJAVA_connecting |
| JDBC 连接串 | jdbc:IRIS://hostname:port/database |
| 驱动文件夹 | &lt;Hop 安装目录&gt;/lib/jdbc |
