# Hop Gui

Hop Gui 是 Hop 的可视化开发环境。你将在 Hop GUI 的各个 perspective 中花费大量时间。

你大部分的时间将用于构建、预览、运行和调试 workflow 和 pipeline，但也可以通过 Hop Gui 的 metadata perspective 进行大量配置，使用 search perspective 搜索任何 metadata，通过 file explorer perspective 管理各种文件甚至进行版本控制等等。

本页面涵盖了一些专门针对 Hop Gui 使用的话题。
请查看 [pipelines](../07-管道/pipelines.md) 和 [workflows](../08-工作流/workflows.md) 章节，了解更多关于在 Hop Gui 中创建和运行 pipeline 和 workflow 的信息。

这里涵盖的内容包括：

- [File Dialog](file-dialog.md)：多功能的 Hop 文件浏览器和选择对话框。
- [Popup Dialog](hop-gui-popup-dialog.md)：用于处理 metadata、workflow、pipeline 和 hop 的多功能弹出对话框
- [Perspectives](perspectives.md)：Hop Gui 中的各种 perspective
- [Working with git](hop-gui-git.md)：在 git 版本控制系统中管理文件
- [Keyboard Shortcuts](shortcuts.md)：Hop Gui 中可用的键盘快捷键列表。
- [Hop Web](hop-web.md)：Hop Gui 的 Web 版本，提供与桌面版本完全相同的功能和用户体验
