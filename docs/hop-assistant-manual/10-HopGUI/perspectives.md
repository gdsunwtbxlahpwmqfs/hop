# Hop Gui Perspective

Hop 内置了多个 perspective。

通过点击左侧工具栏中的图标或使用相应的键盘快捷键来切换当前 perspective。
使用键盘快捷键 `CTRL-SHIFT-arrow-up` 和 `CTRL-SHIFT-arrow-down` 可以在各 perspective 之间循环切换。
