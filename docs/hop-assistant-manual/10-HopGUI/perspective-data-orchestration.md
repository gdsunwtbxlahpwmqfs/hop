# Data Orchestration Perspective

图标：![width="24px"](../assets/images/icons/data_orch.svg)

键盘快捷键：`CTRL-Shift-D`

## 描述

Data Orchestration perspective 是你在 Hop Gui 中花费时间最多的地方。
这是构建、测试和运行 [pipelines](pipeline/pipelines.md) 和 [workflows](workflow/workflows.md) 的地方。

## 用法

可以打开或创建 Pipeline 和 workflow 文件。此时它们将在此 perspective 的新标签页中显示。

有关如何开始使用 pipeline 的更多信息，请参阅 [Pipelines 入门指南](getting-started/hop-gui-pipelines.md)。

有关如何开始使用 workflow 的更多信息，请参阅 [Workflows 入门指南](getting-started/hop-gui-workflows.md)。
