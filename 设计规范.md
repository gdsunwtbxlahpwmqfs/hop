# Qi Hop 设计规范与样式指南

> 本文档基于登录页 `login.css` 提取的设计令牌，结合 RAP 主题（`light-mode.css` / `dark-mode.css`）制定，
> 作为整个 Qi Hop 系统的统一视觉与交互规范。所有新增页面、组件和主题文件均应遵循本规范。

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **一致性** | 相同语义的元素在任意页面使用相同的颜色、圆角、间距和交互效果 |
| **令牌驱动** | 所有视觉属性通过 CSS 自定义属性（Design Token）定义，禁止硬编码颜色值 |
| **明暗双主题** | 每个颜色令牌同时提供亮色与暗色值，通过 `prefers-color-scheme` 自动切换 |
| **可访问性优先** | 遵守 WCAG AA 对比度标准，支持 `prefers-reduced-motion`，保留键盘焦点指示器 |
| **响应式适配** | 移动优先，断点统一为 `767px`（手机）/ `768px–1199px`（平板）/ `≥1200px`（桌面） |

## 2. 颜色系统

### 2.1 主色调（Brand）

蓝色系为全系统主色调，源自登录页品牌色，与 Hop 原生 RAP 暗色主题的 `#3584e4` 强调色保持一致。

| 令牌 | 亮色值 | 暗色值 | 用途 |
|------|--------|--------|------|
| `--hop-blue` | `#3b6ea5` | `#3b6ea5` | 品牌主色，标题、Logo |
| `--hop-blue-dark` | `#2c5282` | `#2c5282` | 主色悬浮加深 |
| `--hop-blue-light` | `#4a90d9` | `#4a90d9` | 渐变末端、辅助 |
| `--hop-accent` | `#3584e4` | `#3584e4` | 强调色：焦点边框、选中态、链接 |

> **RAP 主题映射**：`light-mode.css` 中 `Button:default` 的 `#416693`、`Combo:focused` 的 `#4f7cb1`
> 应逐步向 `--hop-accent` 收敛；`dark-mode.css` 已使用 `#3584e4`，无需调整。

### 2.2 背景色（Background）

| 令牌 | 亮色值 | 暗色值 | 用途 |
|------|--------|--------|------|
| `--bg-primary` | `#f0f2f5` | `#1a1a1a` | 页面底色 |
| `--bg-card` | `#ffffff` | `#242424` | 卡片、面板、对话框 |
| `--bg-input` | `#ffffff` | `#2d2d2d` | 输入框默认背景 |
| `--bg-input-focus` | `#ffffff` | `#333333` | 输入框聚焦背景 |

> **RAP 主题映射**：暗色主题的 Composite 背景 `#242424`、Text 背景 `#2d2d2d` 已与此处一致。

### 2.3 文字色（Text）

| 令牌 | 亮色值 | 暗色值 | 用途 |
|------|--------|--------|------|
| `--text-primary` | `#1a1a2e` | `#e8e8e8` | 标题、正文主文字 |
| `--text-secondary` | `#6b7280` | `#a0a0a0` | 标签、辅助说明 |
| `--text-muted` | `#9ca3af` | `#6e6e6e` | 占位符、提示文字 |

### 2.4 边框色（Border）

| 令牌 | 亮色值 | 暗色值 | 用途 |
|------|--------|--------|------|
| `--border-color` | `#e0e3e8` | `#3d3d3d` | 默认边框 |
| `--border-focus` | `var(--hop-accent)` | `var(--hop-accent)` | 聚焦边框 |

### 2.5 状态色（Status）

| 令牌 | 亮色值 | 暗色值 | 用途 |
|------|--------|--------|------|
| `--error-color` | `#e53e3e` | `#e53e3e` | 错误文字、错误边框 |
| `--error-bg` | `#fef2f2` | `#3d2020` | 错误背景 |
| `--success-color` | `#38a169` | `#38a169` | 成功状态 |
| `--warning-color` | `#dd8800` | `#f0ad44` | 警告状态（建议值） |

## 3. 排版规则（Typography）

### 3.1 字体族

```css
--font-base: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto,
    "Helvetica Neue", Arial, "PingFang SC", "Microsoft YaHei", sans-serif;
```

> 中文优先使用 `PingFang SC`（macOS）和 `Microsoft YaHei`（Windows），西文使用系统默认无衬线字体。

### 3.2 字号层级

| 层级 | 字号 | 字重 | 行高 | 用途 |
|------|------|------|------|------|
| Display | 24px | 700 | 1.3 | 品牌标题、页面主标题 |
| Heading | 18px | 600 | 1.4 | 区块标题 |
| Body-LG | 16px | 400 | 1.5 | 按钮文字、重要正文 |
| Body | 15px | 400 | 1.5 | 输入框文字、正文 |
| Label | 13px | 600 | 1.4 | 表单标签、复选框文字 |
| Caption | 12px | 400 | 1.4 | 错误提示、页脚提示 |

### 3.3 字间距规则

| 场景 | letter-spacing |
|------|----------------|
| 品牌大标题 | `-0.5px` |
| 副标题（大写） | `0.5px` |
| 按钮文字 | `1px` |
| 正文 | `0`（默认） |
| 标签 | `0.2px` |

## 4. 间距系统（Spacing）

采用 **4px 基准网格**，所有间距为 4 的倍数。

| 令牌 | 值 | 典型用途 |
|------|----|----------|
| `--space-xs` | `4px` | 错误文字与输入框间距、图标内边距 |
| `--space-sm` | `8px` | 图标与文字间距、紧凑组件间隔 |
| `--space-md` | `12px` | Alert 内边距、品牌区间距 |
| `--space-lg` | `16px` | 输入框水平内边距、卡片宽间距 |
| `--space-xl` | `24px` | 卡片底部上间距、页脚上间距 |
| `--space-2xl` | `32px` | 标题区域下间距 |
| `--space-3xl` | `40px` | 卡片内边距（桌面） |

### 常见组合

| 组件 | padding | 说明 |
|------|---------|------|
| 登录卡片（桌面） | `40px 36px` | — |
| 登录卡片（平板） | `36px 32px` | — |
| 登录卡片（手机） | `32px 24px` | — |
| 表单组间距（gap） | `18px` | — |
| 输入框高度 | `48px` | 含图标左内边距 `44px` |

## 5. 圆角系统（Border Radius）

| 令牌 | 值 | 用途 |
|------|----|------|
| `--radius-sm` | `6px` | 小按钮、复选框、Tooltip |
| `--radius-md` | `10px` | 输入框、按钮、Alert |
| `--radius-lg` | `16px` | 卡片、对话框 |
| `--radius-full` | `50%` | 圆形头像、Spinner |

> **RAP 主题对齐**：`light-mode.css` 使用 `2px` 圆角（传统桌面风格），
> `dark-mode.css` 已升级至 `4px–6px`。新增 RAP 组件建议使用 `4px` 以上圆角。

## 7. 动效系统（Motion）

### 7.1 过渡时长与缓动

| 令牌 | 时长 | 缓动函数 | 用途 |
|------|------|----------|------|
| `--transition-fast` | `0.15s` | `cubic-bezier(0.4, 0, 0.2, 1)` | 颜色、边框等即时反馈 |
| `--transition-base` | `0.3s` | `cubic-bezier(0.4, 0, 0.2, 1)` | 卡片悬浮、面板展开 |
| `--transition-slow` | `0.5s` | `cubic-bezier(0.16, 1, 0.3, 1)` | 页面入场、卡片滑入 |

### 7.2 关键帧动画

| 动画名 | 效果 | 用途 |
|--------|------|------|
| `cardFadeIn` | 透明度 0→1 | 页面/卡片入场 |
| `cardSlideUp` | Y 轴 `30px`→`0` + 缩放 `0.98`→`1` | 卡片入场（与 fadeIn 叠加） |
| `alertSlideDown` | Y 轴 `-8px`→`0` + 高度展开 | 错误提示出现 |
| `shake` | X 轴 `±6px` 抖动 `0.35s` | 输入校验失败 |
| `spin` | `rotate(360deg)` 线性循环 | 加载 Spinner（`0.7s`） |
| `floatUp` | 从底部浮至顶部 + 渐隐 | 背景装饰气泡（`18s–30s`） |
| `logoBounce` | Y 轴 `0`→`-8px` 循环 | 过渡页 Logo 弹跳 |
| `overlayFadeIn` | 透明度 0→1 | 全屏遮罩出现 |

### 7.3 无障碍降级

```css
@media (prefers-reduced-motion: reduce) {
    *, *::before, *::after {
        animation-duration: 0.01ms !important;
        animation-iteration-count: 1 !important;
        transition-duration: 0.01ms !important;
    }
    .bg-bubble { display: none; }   /* 装饰性动画直接关闭 */
}
```

## 8. 组件规范

### 8.1 按钮（Button）

| 属性 | 主按钮（提交） | 次按钮 | 文本按钮 |
|------|----------------|--------|----------|
| 高度 | `48px` | `40px` | `32px` |
| 圆角 | `var(--radius-md)` | `var(--radius-md)` | `var(--radius-sm)` |
| 背景 | `linear-gradient(135deg, var(--hop-blue), var(--hop-blue-light))` | `var(--bg-card)` + `1.5px` 边框 | `transparent` |
| 文字色 | `#ffffff` | `var(--text-primary)` | `var(--hop-accent)` |
| 字号 | `16px / 600` | `14px / 600` | `14px / 400` |
| 字间距 | `1px` | `0.5px` | `0` |
| 悬浮 | `translateY(-1px)` + 阴影加深 | 边框色加深 | 文字下划线 |
| 按下 | `translateY(0)` + 阴影收浅 | 背景微灰 | — |
| 禁用 | `opacity: 0.5` + `cursor: not-allowed` | 同左 | 同左 |
| 加载 | `pointer-events: none` + 显示 Spinner | 同左 | — |

**光泽效果**（主按钮特有）：
```css
.btn-submit::before {
    content: '';
    position: absolute;
    top: 0; left: -100%;
    width: 100%; height: 100%;
    background: linear-gradient(90deg, transparent, rgba(255,255,255,0.2), transparent);
    transition: left 0.5s ease;
}
.btn-submit:hover::before { left: 100%; }
```

### 8.2 输入框（Input）

| 属性 | 值 |
|------|----|
| 高度 | `48px` |
| 内边距 | `0 14px 0 44px`（左侧留图标位） |
| 边框 | `1.5px solid var(--border-color)` |
| 圆角 | `var(--radius-md)` |
| 字号 | `15px`（手机端 `16px` 防 iOS 缩放） |
| 悬浮 | 边框色 → `var(--text-muted)` |
| 聚焦 | 边框色 → `var(--border-focus)` + `box-shadow: var(--shadow-focus)` |
| 错误 | 边框色 → `var(--error-color)` + `animation: shake` |
| 占位符色 | `var(--text-muted)` |

**左侧图标**：绝对定位 `left: 14px`，跟随输入框聚焦变为 `var(--hop-accent)`。

### 8.3 卡片（Card）

| 属性 | 值 |
|------|----|
| 背景 | `var(--bg-card)` |
| 圆角 | `var(--radius-lg)` |
| 阴影 | `var(--shadow-lg)` |
| 内边距 | `40px 36px`（桌面）/ `32px 24px`（手机） |
| 入场动画 | `cardFadeIn` + `cardSlideUp`（`var(--transition-slow)`） |

### 8.4 提示横幅（Alert）

| 类型 | 背景 | 文字色 | 边框 |
|------|------|--------|------|
| Error | `var(--error-bg)` | `var(--error-color)` | `1px solid rgba(229,62,62,0.2)` |
| Success（建议） | `#f0fdf4` | `var(--success-color)` | `1px solid rgba(56,161,105,0.2)` |
| Warning（建议） | `#fffbeb` | `var(--warning-color)` | `1px solid rgba(221,136,0,0.2)` |

- 内边距：`12px 16px`
- 圆角：`var(--radius-md)`
- 图标：`18×18px`，`flex-shrink: 0`
- 入场动画：`alertSlideDown`

### 8.5 复选框（Checkbox）

自定义样式，隐藏原生 `input[type=checkbox]`：

| 状态 | 外观 |
|------|------|
| 未选中 | `18×18px` 方框，`1.5px solid var(--border-color)`，圆角 `4px` |
| 悬浮 | 边框 → `var(--hop-accent)` |
| 选中 | 背景 `var(--hop-accent)` + 白色对勾图标 |
| 聚焦 | `box-shadow: var(--shadow-focus)` |

### 8.6 加载指示器（Spinner）

```css
.spinner {
    width: 20px; height: 20px;
    border: 2.5px solid rgba(255,255,255,0.3);
    border-top-color: #ffffff;
    border-radius: 50%;
    animation: spin 0.7s linear infinite;
}
```

### 8.7 过渡遮罩（Transition Overlay）

全屏遮罩，用于页面跳转间的过渡（如登录成功后等待 RAP 加载）：

- 背景：渐变（`var(--gradient-bg)`）
- 居中 Logo：`logoBounce` 动画 `1.8s` 循环
- 文字：白色 `16px / 500`，带文字阴影
- Spinner：`32×32px`，白色，`spin 0.8s`

## 9. 响应式断点

| 断点 | 宽度 | 调整内容 |
|------|------|----------|
| 桌面 | `≥1200px` | 基准样式，卡片 `max-width: 420px` 居中 |
| 平板 | `768px–1199px` | 卡片内边距 → `36px 32px` |
| 手机 | `≤767px` | 卡片内边距 → `32px 24px`；输入框字号 → `16px`（防 iOS 缩放）；背景气泡动画放慢至 `30s` |

**视口设置**：
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0, maximum-scale=1.0, user-scalable=no">
```

## 10. 无障碍（Accessibility）清单

- [x] 所有交互元素提供 `aria-label` / `role`
- [x] 错误提示使用 `role="alert"` 或 `role="status"`
- [x] 焦点可见：所有可聚焦元素提供 `:focus-visible` 样式（光环 `--shadow-focus`）
- [x] 颜色对比度 ≥ `4.5:1`（正文）/ `3:1`（大字号）
- [x] 支持 `prefers-reduced-motion: reduce`
- [x] 支持 `prefers-color-scheme: dark`
- [x] 禁用态使用 `opacity` + `cursor: not-allowed`，而非仅变灰
- [x] 表单 `<label>` 与 `<input>` 通过 `for` / `id` 关联

## 12. RAP 主题对齐建议

当前 RAP 主题与登录页存在以下差异，建议逐步统一：

| 属性 | light-mode.css 现状 | 建议目标（对齐登录页） |
|------|---------------------|------------------------|
| 强调色 | `#416693` / `#4f7cb1` / `#5882b5`（分散） | `#3584e4`（统一为 `--hop-accent`） |
| 圆角 | `2px`（偏小） | `4px`（按钮）/ `6px`（面板） |
| 字体 | `Verdana, "Lucida Sans"` | `Segoe UI, Roboto` 系（更现代） |
| 选中态 | `#c5dbef`（偏淡） | `rgba(53,132,228,0.25)`（与暗色一致） |

`dark-mode.css` 已与登录页暗色令牌高度一致（`#242424` / `#2d2d2d` / `#3584e4`），仅需统一圆角与字体即可。
