# 注意事项

## 1. rap 模块 Monaco 资源在 IDE 中不会自动生成

根 `pom.xml` 的 m2e lifecycle-mapping 中，`maven-antrun-plugin:run` 和 `download-maven-plugin:wget` 都设成了 `<ignore />`，意味着 **IDE 增量构建不会生成 Monaco 资源**（避免每次增量编译都触发网络下载和文件拷贝）。

如果要在 IDE 里运行或调试 `rap` 模块，请先在命令行执行一次以下命令，让资源落盘：

```bash
./mvnw generate-resources -pl rap -am
```

执行后资源会生成到 `rap/target/generated-resources/org/apache/hop/ui/hopgui/monaco/` 目录，IDE 即可正常引用。
